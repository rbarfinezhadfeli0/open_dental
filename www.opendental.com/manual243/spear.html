Spear Education

Spear Education is a practice solution consulting company that provides education, coaching, and analytics to improve dental practices at a business level. Spear's real-time analytics run queries in Open Dental which can cause slowness.

There are two options to help counter slowness issues caused by Spear.

  1. Set up a reporting server utilizing one-way replication. See One-Way Replication Setup.
  2. Convert tables from MyISAM to InnoDB. See MySQL InnoDB.
