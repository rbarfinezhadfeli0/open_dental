ODMobile

ODMobile (formerly Open Dental Mobile) is an app installed to a user's mobile device.

In eServices Setup, click Signup. In the Basics tab, enable Mobile.

Note: Fees displayed on the screenshot above may not be current. Refer to Fees for Support and Services for the most up-to-date pricing.

Set Up ODMobile in Open Dental

  1. Sign up for ODMobile in the eServices Signup Portal.
  2. Set up Mobile Settings to use when logging into the app. This only needs done once and is used for all Open Dental apps.
  3. Set up Users to log into ODMobile.
    • Users must have the Mobile Web/ODMobile security permission.
    • Passwords must be strong must be enabled in Global Security Settings.
    • Users must have a strong password.

Install Open Dental on a Mobile Device

ODMobile is designed for phones, not tablets.

  1. Install the app to each desired device. ODMobile is available for iOS and Android.
    • App Store
    • Google Play
  2. In the app, enter the Mobile Settings to connect the app with the database.
  3. Log into the app using an Open Dental username and password.
