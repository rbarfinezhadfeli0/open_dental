Edit Procedure Canada Tab

For Canada users, in the Procedure Info window, click the Canada tab.

Procedure Type Code: By default, none is selected. Change the type code if it is needed on Canada Claims.

Lab Fee 1 / 2: Enter any lab fees associated with this procedure (do not enter on the lab procedure itself). For each lab fee entered on a non-lab procedure, one lab procedure with code 99111 is automatically created (beginning with ^^ to visually indicate it is attached). Also see Canada Lab Fees.
