Version 20.1

See Versions

Version 20.1 was released on 05/12/2020.

Overview of Version 20.1.

Account Module

  • Simplified Income Transfer Manager to allocate all credits to outstanding charges. Income Transfer Manager
  • Ability to determine when interest begins charging on payment plans. Old Payment Plan, Payment Plan
  • Create a Dynamic Payment Plan on an Ortho Case. Ortho Case

Chart Module

  • Chart Module preference to force a user to be a provider to sign a note. Chart Module Preferences
  • Log in added to Ortho Chart. Ortho Chart
  • Tooth Chart button to clear movements for selected teeth only. Tooth Movements
  • 3D Tooth Chart rewritten from scratch to use DirectX 11 instead of DirectX 9. This makes installation easier and it works better.

eServices

  • Patients can add appointment to their calendar when office is using eConfirmations and Automated Thank-Yous. Automated Thank-Yous
  • eReminders and eConfirmations can send using the patient's preferred language. eConfirmations, , eReminders
  • Web Sched New Patient allows for restricting appointment types to specific blockouts. Web Sched New Patient
  • Automated thank-you messages can send a when a patient schedules an appointment. Automated Thank-Yous

Images Module

  • Ability to merge Image Categories. Merge Image Categories
  • Image mounts for radiograph and photo collages. Imaging Module, Mount Definitions

Manage Module

  • Accounting report for Profit and Loss. Accounting Reports
  • Updates to Accounting Balance Sheet. Accounting Reports

Reports

  • Ability to print page range in Audit Trail and User Query windows. User Query
  • Report for overpaid payment plans. Payment Plans Overcharged Report

Security

  • Ability to copy a user. Security
  • Audit trail entry when merging discount plans. Audit Trail Permissions, Merge Discount Plans
  • Log off time user override moved to Security. User Edit
  • Patient level audit trail for insurance plans. Insurance Plan
  • New permissions for editing completed procedures. Permissions, Audit Trail Permissions

Miscellaneous

  • EDS Medical added as option for medical clearinghouse. Electronic Dental Services - Medical
  • Updated PreXion Bridge and added Prexion Aquire. PreXion Bridge, Prexion Acquire Bridge
  • Enhancements to appearance of Show Features. Show Features
  • Showing inactive patients in the Patient Select window defaults to on. Select Patient, Miscellaneous Setup
  • Prompt before closing Open Dental. Log On
  • Changes to appearance of Edit Appointment Window. Edit Appointment
  • Slow Query Log added for troubleshooting. Slow Query Log Tool
  • Paste a file when using DentalXChange Attachment Service. DentalXChange Attachment
  • Additional information added to the employee list. Employees
  • Language translations for sheets. Edit Sheet Def, Edit Patient Information
