Version 7.5

See Versions

Version 7.5 was released on 12/28/2010

During the upgrade process, you will be presented with a choice. It's good to read and understand this ahead of time, before the upgrade. Anyone who upgraded to 7.5 prior to 7.5.17 will not have seen this option. Their conversion will have combined plans.

Insurance Subscribers: Multiple subscribers can be attached to a single insurance plan. This will be the foundation for other exciting features. Currently, this change improves speed in offices that had thousands of subscribers with identical plans. InsPlan.

Outstanding Claims Report: Instead of being a static printout, it is now an interactive window with a large scrolling grid and tools for managing outstanding claims. Outstanding Insurance Claims Report.

Treatment Finder Report: Instead of being a static printout, it is now an interactive window with a large scrolling grid and tools for managing incomplete treatment. Treatment Finder.

Minor Changes
Security permission added to prevent image deletion.
Warns if new patient matches a patient already added.
