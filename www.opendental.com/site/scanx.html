Scan X

See Sensor Compatibility

Air Techniques ScanX works with Open Dental version 21.3.29 or later.

Note: In our testing, the ScanX has issues when switching Windows users. We have had the scanner show as not present. A customer has separately reported that Open Dental can lock up and sometimes lose an image. We recommend not switching Windows users on the ScanX machine. Consider setting up a Windows user for that machine that is set with a longer timeout before it logs out of Windows. Also consider setting that user to have less network permissions and a password that everyone knows, so even if it times out, users would still go back into the same Windows user. Logging into Open Dental would still involve separate Open Dental users.

Utilize a Mount with Flip on Acquire enabled, otherwise images must be flipped manually after scanning.

  1. In Imaging Devices, add a Device.
  2. If ScanX is already properly installed, select ScanX as the Twain Name.
  3. Check Show Twain UI.
  4. In the Imaging Module, in the top Toolbar, create a new Mount. Click Acquire.

  5. Once the TWAIN window comes up, the ScanX is able to accept films. If the ScanX won't engage and drag the film in, then it means the TWAIN window is probably not showing. The scans show in the TWAIN window slightly delayed from real time.

  6. Once no more films are detected, the VistaEasyView software shows.
  7. Click the round button at the upper left to save all marked images. Then, click Close at the bottom of the ScanX-TWAIN Interface window.
