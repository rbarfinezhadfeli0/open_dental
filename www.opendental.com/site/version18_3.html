Version 18.3

See Versions

Version 18.3 was released on 12/10/2018. EHR certified 2014 edition.

Overview of Version 18.3.

Major Changes

  • Canada electronic claim integration with TELUS network. Claimstream
  • Insurance plan level substitution codes. Estimate Downgrades
  • Allow Recurring Charges to run more than once per month. Authorize Recurring Charges

Changes by Category

Appointments

  • Overlapping appointments indicator. Edit Appointment
  • Appointment search by blockout type. Pinboard

Clinics

  • Clinic filters added to ERA and Appointments for windows. Patient Appointments, ERA Match with Claim
  • Restrict users to only view procedure code fees from their clinic. Procedure Codes
  • Preference added to print statements alphabetically instead of by clinic. Manage Module Preferences
  • Speed enhancement made to the recall list when using clinics. Recall List

eServices

  • Mobile Web show blockouts. Mobile Web Operatories, Blockouts
  • Alert added for when the Open Dental Service is not running. Alerts

Insurance

  • Account Module Preference to Exclude 'None' as an option on Custom Tracking Status for custom claim tracking. Account Module Preferences
  • New ERA matching logic. Eras will now better match to claims. ERA Match with Claim
  • Reportable Insurance Filing Code definition category for use with the insurance filing code window and edit insurance plan window. Insurance Filing Codes, Definitions: Insurance Filing Code Group
  • Preference to disallow write-offs greater than the procedure fee. Account Module Preferences
  • Insurance plan level substitution codes. Estimate Downgrades
  • New popup allows option to automatically change from hold to waiting to send when the primary claim is received. Secondary Insurance
  • Indicator added for ERA manual claim selection. ERA Match with Claim
  • Canada electronic claim integration with TELUS network. Claimstream
  • Ability to change width and height of printed claim forms. Claim Forms
  • Ins Fee column in claim edit renamed to Billed to Ins. Claim

Payments

  • $0 dollar entries now allowed in both the Account module and Manage module, Accounting. Account Module, Accounting
  • Account Module Preference, Enforce Valid Paysplits-Enforce Fully now requires payment credits on payment plans to be attached to procedures or have an unearned type. Account Module Preferences
  • Allow Recurring Charges to run more than once per month. Authorize Recurring Charges

Reports

  • Unearned patient income column added to the provider, monthly, and annual production and income reports. Production and Income Reports
  • Referral analysis report now determines new patients based on the patient's first completed procedure date. Referral Analysis Report

Prescriptions

  • Import/Export medication lists. Medications List
  • Added patient instruction field to prescriptions. Includes new RxInstruction sheet type and automation trigger. Rx / Prescription, Rx Instructions Layout, Automation

Security

  • Archived Patient Edit permission to lock the Edit Patient Information window for archived patients. Permissions, Edit Patient Information
  • Add Audit trail for changes to fees when using copy function. Permissions

Bridge Enhancements

  • Program link added for Panda Perio Advanced. PANDA Perio (advanced) Bridge
  • Program link added for Midway Dental Supply. Program Links
  • Additional replacement fields for custom bridges. Custom Bridges

Miscellaneous

  • Added referral.address, referral.cityStateZip, referral.FLName, and referral.LName Output Text Fields to routing slips. Output Text Field Names
  • Improvements made to the DoseSpot registration form. DoseSpot Setup
  • Improvements made to the Previous Versions web page. Previous Versions
  • Option to alphabetize definition categories. Definitions
  • Tasklist search and 'My Inbox' option added to Add Task window. Task Window
  • New form available for special conversion requests. https://opendental.com/site/conversions.html
  • Statements use sheets preference removed. Statements will use sheets by default. Statement Layout
  • Time card daily calculations now exclude the current day when it falls in the same pay period being calculated. Manage Time Cards, TimeCard
  • Option to run aging for accounts with charges posted by repeating charge tool. Repeating Charges
  • Autonotes within autonotes. Auto Note Setup
  • Sorting options added to quick paste notes. Quick Paste Notes Setup
  • Quickpaste notes pull from multiple categories. Quick Paste Notes Setup
  • Added Date Bal Began and Days Bal Began to the available display fields for the Transworld A/R Manager sent and unsent grids. TSI Collections
  • HTML email templates. Email Templates
  • Additional database maintenance methods added at the patient level. Database Maintenance for Patient
  • Preference to disable saving attachments to the Image Module when using the DentalXChange Attachment Service. DentalXChange ClaimConnect E-Claims
