Overview of Version 22.1.

Major Changes

Create perio charts with default probe measurements

  • Quickly create a default perio chart. Insert the default when charting for a patient, then make changes to the patient as needed. Perio Chart

eClipboard enhancements

  • Set frequencies for images on eClipboard. eClipboard Setup, eClipboard Images
  • Updating sheets prompts user when sheet is in use with eClipboard. Edit Sheet Def
  • eClipboard confirmation warning when a patient is loaded to a device in use. eClipboard Troubleshooting
