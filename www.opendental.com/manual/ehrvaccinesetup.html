EHR Vaccine Setup

In EHR, click Vaccine Def.

EHR Vaccine Data must be defined before vaccines can be entered for patients.

Note: Before vaccines can be defined, you must import CVX codes (see Importing Code Systems) and set up EHR Vaccine Drug Manufacturers.


Click Add, or double click an existing vaccine to edit.

Enter the vaccine information.

  1. Click [...] to select a CVX code.
  2. Enter the Vaccine Name.
  3. Click the drop down arrow to select the EHR Vaccine Drug Manufacturers.
  4. Click OK to save.
