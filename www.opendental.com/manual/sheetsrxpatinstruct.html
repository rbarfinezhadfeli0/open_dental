Rx Instructions Layout

Rx Instructions are a type of sheet that can be used to provide prescription instructions intended for the patient.

In Sheets, double-click on a custom sheet with the type of RxInstruction.

Alternatively, in Sheets:

  • Click New and choose RxInstruction as the Sheet Type to create a new RxInstruction sheet from blank.
  • Select an existing Custom RxInstruction sheet and click Duplicate to create a copy of an existing RxInstruction sheet.
  • Select the existing Internal RxInstruction sheet and click Copy to create a copy of the default RxInstruction sheet that can be edited.

Also see: Sheet Def Properties and Sheet Field Types.

To customize a RxInstruction sheet type, see Edit Sheet Def.

Specific instructions for individual prescriptions are entered in the Patient Instructions field of the Edit Rx window.To print the sheet for a patient click the Pat Instr from this window.

The sheet can also be printed automatically with the PrintRxInstruction action using Automation.

Note: If multiple custom RxInstruction sheets exist, the top-most (i.e., first alphabetically) sheet will be used when printing Patient Instructions.
