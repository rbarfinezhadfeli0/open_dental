Definitions: Proc Code Categories

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Proc Code Categories.

![](images/definitionsProcedureCodeCats.png)

Proc Code Categories determine the category options available to group the list of [Procedure Codes](procedurecodes.html).

Assign procedure codes to categories in the [Edit Procedure Code Window](procedurecodeedit.html).

Double-click an existing category to edit or click **Add** to create a new category. The following Edit Definition window opens.

![](images/definitionEditProcedureCodeCats.png)

**Hidden**: Check to hide this category in the Procedure Codes list. This also hides any procedures assigned to the category.

**Name**: Enter a description for the category

Click **Save** to keep changes.
