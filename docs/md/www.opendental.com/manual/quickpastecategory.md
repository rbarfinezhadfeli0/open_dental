Quick Paste Category

Organize Quick Paste Notes of similar type by category and set which areas notes are available for use.

In [Quick Paste Notes Setup](quickpastenotessetup.html), click **Add**.

![](images/quickPasteCatEdit.png)

Alternatively, double-click an existing category to edit.

Each category is assigned a default text box type that supports quick notes and only note shortcut abbreviations in this category work in those text boxes.

**Description**: The name of the category.

**Default for Types**: The text box type the category's note shortcuts can be used with. Multiple types can be assigned to a single category. Ctrl + click to select multiple types.

* **None**: Do not use. This is only the default when adding a new category.
* **Procedure**: [Procedure Code](procedurecodeedit.html), Completed Note and TP'd Note fields; [Procedure Info](procedureedit.html), Note and E-Claim Note fields; [Procedure Group Note](procgroupnote.html), Note field [Perio Chart](perio.html) Exam Notes field.
* **Appointment**: Note field in [Edit Appointment](aptedit.html).
* **CommLog**: Note field in [Commlog](commlog.html).
* **Adjustment**: Note field in [Edit Adjustment](adjustments.html).
* **Claim**: Claim Note field in [Edit Claim - General Tab](claimtabgeneral.html).
* **Email**: Works in [Email](email.html) fields, body of email, and address lines.
* **InsPlan**: Works in the Plan Note and Note field in [Edit Insurance Plan](insplan.html).
* **Letter**: Obsolete. Used in the old letter merge feature. Use [Sheets](sheets.html) instead.
* **MedicalSummary**: Medical Summary field in the [Medical](medical.html) window, accessed from the Chart Module.
* **ServiceNotes**: Service Notes field in the Medical window, accessed from the Chart Module.
* **MedicalHistory**: Medical History field in the Medical window, accessed from the Chart Module.
* **MedicationEdit**: Note field in the [Edit Medication](medicationmaster.html) window.
* **MedicationPat**: Notes for this Patient field in [Medication for Patient](medicationspatient.html).
* **PatAddressNote**: Address and Phone Note field in [Edit Patient Information](patientedit.html).
* **Payment**: Note field in the [Patient Payment](payment.html) window.
* **PayPlan**: Note field in [Payment Plan](paymentplandynamic.html), [Insurance Payment Plan](paymentplanins.html), and [Old Payment Plan](paymentplanpatient.html).
* **Query**: Query Text field in [Query Favorites](queryfavorites.html).
* **Referral**: Notes and Display Notes fields in [Edit Referral](referralsetup.html).
* **Rx**: Notes field and Patient Instructions field in [Edit Rx](rx.html).
* **FinancialNotes**: In the [Account Module](account.html), both the Fam Urgent Fin Note and the Family Financial note fields.
* **ChartTreatment**: In the [Chart Module](chart.html), the odontogram notes field, normally under the 3D tooth chart.
* **MedicalUrgent**: Medical Urgent field in the Medical window, accessed from the Chart Module.
* **Statement**: Works in the Account Module, [Statement](statement.html), More Options in the Note and Bold Note fields. Works in the Manage Module, [Billing](billing.html), General Note field and [Billing Defaults](billingdefaults.html), General Message, Body, Invoice Note, and SMS Statement fields. (Does not work in the Dunning message fields)
* **Recall**: [Edit Recall](recall.html) Administrative Note field.
* **Popup**: Select patient, click [Popups](popups.html) button, click add or edit existing. Popup Message field only.
* **TxtMsg**: Text Message field inside of the [Text Message](textmessages.html) window.
* **Task**: [Task Window](taskswindow.html), both the Description and Note fields.
* **Schedule**: [Edit Schedule](scheduleeditdaywindow.html), Note field only.
* **TreatPlan**: [Saved Treatment Plan](treatmentplanedit.html), Note field only.
* **ClaimCustomTrack**: [Edit Claim - Status History Tab](claimtabstatus.html), Custom Tracking Status Update (add or edit a status), Note field only.
* **AutoNotePrompt**: Used in the Text Prompt window and Prompt preview for [Auto Notes](autonotes.html).
* **ReadOnly**: Not in use.
* **Lab**: [Edit Lab Case](labcaseedit.html), Instructions field.
* **Equipment**: [Equipment](equipment.html). Add or edit equipment window, works in the Status field only.
* **Etrans834Import**: [Import Ins Plan 834](importinsplans834.html), import path text box.
* **InCaseOfEmergency**: Not in use.
* **ProgramLink**: [Program Link Output File](programlinkoutputfile.html), template textbox.
* **Sheet**: [Fill Sheet](sheetsfillout.html) window, any editable text fields (i.e., Static Text or Input Fields).

Click **Save** to exit the window and keep changes.
