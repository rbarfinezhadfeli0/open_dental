Dental School Courses

Each course tracked in Open Dental can be added to the Dental School Courses list.

In the [Main Menu](mainmenu.html), click Lists, Dental School Courses.

![](images/dentalCourses.png)

Only available when the [Dental Schools](dentalschools.html) feature is enabled in [Show Features](showfeatures.html). Courses must be defined before [Dental Student Evaluations](dsevaluations.html) can be created.

Courses are sorted by *Course ID*. Alphabetically first, then ascending numerically .

Click **Add** to create new course or double-click an existing course to edit.

![](images/dentalCoursesAdd.png)

* **Course ID**: Enter the identification code for the course.
* **Description**: Enter the course name or title.
* **Delete**: Remove the course from the Courses list.

Click **Save** to finish adding a new course or keep changes.
