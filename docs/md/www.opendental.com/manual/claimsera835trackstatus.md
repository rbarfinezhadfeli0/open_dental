Track ERA

Use Custom Track Statuses to make notes on ERAs that may need additional follow up.

In an [ERA](era.html), in the Claims Paid grid, right-click a problem claim, and select **Add Tracking Status**.

![](images/ERATrackingStatus.png)

If a claim on [ERAs](claimsERA835.html) has special circumstances, users can add the claim to the [Outstanding Insurance Claims Report](reportoutins.html) with a custom tracking status and error code.

Note: If a matching claim is not found, users are prompted to manually locate it. See [ERA Match with Claim](claimsERAmatching.html).

**Custom Track Status**: Select the status. Customize status options in [Definitions: Claim Custom Tracking](definitionsclaimcustomtracking.html).

**Error Code**. Select the error code. Customize error code options in [Definitions: Claim Error Code](definitionsclaimerrorcode.html).

**Notes**: Add any related information.

Click **Save** to add the tracking status and additional information to the claim. The claim now appears on the Outstanding Insurance Claims report and is detached from the ERA.
