Sheet Special Field

Special fields are Sheet Field Types used in Treatment Plan Layout, Referral Letter Layout, Chart Layout, and Patient Dashboard.

In [Edit Sheet Def](sheetsetup.html), click **Special**.

![](images/sheetsTreatmentPlanSpecial.gif)

Alternatively, in Sheet Def Edit double-click an existing Special field in the preview area or Fields list to edit.

**FieldName**: There mulitple options depending on which sheet a user is editing. See below for examples of each field.

**X and Y Pos**: Set the location of the upper left corner of the image in relation to the X and Y axis.

* X = horizontal.
* Y = vertical.

**Width / Height**: The width and height of the image.The tooth chart will always have a fixed width to height ratio. Text size may not change even if the size of the field is increased.

## Tooth Chart

The Tooth Chart fields can be added to the [Treatment Plan Layout](sheetstreatplan.html), [Referral Letter Layout](sheetsreferrals.html), [Chart Layout](layout.html), and [Patient Dashboard](patientdashboard.html).

* For Treatment Plan Sheets, the tooth chart displays all procedures.
* For Referral Letters, the tooth chart only displays procedures completed the same day the letter is generated.
