Payment Plan Templates

Set up Payment Plan Templates to quickly determine payment plan terms.

In the [Main Menu](mainmenu.html), click Setup, Account, Pay Plan Template.

![](images/payPlanTemplates.png)

Payment Plan Templates are used when creating [Payment Plans](paymentplandynamic.html) from Open Dental or from within [ODTouch Payment Plans](odtouchpayplan.html).

Filters:

* **Clinic**: Filter templates by clinic.
* **Show Hidden**: Display templates marked as *Hidden*.

Click **Add** to create a new template, or double-click an existing template to edit.

![](images/payPlanTemplateEdit.png)

**Template Name**: Name the template.

**Clinic**: Select the clinic the template applies to.

Set the terms for **Down Payment**, **APR**, **Payment Amount** or **Number of Payments**, **Charge Frequency**, and **Handle Treatment Planned**. For details on these terms, see [Payment Plan](paymentplandynamic.html).

Click **Save** to close the window and retain changes to the template. Check **Hidden** if the template is no longer in use.
