Shutdown Workstations

Use this tool to shut down the Open Dental program from any workstation that is running.

In the [Main Menu](mainmenu.html), click Tools, Misc Tools, **Shutdown All Workstations**.

![](images/shutdownWorkstations.png)

* **Shutdown Selected**: Highlight one or more workstations, then click to shut them down.
* **Shutdown All**: Shutdown all workstations.

Once an option is selected, the following prompt appears. This gives users on the selected workstations 15 seconds to save any changes before the program closes.

![](images/shutdownWarning.png)

Click **OK** to continue.
