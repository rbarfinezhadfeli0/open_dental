ODMobile

ODMobile (formerly Open Dental Mobile) is an app installed to a user's mobile device.

In [eServices Setup](eservicessetup.html), click **Signup**. In the Basics tab, enable *ODMobile*.

![](images/MobileSignup.png)

Note: Fees displayed on the screenshot above may not be current. Refer to [Fees for Support and Services](fees.html) for the most up-to-date pricing.

## Set Up ODMobile in Open Dental

1. Sign up for ODMobile in the [eServices Signup](eservicessignup.html) Portal.
2. Set up [Mobile Settings](mobilesettings.html) to use when logging into the app. This only needs done once and is used for all Open Dental apps.
3. Set up [Users](securityusers.html) to log into ODMobile.
   * Users must have the *ODTouch/ODMobile* security permission.
   * *Passwords must be strong* must be enabled in [Global Security Settings](securitysettings.html).
   * Users must have a strong password.

## Install Open Dental on a Mobile Device

ODMobile is designed for phones, not tablets.

1. Install the app to each desired device. ODMobile is available for iOS and Android.
   * [App Store](https://apps.apple.com/us/app/open-dental/id1481881571?ls=1)
   * [Google Play](https://play.google.com/store/apps/details?id=com.opendental.opendental)
2. In the app, enter the Mobile Settings to connect the app with the database.
   ![](images/odmobileSignupLogin.png)
3. Log into the app using an Open Dental username and password.
   ![](images/odmobileUserLogin.png)
