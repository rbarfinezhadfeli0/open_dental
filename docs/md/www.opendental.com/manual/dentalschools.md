Dental Schools

Open Dental works well in dental schools or dental hygiene schools.

## Setup

Dental Schools must be set up in the following order.

1. Enable Dental Schools in [Show Features](showfeatures.html).
2. Create [Dental School Classes](dsclasses.html).
3. Create [Dental School Courses](dscourses.html).
4. Create User Groups for students and instructors in [Dental School Security](dssecurity.html). Assign permissions to each group.
5. In [Dental School Setup](dsusergroupdefaults.html):
   * Assign the User Group defaults.
   * Set up [Dental School Grading Scales](dsgradingscale.html).
   * Set up [Dental School Evaluations](dsevaluationsetup.html).
6. In [Dental School Providers](dsproviders.html):
   * Add [Dental School Instructors](dsinstructors.html).
   * Add [Dental School Students](dsstudents.html).
7. Add, copy, and manage course [Requirements Needed](requirements.html).

Dental Schools should have their IT team set up the server and workstations with Open Dental. Also see [Dental School Servers](dentalserverreq.html).

## Using Dental Schools

* Create [Dental Student Evaluations](dsevaluations.html).
* [Fill out Student Evaluations](dsfillevaluation.html).
* Generate [Dental Student Reports](dsreports.html) as needed.
