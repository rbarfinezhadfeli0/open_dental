EHR Setup Window

In the [Main Menu](mainmenu.html), click Setup, [Chart](chart.html), EHR.

![](images/ehrSetup.png)

Before entering patient data for [EHR](../site/ehrabout.html), users must define information so that it is available for selection. Several setup areas are accessed from the EHR setup window.

Note: This window is not available for Open Dental Cloud users.

**Settings**: Define global meaningful use stage settings, severity levels for drug interaction alerts, and default encounter and pregnancy codes used in CQMs.

**Allergies**: Access the [Allergy List](allergiesmaster.html).

**Vaccine Def**: Open [EHR Vaccine Setup](ehrvaccinesetup.html) to define any vaccines administered by the practice.

**Drug Manufacturers**: Set up [EHR Vaccine Drug Manufacturers](ehrvaccinedrugman.html) for the vaccines administered at the practice.

**Drug Units**: Set up [EHR Vaccine Drug Units](ehrvaccinedrugunits.html) for the vaccines administered at the practice.

**Reminder Rules**: Set up rules for patient reminders. Rules are not required to document that a reminder has been sent.

**Inbound Email**: Define [Email Setup](emailsetup.html).

**Educational Resources**: Define [Educational Resources](../site/ehredresources.html) that will be triggered for patients based on specific conditions.

**Patient Portal**: Set up the [Patient Portal Feature](../site/patientportal.html).

**Provider Keys**: Enter [EHR Annual Provider Keys](ehrproviderkeys.html).

**Code Systems**: Download and view coding systems that are attached to patient data.

* **Code System Importer**: Download a coding system. See [Importing Code Systems](codesystemimport.html).
* **ICD9s**: View or search ICD-9 or [ICD-10 Codes](icdcodes.html) that have been imported.
* **RxNorms**: View or search [RxNorms](rxnorms.html) that have been imported.
* **SNOMED CTs**: View or search [SNOMED CT Codes](snomedct.html) that have been imported.
* **LOINCs**: View or search [LOINC Codes](loinc.html) that have been imported.

**CDS Triggers**: Configure Clinical Decision Support (CDS) interventions and permissions by user. See [EHR Configure CDS Rules](ehrcdsconfigure.html) and [EHR CDS Permissions](ehrcdspermissions.html).

**Time Synchronization**: View the [EHR NIST Time Synchronization](ehrnistsync.html) window.

**Internal OID Registry**: Set up [EHR Object Identifiers (OIDs)](ehroid.html).

**Emergency Now**: Turn on/off temporary [EHR Emergency](ehremergency.html) access.
