Public Health

The public health feature allows additional features and fields to display in Open Dental.

In [Show Features](showfeatures.html), check *Public Health* to enable features.

When enabled, the following features and fields are available:

* [Edit Patient Information: Public Health Tab](patienteditpublichealth.html)
* [Site List](sites.html)
* [Edit Treatment Plan](treatmentplanedit.html), *Responsible Party* field
* [Import Ins Plan 834](importinsplans834.html)
* [Public Health Screening](publichealthscreenings.html)
* [UDS Report](reportsuds.html)
* [Raw Screening Data Report](reportsrawscreeningdata.html)
* [Raw Population Data Report](reportsrawpopulationdata.html)
* [Counties](counties.html)
* [Practice Setup](practice.html), *Default Proc Place Service* field
