Standard Reports

Standard reports allow a user to track and view data.

In the [Main Menu](mainmenu.html), click [Reports](reports.html), Standard.

![](images/reports.png)

**Setup**: Click to set up and customize defaults and user access for standard report. Options include:

* [Display Settings](reportdisplaysettings.html): Customize which reports list on the window.
* [Security Permissions](reportsecurity.html): Restrict access to reports by user group.
* [Report Server](reportserver.html): Set up a report server to run Production and Income Reports. This can be useful for large offices to prevent lockups and slowness in a live database.
* [Misc Settings](reportmiscsettings.html): Set report defaults.

**User Query**: Open the [User Query](queryoverview.html) window to run custom reports.

**Laser Labels**: Print [Laser Labels](laserlabels.html) for a selected list of patients or insurance companies.

**Production and Income Reports**

* Today: Generate a report for today's date using default settings.
* Yesterday: Generate a report for yesterday using default settings.

  Note: If run on a Monday the report is for the previous Friday. If work was done over the weekend, use *More Options* instead.
* This Month: Generate a report the current month (1 - 31) using default settings.
* Last Month: Generate a report for last month (1 - 31) using default settings.
* This Year: Generate a report for the current year (Jan 1 - Dec 31) using default settings.
* More Options: Open the [Production and Income Report](reportprodinc.html) window to customize report options.
