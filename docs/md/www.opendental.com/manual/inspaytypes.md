Definitions: Insurance Payment Types

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Insurance Payment Types.

![](images/definitionsInsPaymentTypes.png)

The Insurance Payment Types definition determines the payment type options available when [Finalizing Insurance Payments](claimpayfinalize.html).

Also set which payment types to exclude when creating a [Deposit Slip](depositslip.html). The [Daily Payments Report](reportdailypayments.html) is grouped by insurance payment type and can optionally be filtered by type.

Double-click an existing item or click **Add** to create a new item. The Edit Definition window opens.

![](images/definitionEditInsPaymentTypes.png)

**Name**: Enter the payment type.

**N=Not selected for deposit**: Enter N to exclude this type from deposit slips or leave blank to include this type.

**Hidden**: Check to hide this item.

**Delete**: Click to delete this item. If the item has been attached to an insurance payment it cannot be deleted.
