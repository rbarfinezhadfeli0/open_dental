Definitions: Fee Colors

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Fee Colors.

![](images/definitionsFeeColors.png)

The Fee Color definition determines the text color of global fees, provider-specific fees, clinic-specific fees, and provider/clinic-specific fees in the [Procedure Codes](procedurecodes.html) window.

To edit a definition color double-click an item to open the Edit Definition window.

![](images/definitionEditFeeColor.png)

Click the colored block to select a [Color](colorselection.html) then click OK to save. The item name cannot be changed or hidden.
