EHR CDS Interventions

See [EHR Dashboard](ehrdashboard1.html).

CDS interventions provide evidence-based clinical guidance when a user is making a clinical decision. If a user has the appropriate [EHR CDS Permissions](ehrcdspermissions.html) and enters data for a patient that triggers a [CDS Rule](ehrcdsconfigure.html), an intervention will pop up on-screen.

![](images/ehrcdsintervention.gif)

Each intervention identifies the conditions that triggered it, as well as instructions and bibliographical information.

![](images/infobutton.gif) Info button: Shows when a user has the *Show i* CDS Permission. Click this button to access additional clinical information via [MedLinePlus](http://www.nlm.nih.gov/medlineplus/). See [EHR InfoButton](ehrinfobutton.html).

Click OK to close the intervention.

Click **Cancel Current Action** to cancel the addition of the data that triggered the intervention. This option is not available for all interventions.
