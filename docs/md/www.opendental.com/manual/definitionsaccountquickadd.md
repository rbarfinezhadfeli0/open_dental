Definitions: Account Procs Quick Add

Use Account Procs Quick Add definitions to customize the list available Quick Procs in the Account Module.

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Account Procs Quick Add.

![](images/definitionsAccountProcsQuickAdd.png)

The [Quick Procs](quickprocs.html) dropdown in the Account Module is empty by default. Add items in definitions to build the dropdown list as needed.

Double-click an existing item to edit or click **Add** to create a new item. The Edit Definition window opens.
![](images/definitionEditProcQuickAdd.png)

* **Hidden**: Check to hide the item in the Quick Procs dropdown in the Account Module. The item is still available in Definitions.
* **Name**: Item description. This identifies the item in the Quick Procs dropdown. When procedures are added to the patient's account, [Procedure Code](procedurecodeedit.html) descriptions are used.
* **Procedure Codes**: Enter the procedure code or group of codes associated with the item. To enter more than one code, separate each code with a comma and do not add spaces (e.g., D0140,D0220). This field is case-sensitive, so enter the code as it appears in the [Procedure Codes](procedurecodes.html) list.

Click **Save** to exit the window and keep changes.

**Delete**: Permanently remove this item from Definitions and the Quick Procs dropdown.
