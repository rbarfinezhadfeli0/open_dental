# Version 5.9

See [Versions](versions.html)

Version 5.9 Released on 9/5/08.

Install [dotNet 3.5](dotnet.html) if you have time, or a few of the new features might not work for you.

**Feature Request Voting:** Users are able to request new features from an interface within Open Dental. Users can track their request history, search requests that others have made, and vote for their favorites. Users can add discussion entries to pending feature requests. [Feature Requests](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7featurerequests.html).

**Prescriptions:** Added Controlled Substance field which handles display of DEA# and automatic printer selection. Moved prescriptions over to sheets framework to allow full customization. This allows unlimited flexibility in Rx layout, and permits compliance with any local laws. [Rx Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7rxsetup.html).

**Sheets:** Added rectangles, lines, and checkboxes. [Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheets.html).

**UAppoint integration:** Open Dental keeps the UAppoint server synched with any changes in patients, recall, and appointments. Customers using the UAppoint service will be able to let patients request specific appointment times based off of this synched data. The office receptionist would receive the appointment request by email and manually schedule it.

Insplans can be hidden.
 Ins Carriers can be hidden.
 Carrier list interface enhanced with search, reselection, and plan count.
 Palmer notation tooth numbering.
 Patient select window is now resizable.
 Appointment time locked checkbox gets checked automatically when a user manually alters the appointment time.
 Appointment can be set complete from the Chart module.
 Users are no longer allowed to set procedures complete from the Chart module without turning on that option.
 Can edit the dates of multiple procedures simultaneously.
 Payments can now be split between regular accounts an payment plans without malfunction.
