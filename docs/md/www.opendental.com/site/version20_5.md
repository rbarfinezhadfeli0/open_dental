# Version 20.5

See [Versions](versions.html)

Version 20.5 was released on 03/11/2021.

![](images/youtubeLogo.png) Overview of [Version 20.5](https://youtu.be/f_9pbd2JWhI)

## Major Changes

**Paint Types**: Created new paint types for more tooth chart customization. [Procedure Code](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodeedit.html)

* New paint types created for *Space Maintainers* and *Retained Root*.
* Paint type Watch changed to Text

**Appointment Module**: New buttons to jump appointment calendar 3, 4 or 6 months. [Appointments Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html)

**Discount Plans**: Added effective dates for Discount Plans.

* Added subscriber level effective dates for Discount Plans. [Discount Plan Subscriber Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7discountplan.html)
* Effective date included in Discount Plan Report. [Discount Plans Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportdiscountplan.html)

**Procedure Codes**: New 2021 COVID-19 CDT Codes. [Procedure Code Tools](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodetools.html)
