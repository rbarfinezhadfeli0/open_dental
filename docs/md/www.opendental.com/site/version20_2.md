# Version 20.2

See [Versions](versions.html)

Version 20.2 was released on 07/20/2020.

This version is notably smaller than normal. Most of 20.2 was developed during COVID-19. As a result, our team focused more on larger features that would benefit our customers during this time.

**eServices**

* HTML Raw editing available for eService emails. [HTML Email](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailhtml.html)
* Changed navigation from a dropdown menu to a list for eServices Setup. [eServices Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eservicessetup.html)
* Send mass emails. [Mass Email Feature](massemail.html)
* Open Dental Mobile. [ODMobile](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7opendentalmobile.html) (Also available in 20.1)
* Arrival messages makes it easier to check patients in from their car. [Arrivals](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7arrivalssetup.html) (Also available in 20.1)

**Security**

* Date limitation added to Payment Create permission. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
* Added security setting to allow users to edit their own time card in the current pay period. [Global Security Settings](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securitysettings.html)
* Added security setting for clinic users to maintain selected patient when logging off. [Global Security Settings](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securitysettings.html)

**Miscellaneous**

* Quickly organize fee schedules by selecting its placement in the list. [Fee Schedules](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7feeschedules.html)
* Added shortcut to open the wiki. [Wiki](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7wiki.html)
* Updates to graphic preferences. [Graphics Preferences](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7graphics.html)
* Automatically save patient forms to the Imaging Module. [Fill Sheet](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetsfillout.html), [Definitions: Image Categories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7definitionsimagecat.html)
* Double-click a provider or employee to quickly view their schedule. [Schedule Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7schedule.html)
