# Version 7.7

See [Versions](versions.html)

Version 7.7 was released on 3/1/2011. It was decided on 3/15/2011 to discontinue release of 7.7 due to bugs in MySQL. The stable version then became 7.6 instead of 7.7. Future upgrades will skip 7.7 and jump straight to 7.8.

Version 7.7 included the following new features:

**Oracle:** Some support for Oracle database in addition to MySQL database. Most reports not functional. Aging not functional.

**Tasks:** Can now have multiple dated notes attached, can be sent more easily to other task lists, unread status can be tracked by user. [Task List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7tasksarea.html).
