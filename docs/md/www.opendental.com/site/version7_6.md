# Version 7.6

See [Versions](versions.html)

Version 7.6 was released on 1/25/2011.

**Web Forms:**  Easily place registration forms and medical history forms online. After patient fills out form, the information easily flows back to the dental office. [Web Forms](webforms.html).

**Prospective Patients:** Patient status for 'prospective'. Operatory can be set to prospective, and patient status can automatically be set by moving between operatories. [Operatories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7operatories.html).

**Minor Changes**
 Task edit window is now always modal, allowing user to jump back and forth between main program.
 Appointment view is now automatically remembered are reused on a per-workstation basis.
 Xcharge username and password support.
 Patient Select window, search by hyphenated last name.
