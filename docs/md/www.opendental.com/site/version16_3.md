# Version 16.3

See [Versions](versions.html).

Version 16.3 was released on 12/22/2016. EHR certified 2014 edition.

16.3 will not work on workstations running Windows XP or Windows Server 2003.

.NET 4.5. is required. See [.NET](dotnet.html) to upgrade (Windows Vista and 7 users).

## Major Changes

2017 CDT Codes. [CDT Codes](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodetools.html)

eConfirmations: [eReminders and eConfirmations](ereminders.html)

* Automatically send eConfirmations for scheduled appointments via SMS text message and/or email.
* Patients can respond to an eConfirmation via URL link or using a text message code.
* When patient eConfirms an appointment, appointment confirmation status automatically updates.

Documents and Images: Option to store documents and images offsite on the cloud using Dropbox. [Store Images in Dropbox](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7dropbox.html)

Payment Plans

* Option to customize the layout of printed payment plan terms using sheets. [Customize Printed Payment Plans Using Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentplansheets.html)
* Add an electronic signature box to a custom payment plan layout. [Sheets: Signature Boxes](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetssignature.html)
* Payment plan terms that are saved or printed using a custom sheet layout are saved in the Images module. [Sign and Print Payment Plan Terms](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentplanprint.html)
* Designate a payment plan image category in the Images module. [Definitions Payment Plan Categories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7definitionspaymentplancategories.html)
* Choose type of payment plan (patient or insurance plan) from a Payment Plan tool button dropdown. [Account Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7account.html)

Task Reminders:

* Create reminder tasks that appear at a set interval daily, weekly, monthly, or yearly. [Task Reminders](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7taskreminders.html)
* View reminder tasks in their own tab in the Tasks grid. [Task Reminders](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7taskreminders.html)
* Add a reminder task using the dropdown menu on the Tasks button. [Task Reminders](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7taskreminders.html)
* View number of unread and read reminder tasks in the Appointments module. [Appointments Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html)

Web Sched:

* Allow new patients to schedule their first appointment online using Web Sched New Patient Appts. [Web Sched New Patient Appts](websched.html)
* Appointments scheduled by new patients online automatically show in the appointment schedule. [Web Sched New Patient Appts](websched.html)
* Appointments report includes options for appointments scheduled using Web Sched New Patient Appts. [Appointments Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportappts.html)
