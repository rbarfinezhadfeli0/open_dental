Database Management Systems

See [Advanced Topics](advancedtopics.html).

A Database Management System is a system that maintains a database.

Open Dental supports the following database management systems:

* [MariaDB](mariadb.html)
* [MySQL 5.5](mysql55.html) or [MySQL 5.6](mysql56update.html)
