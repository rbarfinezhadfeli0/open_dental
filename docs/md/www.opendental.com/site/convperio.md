Conversions: Perio

See [Use Converted Database](usingodconversion.html).

During a final [Conversion](conversions.html), we default every patient to a 6 month Prophy recall type. Before we deliver the final conversion, we run a script that changes all patients with completed procedures for D4910, D4341, and D4342 in the last three years to a four-month Perio recall.

To view a report of which patients were converted to Perio to go  [User Query Favorites](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7queryfavorites.html). Double-click the *Conv - Perio Patients* query

![](images/convPerioQuery.png)

Change an individual patient's recall in the [Family Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7family.html), [Recall](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recall.html) area.

To change default intervals, see [Recall Types](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recalltypes.html). Be cautious when editing recall types as changes affect the entire recall system.
