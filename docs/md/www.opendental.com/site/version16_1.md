# Version 16.1

See [Versions](versions.html).

Version 16.1 was released on 2/23/2016.

## Major Changes

**Graphic Reports:**  New tool to track the growth of your practice using color coded charts and graphs. [Graphic Reports](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportsgraphic.html)

* Track production and income, new patients, accounts receivable, and broken appointments for the practice and by provider.
* Customize reports. [Graphic Report Setup Mode](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportsgraphicsetup.html)

**Insurance Verification:** New tools to track verification of insurance plans and benefits. [Insurance Verification List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insverifylist.html)

**ERAs:** Automatically download ERAs from ClaimConnect. [Electronic EOBs (ERAs)](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimsera835.html)

**Security:** Users can be associated with both a provider and employee. [Security User Profiles](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securityusers.html)

**Clinics:** Users can be assigned to a default clinic and allowed access to multiple clinics. [Security User Profiles](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securityusers.html)
