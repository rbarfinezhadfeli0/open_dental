# Version 15.1

See [Versions.](versions.html)

Version 15.1 was released on 5/5/2015.

**Clinics:**[Clinics](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7clinics.html)

* New Clinic selection menu in the main menu. [Main Menu](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mainmenu.html)
* The following reports can be filtered by clinic: [Appointments Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportappts.html), [Aging of A/R Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportaging.html)
* Assign appointment views to clinics. [Appointment Views](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointmentviews.html)
* Waiting room list filtered by clinic. [Waiting Room](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7waitingroom.html)
* Employee list filtered by clinic. [Appointments](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html)
* Security enhancements:
  + Associate users to a default clinic or restrict users to certain functionality by clinic.[User Security Profiles](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securityusers.html)
     - Billing. [Billing](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billing.html)
     - Reports. [Standard Reports](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportsstandard.html)
     - View or send claims. [Send or Print Batch Insurance Claims](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimsend.html)
     - Set schedules. [Schedules](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7schedule.html)
     - Time clock. [Time Clock](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timeclock.html)- Manage time cards. [Time Card Manage](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timecardmanage.html)
  + New logic for selecting a patient when the user is restricted by clinic. [Select Patient](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientselect.html)

**Central Enterprise Management Tool:**  Run the annual Production and Income Report from multiple databases at once. [Central Enterprise Management Tool](central.html)

**Web Sched:** New eService allows patients to schedule recall appointments online. Sign up is required and there is a monthly fee. [Web Sched](websched.html)

**E-mail:** Send standard encrypted e-mail. [Standard Encrypted E-mail](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7encryptedstandard.html)

**Patient Portal:** Enhanced user interface improves patient experience using the portal. [Patient Portal: What the Patient Sees](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7portalpatientsees.html)

* Patients can view their payments and statements.
* Patients can change their own password.
* EHR summary of care is automatically sent to the patient portal when a patient is referred to another provider. [Referrals](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7referrals.html)
* Automatic log-off after a period of inactivity.

**Reports:** Daily Payment report can be sorted by insurance payment type. Improved backend for many reports using the complex report system. [Daily Payments Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportdailypayments.html),  [Complex Report System](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportcomplex.html)

**Statements:** Customize statements using Sheets (must be enabled). New features include the ability to remove or add content including aging amounts and payment plans, add payment information fields, customize where practice name and address are located, and customize notes (size, content). [Customize Statements Using Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7statementsheets.html).

**Direct Messaging:** Register public key certificates with Open Dental for Direct messaging. [Public Certificate Hosting](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailcerthost.html)

**HL7:** Technical guide details inbound and outbound message types supported by Open Dental. [Open Dental HL7 Interface Specifications manual](../resources/HL7MessagingInterfaceSpecs.pdf) (PDF)
