# Version 6.5

See [Versions](versions.html)

Version 6.5 Was released on 5/17/09

**Electronic Billing:** Instead of printing statements, upload them to the billing service which prints and mails them. [E-Bills](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billingdefaults.html).

**Conditions:** This is another status for procedures. It allows charting of caries, etc without having to chart planned treatment.

**Preauth Estimate Entry:** When preauthorizations come back, there is a new window to enter the estimates. The numbers will flow into the existing treatment plans.

**Bridge to eClinicalWorks:**[eClinicalWorks](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeeclinicalworks.html).

**Aging Calculation:** nearly instantaneous now.

**Minor Changes:**
 Easy Options window name changed to "Show Features". [Show Features](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7showfeatures.html).
 Medical fields can be hidden. [Show Features](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7showfeatures.html).
 Site box in patient edit window now has dropdown completion as user types. [Sites](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sites.html).
 Responsible party field picker uses family members first. [Public Health](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7publichealth.html).
 Billing "Exclude if insurance pending" now excludes even if claim estimate is zero. [Billing](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billing.html).
 Sheets multiline text rendering overhauled for more accurate layout of line spacing. [Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheets.html).
 Added option to ignore walkout statements when calculating dates for monthly billing. [Billing](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billing.html).
