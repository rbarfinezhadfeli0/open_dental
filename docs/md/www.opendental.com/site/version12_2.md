# Version 12.2

See [Versions](versions.html)

Version 12.2 was released on 6/28/2012

**Claim Fields:** More claim fields added for 5010 and DentiCal: Original Reference Number, CLM01 Claim Identifier, Correction Type, and Date Resent. [Claim](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimedit.html).

**Scanning:** Six more scanning settings, and the settings are per workstation. This should help with some duplex scanning problems as well as speeding up the scanning process. [Imaging Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7imagingsetup.html).

**eCW:** Third integration mode called "full" which opens up nearly all Open Dental features yet still passes all HL7 messages back and forth. [eCW](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeeclinicalworks.html).

**Timecards:** Report for all employees on one page, exportable for payroll service. Timecards print for all employees at once, multiple pages with detail. [Time Clock](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timeclock.html).

**Minor Changes**
 Appointment color override per appointment. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html).
 Certain procedure codes assigned to one provider by default. [Procedure Code Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodeedit.html).
 eCW completed procedures automatically attached to appointments. [eCW](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeeclinicalworks.html).
 QB bridge supports choosing from multiple income accounts when making deposit. [QuickBooks](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7quickbooks.html).
 Security permissions for adding/removing a referral to/from patient. Included in audit trail. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html).
 Claim Send window now has a date column and sorts by date. [Sending and Printing Insurance Claims](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimsend.html).
 Appointments for Patient window, added clinic and provider columns. [Scheduling an Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7apptspatient.html).
 X-Charge, better support for voids and returns. [X-Charge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7xcharge.html).
 Equipment is now searchable by location. [Equipment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7equipment.html).
 BioPAK bridge. [BioPAK](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgebiopak.html).
 RayMage bridge. [RayMage](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeraymage.html).
 New Claim Button Behavior. [Account Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7account.html).
