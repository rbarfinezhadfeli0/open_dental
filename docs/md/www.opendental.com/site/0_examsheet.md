Exam Sheet

Exam Sheets are used to track findings during exams.

In the [Chart Module](0_chart.html) toolbar, click Exam Sheet.

![](images/examSheets.png)

Multiple Exam Sheets can be created and customized using the [Sheets Feature](0_sheets.html). They can be signed electronically, used in [Automation](0_automation.html), and are visible in the [Chart Module](0_chart.html).

## Editing Exam Sheets

Click Add to create a new Exam Sheet. Double-click an existing sheet to edit.

![](images/examSheetFill.png)

Exam Sheets can be filled out electronically, and printed or emailed.
