# Version 19.2

See [Versions](versions.html)

Version 19.2 was released released on 10/02/2019.

![](images/YouTube_PlayButton_15px.png) Overview of [Version 19.2](https://www.youtube.com/embed/zRiPq8Uvu9M).

**Account Module**

* Improvements to Income Transfer logic. [Income Transfer Manager](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7incometransfermanager.html)
* Automatic ACH payment status updates. [PaySimple](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paysimple.html), [Credit Card Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentpatientcreditcard.html)
* Preference to allow prepayments to attach to treatment planned procedures. [Hidden Splits](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7hiddensplits.html), [Account Module Preferences](https://www.opendental.com/manual192/modulesetupaccount.html)
* Preference to apply a procedure's credits to the procedure's charges before aging. [Account Module Preferences](https://www.opendental.com/manual192/modulesetupaccount.html)
* Preference to allow a recurring charge to process when patient has no balance or a credit. [Account Module Preferences](https://www.opendental.com/manual192/modulesetupaccount.html), [Authorize Recurring Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recurringchargesauthorize.html)

**Appointments Module**

* Copying schedule no longer overwrites holidays. [Schedule Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7schedule.html)
* View production per operatory. [Production Totals](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7productiontotals.html)
* Past due option added to appointment views. [Appointment View Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointmentvieweditwindow.html)
* Preference to prevent changes to completed appointments with completed procedures attached. [Appointments Module Preferences](https://www.opendental.com/manual192/modulesetupappts.html)

**Chart Module**

* Emails allow for raw HTML templates. [HTML Email](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailhtml.html)
* New command options for Voice Perio Charting. [Voice Perio Charting](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7periochartingvoice.html)

**eServices**

* eClipboard allows patients to check-in, fill out forms, and patient pictures. [eClipboard Feature](eclipboard.html)
* Web Sched Recall, Web Sched Notify, Web Sched ASAP, and Webmail can now use HTML. [HTML Email](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailhtml.html)
* Mobile Web shows broken appointments. [Mobile Web Operatories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mobilewebuseops.html)
* Added alert when an eService email fails more than three consecutive times. [Alerts](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7alerts.html)
* Ability to delete a WebSched ASAP blockout. [Web Sched ASAP](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7webschedasap.html)

**Insurance and Claims**

* Eligibility requests include insurance history dates. [Insurance History](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7inshistory.html), [Electronic Eligibility and Benefits](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ebenefits.html)
* Ability to automatically import electronic eligibility and benefits. [Carriers](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7carriers.html)
* Custom claim tracking copied to split claims. [Split Claim](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimpaymentsplit.html)
* Better handling of primary and secondary claims that allow procedures marked as *Do Not Bill to Ins.* [Secondary Insurance](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7inssecondary.html)
* Fields added to 1500 claim form for Current Illness, Injury, or Pregnancy. [Edit Claim - Medical Tab](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimmedical.html), [HCFA 1500 Claim Form](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimform1500.html)
* Allow batch attachments through DentalXChange. [DentalXChange Attachment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7dentalxchangeattachmentservice.html)
* Canada carriers with claim history can be combined. [Canada Carriers](canadacarriers.html)

**Treatment Plan Module**

* Additional options for Treatment Plan display fields. [Display Fields](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7displayfields.html), [Treatment Plan Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7treatmentplan.html)
* Visual indicator in Treatment Plan when procedure is attached to an appointment. [Treatment Plan Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7treatmentplan.html)

**Manage Module**

* Excluded tab added to TSI interface. [TSI Excluded Accounts](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7tsiexcluded.html)
* Re-enabled the Archive Tool. [Backup Tool](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7backuptool.html)
* Time card notes now print first two lines. [Manage Time Cards](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timecardmanage.html), [TimeCard](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timeclockedit.html)
* Preference to allow total payments when processing an ERA. [Manage Module Preferences](https://www.opendental.com/manual192/modulesetupmanage.html)
* Ability to filter ERA by the Control ID. [ERAs](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimsera835.html), [Show Features](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7showfeatures.html)
* Reminder tasks are now tracked in the Audit Trail. [Task Reminder](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7taskreminders.html)
* Added the ability to filter task lists by one or more regions or clinics. [Tasks Area](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7tasksarea.html)
* Output text field created for cash payment on deposit slips. [Deposit Slip Layout](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetsdepositslip.html)

* Bridge to Illinois PDMP. [PDMP Bridge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgepdmpillinois.html)

* Patient Database Maintenance includes tabs for Hidden and Old tabs. [Database Maintenance for Patient](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7databasemaintenancepat.html)
* Renamed *Aging is enterprise* preference to be more specific. [Enterprise Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7enterprisesetup.html)
* Fhir - added ability to include procedures when creating appointments. [API Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7fhir.html)
