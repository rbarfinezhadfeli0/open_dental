# Version 14.1

See [Versions](versions.html)

Version 14.1 was released on 3/31/2014.

**EHR:** Enhancements for MU and 2014 certification.
 - New measure calculations for Stage 2 Meaningful Use. [EHR Stage 2 Measure Reports](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrmeasures.html)
 - Three new EHR measure calculations to handle exclusions for vital signs. [Vital Signs](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7vitalsignsenter.html)
 - CPOE medication orders include the optional alternate measure for measure calculation. [CPOE Medication Orders](ehrcpoe.html)
 - Use the Electronic Prescription interface for Computerized Provider Order Entry (CPOE) of medication orders. [Electronic Prescriptions](electronicrx.html)
 - Create CPOE laboratory and radiology orders. [EHR CPOE Radiology and Lab Orders](ehrcpoe.html)
 - Set up Clinical Decision Support interventions based on specific conditions for specific users. [CDS Interventions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrcdsintervention.html)
 - Send and receive secure e-mail to and from patients using the enhanced Patient Portal. [Secure Web Mail](securewebmail.html)
 - Provide patients with the ability to access, download and transmit their health information online using the enhanced Patient Portal. [Patient Portal](patientportal.html)
 - Send and receive encrypted e-mail from one provider to another using the Direct standard. [Direct Messaging](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7direct.html)- Vital signs includes optional exclusions. [Vital Signs](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7vitalsignsenter.html)
 - Export a detailed summary of care CCDA for all patients. [Summary of Care](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrsummaryofcaresend.html)
 - Enhanced medication reconciliation of allergies, medications, and problems. [Medication Reconciliation](ehrmedicationreconcile.html)
 - Record family health history. [Family Health History](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7familyhealthenter.html)
 - Enter data for and report on 2014 Clinical Quality Measures (CQMs). [Clinical Quality Measures](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrcqm.html)
 - Enhanced patient lists. [Patient Lists](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrpatientlistgenerate.html)
 - Import medical laboratory results using HL7 Lab Results Interface (LRI). [Lab Results](ehrcpoe.html)
 - Record electronic, searchable notes in patient records. [Edit Procedures](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedureedit.html)
 - Support for medical codes including SNOMED CT, ICD9, ICD10, RxNorms, CPT, CDCREC, CVX, HCPCS, LOINC, SOP, and UCUM. [Code System Import](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7codesystemimport.html)
 - Support for InfoButton through MedLine Plus. [InfoButton](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrinfobutton.html)
 - Generate electronic syndromic surveillance data (appointment observations) for submission to registries. [Syndromic Surveillance](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrsyndromicdataobs.html)
 - Enhanced entry and export of immunization data. [Vaccines](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrvaccine.html)
 - NIST time sync. [NIST Time Synch](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrnistsync.html)
 - More family relationship options on the Edit Patient Information window. [Family Relationships](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7familyrelationship.html)

**E-mail:** Enhanced to include an inbox. [E-mail Inbox](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailinbox.html)

**Timecard:** Enhanced to correspond with ADP format requirements. Options for differential and overtime hours added. Time card management includes additional features to automate calculations and simplify error fixing. [Time Card Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timecardsetup.html), [Time Card Manage](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timecardmanage.html)

**Minor Changes**Electronic Dental Services (EDS) clearinghouse. [Electronic Dental Services](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclaimseds.html)Wiki lists search and scrollbar behavior is enhanced. [Wiki Lists](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7wikilists.html) Additional payment frequency options added to payment plan amortization schedules. [Payment Plans](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentplanpatient.html)
 When anything is printed or copied within Open Dental, a log entry is added to the audit trail. [Audit Trail](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7audittrail.html)
 Changes to popups are now tracked in a popup audit trail and deleted popups are permanently archived. [Popups](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7popups.html)
 Schedules enhanced to include graphical chart of providers and employees scheduled on a single day. [Schedules](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7schedule.html)
 MedicalInfoViewed permission logs an audit trail entry when the Medical window is viewed. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
 PatProbListEdit permission tracks additions, changes, and deletions to a patient's problems. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
 PatMedicationListEdit permission tracks additions, changes, and deletions to a patient's medications. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
 PatAllergyListEdit permission tracks additions, changes, and deletions to a patient's allergies. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html), [Allergies](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7allergiespatient.html)
 ClaimX/ExtraDent added as an electronic billing option. [Billing Defaults](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billingdefaults.html)
 E-mail override added to dunning messages. [currentMonth] and [nameFLnoPref] added as e-mail variables. [Billing](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billing.html)
 Formatting can be applied to multiple lines at once in the Wiki.
 Reduced number of clicks required in the Supply Inventory windows. [Supply Inventory](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7supplyinventory.html)
 For Canadian users supernumerary teeth can be entered as 99. [Chart Module Preferences](https://www.opendental.com/manual174/modulesetupchart.html)
