# Version 12.4

See [Versions](versions.html)

Version 12.4 was released on 2/12/2013

2013 Codes: The CDT codes for 2013 were released on 12/31/2012 as part of version 12.4.28. Changes were also made to areas like recall and insurance benefits to use the new fluoride code D1208. [CDT Codes](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodetools.html).

**Images in Database:** The automatic update process now works if storing images in the database. [Store Images in Database](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7imagesstoreindb.html).

**HL7 Service:** Multiple HL7 services allowed on the same computer, each for a different database. Also, Open Dental tracks which HL7 service is connected to it, and it won't let a second HL7 service connect.

**Generic HL7 Interface:** Total control over the message structure. Add new custom interfaces as needed. TCP/IP interfaces available to avoid folder sharing issues. [Generic HL7](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7hl7genericmsg.html)

**HL7 Message History:** See history of messages sent and received. [Generic HL7](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7hl7genericmsg.html)

**Electronic Rx:** Can send Rx electronically, but it's still a double entry process. So this feature is considered to be incomplete for general use. [Electronic Rx](electronicrx.html).

**Canada Fee Schedule Tools:** Import carrier fees for a province for a certain year. [Canada](canada.html).

**Minor Changes**
 Recall table now has DateScheduled column. This will make the recall list fill faster.
 A memo can now be added to deposit slips. [Deposit Slip](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7depositslip.html)
 Added a tool to easily and safely switch between MyISAM and InnoDb storage engines. [Database Maintenance](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7databasemaintenance.html)
 Insurance plans default to assignment of benefits checkbox. [Insurance Plans](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insplan.html)
 Supply and Inventory interface overhauled. [Supply](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7supplyinventory.html)
 Perio charting now supports positive gingival margins aka hyperplasia. [Perio](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7perio.html)Equipment Setup permission added. [Security](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7security.html).
 Medical history can now be printed. [Medical](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7medical.html).
 Employee schedules grid in the appointment module can now show schedule notes.
 Security permission added to restrict access to the billing window. [Security](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7security.html).
 Security permission added for editing problems. [Security](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7security.html).
 Checkboxes and Input sheet fields are now reportable. [Sheets Radio Buttons](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheets.html)
 Multiple problems at a time may now be selected and added to a patient's medical history. [Problems](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7problempatient.html).
