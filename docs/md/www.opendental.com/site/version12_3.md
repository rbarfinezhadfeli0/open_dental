# Version 12.3

See [Versions](versions.html)

Version 12.3 was released on 10/4/2012

**Text Messages:** Send text messages to patients for appointment confirmations and more via CallFire for only $0.03 per message. [CallFire](../manual232/bridgecallfire.html).

**Import Medical History:** Patient allergies, medications, and problems can be imported using sheets. Works for both kiosk and web forms. [Patient Forms Import](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientformsimport.html).

**Import Insurance Plans:** Primary and secondary insurance plans can be imported using sheets. Works for both kiosk and web forms. [Patient Forms Import](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientformsimport.html).

**Program Links:** Local path overrides. [Program Links](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7programlinks.html).

**Minor Changes**
 SecurityLog.FKey added so that appointment audit is for a single appointment only.
 Sheet def edit, rectangle selection on perimeter so that underlying items can be clicked on.
 Option for appointment time locked by default. Also prevents recalc of time when changing procs.
 Making appointments is faster now when double clicking or when using new buttons at the right.
 Display field for procedure code abbreviation in Chart module Progress notes.
 Age limit for prophy and fluoride on child recall appointment, instead of being hard coded to 12 years old.
 QuickBooks, added button for sending an existing deposit to QB. Also, memo can be sent. [QuickBooks](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7quickbooks.html).
 Batch insurance payment window, search field to filter by carrier. [Batch Insurance Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimpaymentbatch.html).
 Security permission for creating a new carrier.
 Appointment confirmation list can now be sorted to exclude or only show recall appointments.
 Titlebar can now be set to show a patient's birthdate instead of PatNum or ChartNumber. [Miscellaneous Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7miscsetup.html).
 Security permission ImageDelete can have a date or days limitation.
 Security permission for Dashboard added. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html).
 Default ICD9 code for procedures. [Chart Module Preferences](https://www.opendental.com/manual174/modulesetupchart.html).
 When using clinics, the payment edit window can show family balances grouped by provider. [Payments](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7payment.html).
 Security permission for editing Auto Notes and Quick Notes added.
 Tscan bridge. [Tscan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgetscan.html).
 ClioSoft bridge. [ClioSoft](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgecliosoft.html).
 5010 999 eclaim acknowledgments.
 5010 277 eclaim acknowledgments.
 Installment Plans have been added to the Account Module as an alternative to Payment Plans. [Installment Plans](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7installmentplan.html).
 Batch insurance payment window now shows clinics. [Batch Insurance Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimpaymentbatch.html).
 When adding new providers, they are added above the currently selected providers.
 Procedure button with exactly 6 codes on it of PA,PA+,PA+,PA+,PA+,PA+, will assign tooth numbers to the PA+ codes of 8,14,19,24, and 30 respectively.
