# Version 6.3

See [Versions](versions.html)

Version 6.3 was released on 1/23/09.

**ASAP List:** A list of appointments for patients available on short notice. [ASAP List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7asaplist.html).

**Sheets:** Signature box added. Stylus or Topaz may be used. New sheet type for consent forms. [Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheets.html). [Consent Forms](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7consentforms.html).

**Letters:** Quick letters moved over to the Sheets framework, allowing much more control and use of static images. This also replaces the stationery feature. Letters are now archived. Includes both patient letters and referral letters. [Patient Letters](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7letterspatient.html), [Referral Letters](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7letterreferral.html).

**Mobile Version:** A program has been built for the Windows Mobile Pro platform. Synchronizing functionality has been added to the main program. Includes viewing of appointments and patients.

Option to handle blank co-pay fee schedule entries as zeros. [Insurance Plan Types](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insplantypes.html).
 Changes to statement for Australia.
