## Major Changes

**Advertising Postcards Feature**

* Set up office with a Postcard Account. [Postcards Setup (Advertising)](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7postcardssetup.html)

**Secure Email Feature**

* Enable and set up secure email: [Secure Email Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7secureemailsetup.html)
* Send secure email messages. [Email Message Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailmessage.html)
* Supports HTML formatting. [HTML Email](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailhtml.html)
* Recipients use 2-factor authentication to view and reply to messages. [Secure Email: Workflow](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7secureemailworkflow.html)
* Display secure email messages in Patient Portal. [Patient Portal: What Patient Sees](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7portalpatientsees.html)

**Updated CDT Codes for 2022**. [Procedure Code Tools](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodetools.html)
