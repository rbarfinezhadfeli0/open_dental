MySQL 5.0

See [MySQL](mysql.html)

Follow this process to upgrade to MySQL version 5.0. This only needs to be done on the server.

1. Download: [MySqlUpgrade50.exe](../resources/MySqlUpgrade50.exe).
2. Select open when given a choice, or double-click the file in the downloads folder.
3. Select Unzip to save the files to the given location and to automatically run the installation program.
   * If, for some reason, this needs run again later, a copy is on the hard drive at the specified location.
4. Click Install and follow the instructions to complete the [MySQL](mysql.html) installation.
