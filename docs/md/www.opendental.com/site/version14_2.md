# Version 14.2

See [Versions](versions.html).

Version 14.2 was released on 8/21/2014.

**Patient Portal**: Open Dental hosted patient portal. [Patient Portal](patientportal.html)

**Canadian Procedure Codes:** CDA codes are now available for download. [Procedure Code Tools](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodetools.html)

**Blue Theme:** The first phase towards adding themes to Open Dental is complete. [Miscellaneous Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7miscsetup.html)

**Medical Claim Form:** 1500 claim form updated from version 08/05 to version 02/12. [Medical Insurance](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7medicalins.html)

**Sheets:** Patient images can be added to sheets. [Sheets: Patient Images](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetspatimage.html)

**EOBs/ERAs:** Download or print electronic EOBs. This completes [Feature Request](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7featurerequests.html) #3264. #138 is still in progress. [ERA 835](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimsera835.html)

**Minor Changes**
 EHR Stage 1 Timely Access and Electronic Copy measures replaced with Electronic Copy Access. [EHR Electronic Access](ehreaccess.html)
 VistaDent bridge. [VistaDent](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgevistadent.html)
 TigerView bridge enhanced to pull thumbnails into Open Dental. [TigerView](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgetigerview.html)
 Rx Edit security permission. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
 Added more static text fields to sheets. [Static Text Fields](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetsstatictext.html)
 Insurance Payment Type category added to definitions (e.g. EFT, check). [Definition: Insurance Payment Types](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7inspaytypes.html)
 Added patient name and page number to printed progress notes. [Chart](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7chart.html)
 Time card management enhanced for easier navigation through multiple employees. [Time Clock](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7timeclock.html)
 Software name, provider name, and date information added to several print outs and windows needed for EHR audits. [Measure Calc](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrmeasures.html), [Patient Lists](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrpatientlistgenerate.html), [CDS](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ehrcdsconfigure.html), [Update](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7update.html)
 Auto Notes added to the Ortho Chart. [Ortho Chart](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7orthochart.html)

 Multiple ICD9 diagnosis codes for procedures. [Procedures - Medical Tab](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7proceduremedical.html)
 Basic e-mail inbox. This completes [Feature Request](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7featurerequests.html) #3263. #360 is in progress. [E-mail Inbox](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7emailinbox.html)

 Patient age locks when a date deceased is entered. EHR only. [Edit Patient Information](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientedit.html)
 Providers can send attachments to patients using Secure Web Mail. [Secure Web Mail](securewebmail.html)
