# Version 18.3

See [Versions](versions.html)

Version 18.3 was released on 12/10/2018. EHR certified 2014 edition.

![](images/YouTube_PlayButton_15px.png) Overview of [Version 18.3](https://www.youtube.com/embed/hqe-SADXiEM).

## Major Changes

* Canada electronic claim integration with TELUS network. [Claimstream](claimstream.html)
* Insurance plan level substitution codes. [Estimate Downgrades](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7downgrades.html)
* Allow Recurring Charges to run more than once per month. [Authorize Recurring Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recurringchargesauthorize.html)

## Changes by Category

**Appointments**

* Overlapping appointments indicator. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html)
* Appointment search by blockout type. [Pinboard](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7apptpinboard.html)

**Clinics**

* Clinic filters added to ERA and Appointments for windows. [Patient Appointments](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7apptspatient.html), [ERA Match with Claim](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimseramatching.html)

* Preference added to print statements alphabetically instead of by clinic. [Manage Module Preferences](https://www.opendental.com/manual183/modulesetupmanage.html)
* Speed enhancement made to the recall list when using clinics. [Recall List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recalllist.html)

**eServices**

* Mobile Web show blockouts. [Mobile Web Operatories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mobilewebuseops.html), [Blockouts](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7blockouts.html)

**Insurance**

* Account Module Preference to *Exclude 'None' as an option on Custom Tracking Status* for custom claim tracking. [Account Module Preferences](https://www.opendental.com/manual183/modulesetupaccount.html)

* Reportable Insurance Filing Code definition category for use with the insurance filing code window and edit insurance plan window. [Insurance Filing Codes](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insfilingcodes.html), [Definitions: Insurance Filing Code Group](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7definitionsinsfilingcodes.html)

* Insurance plan level substitution codes. [Estimate Downgrades](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7downgrades.html)
