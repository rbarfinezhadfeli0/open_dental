# Version 6.4

See [Versions](versions.html)

Version 6.4 was released on 3/16/09

**Recall:** Recall now includes ALL patients, regardless of any triggers. Multiple reminders can be tracked, with separate messages for each. [Recall](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recall.html).

**Sheets:** A custom sheet can be copied. Fields can be added within the text of paragraphs. [Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheets.html).

**Auto Notes:** Overhauled interface is more user-friendly, includes multiSelect prompts, and has enhanced functionality. [Auto Notes](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7autonotes.html). **(caution: If you are using AutoNotes, read that page in the manual before upgrading. AutoNotes will need to be re-entered.)**

**Minor changes:**
 The lab case list now has an option to show lab cases unattached to appointments.
 in Lists, Insurance Plans. You can hide all inactive plans with one button.
 Moving an appointment now more intelligently handles the IsHygiene checkbox.
