Patient Dashboard

The patient dashboard is a customizable view of basic patient information.

In the Main Menu, click Tools, Patient Dashboards.

![](images/patientDash.png)

The dashboard is optional and can be set per user. When enabled, it is docked on the right side of the program.

![](images/youtubeLogo.png) See our [QuickTip: Patient Dashboard](https://youtu.be/3_59HW7wDcw) video for a brief overview of setup and usage.

The Patient Dashboard can be used to display information at a glance without leaving the current module. This includes:

* Patient picture
* Balance
* Birthdate and age
* Insurance information
* [Graphical Tooth Chart](0_graphicaltoothchart.html)
* Upcoming appointments
* and more!
