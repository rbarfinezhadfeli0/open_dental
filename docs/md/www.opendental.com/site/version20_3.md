# Version 20.3

See [Versions](versions.html)

Version 20.3 was released on 10/13/2020.

![](images/youtubeLogo.png) Overview of [Version 20.3](https://youtu.be/8Si8-n-vfvg)

## Major Changes

**2021 CDT Codes**: The 2021 CDT codes are available in version 20.3.48 or 20.4.28 or greater. [Procedure Code Tools](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodetools.html)

**New CareCredit Integration**: Maximize treatment acceptance with the full CareCredit integration.

* Pre-screen patients for approval using QuickScreen. [CareCredit Action](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7carecreditaction.html)
* Take CareCredit payments and issue refunds. [CareCredit Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7carecreditpayment.html), [CareCredit Refund](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7carecreditrefund.html)
* Review transactions. [CareCredit Transactions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7carecredittransactions.html)
* ![](images/youtubeLogo.png) See our video: [CareCredit Integration Demo](https://youtu.be/owhEAk6TIdI)

**Bring Your Own Device (BYOD)**: BYOD allows patients to check-in for appointments, complete patient forms, and take a patient picture, all from their mobile device.

* Enable BYOD in [eClipboard Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclipboardsetup.html).
* Send a link via text message to the patient. Texts can be sent manually or automatically using Arrivals. [BYOD: Workflow](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7byodworkflow.html)

**Web Sched Existing Patient**: Allow existing patients to schedule their appointments online. Available in V20.3.49. [Web Sched Existing Patient](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7webschedexistingpatsetup.html)

**Updated Blue Book Feature**: The improved Blue Book feature makes estimates easier than ever for your out-of-network insurance plans.

* As claims are received, enter the allowed fees to improve estimates for future procedures. [Receive Claim](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimreceive.html)
* Set up preferred Blue Book calculations. [Insurance Blue Book Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bluebooksetup.html)
* ![](images/youtubeLogo.png) See our video: [Blue Book Webinar](https://youtu.be/Jghg56V6uRA)

**Language Translations for Web Forms**: When language translations are created for sheets, patients can select their preferred language when filling web forms. [Web Forms: What Patient Sees](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7webformspatient.html)

**Patient Responsibility Column**: Now available in Edit Claim and Enter Payment windows. [Account Module Preferences](https://www.opendental.com/manual203/modulesetupaccount.html), [Receive Claim](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimreceive.html)

* ![](images/youtubeLogo.png) See our video: [QuickTip: Adding the Patient Responsibility Column](https://youtu.be/q6hGlEB_gAU)

## Changes By Category

**Account Module**

* Assign a Payment Type override to individual Recurring Charges. [Authorize Recurring Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recurringchargesauthorize.html)
* Account Module preference disallows primary claim status to be marked as 'Hold Until Pri Received'. [Account Module Preferences](https://www.opendental.com/manual203/modulesetupaccount.html)
* EdgeExpress added as alternative to XCharge. [Edge Express Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7edgeexpress.html)

**eServices**

* Automated email birthday messages. [Mass Email Birthdays](https://www.opendental.com/manual203/massemailbirthdays.html)
* Alert created when the MySQL and eConnector have different times. [Alerts](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7alerts.html)
* Option to exclude a patient from receiving any automated appointment messaging. [Edit Patient Information](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientedit.html)

**Miscellaneous**

* Enhanced security so providers must be associated with a user to access PDMP. [PDMP Bridge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgepdmpillinois.html)
* Added fields to send electronically on HCFA 1500 Claim Form. [HCFA 1500 Claim Form](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimform1500.html)
* User setting to show existing other color on tooth chart when a different provider in the practice completed work. [User Settings](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7usersettings.html)
* Automatically load task search when searching by tasknum. [Task Search](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7tasksearch.html)
* Appointment buttons moved to top of Appointment Module. [Appointments Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html)
* Enterprise preference to turn off email address auto-complete in Edit Email Message window. [Enterprise Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7enterprisesetup.html)
* Daily Production, Production and Income report, and Monthly Production Goals reports calculate treatment plan discounts into adjustments. [Monthly Production Goal Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportproductiongoal.html), [Production and Income Reports](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportprodinc.html)
* Filter out archived wiki pages when adding internal links. [Wiki Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7wikiaddedit.html)
* Changing the referral associated to a patient logs an entry in the Audit Trail. [Referrals](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7referrals.html)
* Determine level of data sent to secondary insurance carriers. [Family Module Preferences](https://www.opendental.com/manual203/modulesetupfamily.html), [Carriers](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7carriers.html)
* Customize border color for windows within Open Dental. [Definitions: Misc Colors](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7definitionsmisccolors.html)
* Added Procedure Buttons to Procedure Code Tool for Canada users. [Canada Procedure Code Tools](canadaproccodetools.html)
