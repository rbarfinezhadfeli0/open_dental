XDR Sensors

See [Sensor Compatibility](sensorcompatibility.html)

XDR sensors are supported.

In [Imaging Devices](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7imagingdevices.html), add a Device.

![](images/imagingXDR.png)

Check the box to Show Twain UI.

In the [Imaging Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7images.html), in the top Toolbar, create a new Mount. Click Acquire.
