CEMT User Settings

In [CEMT](central.html), click File, User Settings.

![](images/cemtUserSettings.png)

User Settings only display when using [Middle Tier](middletier.html).

**User**: Displays the currently logged-in user.

**Log me in automatically**: Determines whether the selected user automatically logs in when accessing CEMT.
