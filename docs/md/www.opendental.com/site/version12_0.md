# Version 12.0

See [Versions](versions.html)

Version 12.0 was released on 2/10/2012

**Coordination of Benefits:** COB rules can be set per insurance plan as well as a global default. [COB](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7cob.html).

**EOB scanning:** Insurance Payments have a place to attach scanned EOBs for later reference. [Finalize Insurance Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimpayfinalize.html).

**Command-line Arguments:** Many more command-line arguments added so that specific Open Dental databases, users, and patients may be launched from batch files, from shortcuts, and from other programs. [Command-line](commandline.html).

**Central Enterprise Management Tool:** Customers with multiple locations and separate databases now have a way to magage and interact with those separate databases from a centralized tool. [Central Manager](central.html).

**Popups for Family:** Popups can now be family level or even superfamily level. Multiple popups per patient and/or family. [Popups](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7popups.html).

**Minor Changes**
 Billing default messages for email subject and body. [Billing](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billing.html).
 Email statement fields: nameF, nameFL, PatNum. [Billing](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7billing.html).
 CC recurring charges can be applied to payment plans. [Recurring Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recurringcharges.html).
 Consent forms can now be sent to the kiosk.
 Security permission for changing subscriber on existing plan. [Security](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7security.html).
 Chart module can be made to load more quickly by not showing as many object types in the Show tab.
 Chart module Progress Notes, order of rows has been overhauled. [Show Tab Chart](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7showtabchart.html).
 Appointment views, when show ops for sched provs, clinic filter added. [Appointment Views](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointmentviews.html).
 UB04 claimform now included. [Medical Ins](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7medicalins.html).
 FeeSchedules can no longer be hidden if in use by a Provider. [Fee Schedules](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7feeschedules.html).
 Claimform 1500 now part of database for everyone rather than requiring a download.
 Sheets with multiple large background images work better.
 Clinics: Filter recall list, confirmation list, unscheduled list, and billing list. [Appointment Lists](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointmentlists.html).
