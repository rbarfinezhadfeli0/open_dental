Confirmation List

The Confirmation List is a way to manually manage reminders and confirmations for scheduled appointments.

In [Appointments Module](0_appointments.html), click Lists, **Confirmations**.

![](images/confirmationList.png)

Track communication with patients and send batch emails, postcards, and text messages to confim appointments. Users can browse to other windows while the Confirmation List is open.

Confirmation messages are customizable.

![](images/YouTube_PlayButton_15px.png) See our video: [Confirmation List](https://youtu.be/oVh2qGXvg-w).

## List Options

Use the View options to filter by appointment confirmation status, appointment date, and more to determine which appointments are included on the Confirmations List. Use the Set Status options to quickly update appointment confirmation statuses for one or more patients.

![](images/confirmationListView.png)

## Communication Options and More

Use the buttons at the bottom to send out confirmations or generate lists.

![](images/confirmationListButtons.png)

* **Postcard Preview**: Generate postcards to mail to patients.
* **Label Preview**: Generate mailing labels to print on a standard sheet of 30.
* **Email**: Send confirmation emails.
* **Text**: For [Integrated Texting Feature](textintegrated.html) users, send confirmation text messages.
* **Run Report**: Click to create a customized confirmation report and send the data to a simple text file. This is useful for using mail merge in Word or when generating custom postcards, letters, etc. outside of Open Dental.
* **Print List**: Print the confirmation list as shown.
