# Global Payments

Global Payments is a [Native Payment Integration](nativepaymentintegrations.html) within Open Dental.

![](images/headerGlobalPayments.png)

Global Payments is available internationally, including in Canada. Visit [globalpaymentsintegrated.com](https://www.globalpaymentsintegrated.com/en-us) for more information and to sign up.

## Getting Started

Contact [Global Payments](https://www.globalpaymentsintegrated.com/en-us) to sign up. Once you're signed up, set up the program link in Open Dental using the credentials provided by Global Payments.

## Using Global Payments

Easily process credit card payments from the [Patient Payment](0_patpayment.html) window, [Insurance Payment](0_insurancepayments.html) window, or [Payment Portal](patientportal.html).

![](images/globalpaymentsGraphic.gif)

Payment returns and voids can also be processed from Open Dental. Transactions are automatically posted to the [Account Module](0_account.html).
