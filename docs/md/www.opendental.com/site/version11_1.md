# Version 11.1

See [Versions](versions.html)

Version 11.1 was released on 12/12/2011

Prepare ahead of time: Version 11.1 requires [dotNet](dotnet.html) 4.0, which should already be installed on newer computers and computers that get regular updates.

**Scanning:** Automatic document feeder support added. Scans multiple pages to a single pdf. Scans duplex (double sided). [Scanning and Importing](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7images.html).

**Sheets Setup:** Toolbox and hotkeys added to allow click and drag to select, align left or top, copy/paste, rubber stamp, and tab order. [Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheets.html).

**Dashboard:** A single window showing many important practice parameters in graphical format. Dashboard.

**Batch Insurance Payment:** EOBs and the insurance checks that come with them can now be entered in a single batch window. [Batch Insurance Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimpaymentbatch.html).

**E-claims:** Medical and Institutional e-claims. New 5010 format available for all 3 claim types. Many fields added to support the new formats. [Medical Eclaims](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7medicalins.html).

**Referred Procedure Tracking:** Individual procedures can now be referred out and their status tracked in a tool. [Referred Proc Tracking](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7referrals.html).

**Superfamilies:** Groups of families for unique situations such as nursing homes. [Superfamily](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7superfamily.html).

**Appointment Printing:** The daily printout now prints sharper and has size customization options.  [Appointments](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html).

**Minor Changes**
 Payer ID table can be edited to control which electronic IDs are treated as Medicaid.
 Unscheduled list, multiple appointments can be deleted at once.
 Medications list can be printed.
 All patients for one provider can be moved to another provider. [Providers](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7providers.html).
 Sheets, many more fields added, especially for routing slips.
 Image export.
 Procedurecode list, the Show Hidden button now has a default setting and is initially set to not hide anything.
 Sending claims, clinic must be same for batch.
 Chart module, when show by teeth, group notes for associated teeth now show.
 Security permissions added for Providers, EcwAppointmentRevise, ReferralAdd, ProcedureNote, and perio red numbers. [Security](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7security.html).
 Apixia bridge. [Apixia](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeapixia.html)
 Appointment search behavior can now also look for open operatories and has better provider options. [Appointments](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html).
 Chart module optimized to run twice as fast when entering new procedures. Only noticeable for users with slow connections.
 Group notes can now aggregate individual procedure notes. [Procedure Group Note](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procgroupnote.html).
