# Version 18.4

See [Versions](versions.html)

Version 18.4 was released on 03/20/2019. EHR certified 2014 edition.

![](images/YouTube_PlayButton_15px.png) Overview of [Version 18.4](https://www.youtube.com/embed/R_XOvNIIyIA).

## Changes by Category

**Appointments Module**

* Automatically detach treatment planned procedures from completed appointments. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html)
* When block appointment scheduling blockout types are added on top of an existing standard blockout type, the standard blockout type will be removed. [Appointment Module Preferences](https://www.opendental.com/manual184/modulesetupappts.html), [Blockouts](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7blockouts.html)
* Enhancements to Unscheduled List. [Unscheduled List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7unscheduled.html), [Appointments Module Preferences](https://www.opendental.com/manual184/modulesetupappts.html)
* Filter ASAP list by procedure code. [ASAP List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7asaplist.html)
* Add functionality to the Recently Contacted tab in Recall. [Recall List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recalllist.html)
* Prompt user when making an appointment if an unscheduled appointment exists. [Unscheduled List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7unscheduled.html)

**Chart Module**

* Procedure Status for *In Process*. [Chart Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7chart.html), [Procedure Buttons](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurebuttons.html), [QuickTip: In Process Status for Multi-Visit Treatment](https://www.youtube.com/watch?v=_W5Y1S8NZ0A)
* Easily chart quadrant procedures. [Enter Treatment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7entertreatment.html)
* Right click procedure to set to EO or EC status. [Chart Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7chart.html)
* Additional fields added to the Edit All Procs window. [Chart Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7chart.html)

**eServices**

* Edit and send eConfirmations, eReminders, and Patient Portal invites as HTML. [Automated Messaging Feature](ereminders.html)
* Filter the mobile web pharmacy list by clinic. [Mobile Web Pharmacies](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mobilewebusepharmacies.html)
* Text Messaging window can now group by phone number. [Text Messaging Mailbox](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7textmsginbox.html)
* Mobile friendly Web Forms. [Web Forms: What Patient Sees](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7webformspatient.html)
* Mobile friendly Patient Portal. [Patient Portal: What Patient Sees](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7portalpatientsees.html)

**Insurance**

* ERA write-offs posted by plan type. [Manage Module Preferences](https://www.opendental.com/manual184/modulesetupmanage.html), [ERA](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7era.html)
* Split Claim option added to ERA payment window. [ERA](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7era.html)
* View ERA'a and EOB's from Edit Claim window. [Claim](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimedit.html)
* Enhancements to frequency limitations. [Insurance History](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7inshistory.html)

**Preferences**

* Additional options for themes. [Miscellaneous Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7miscsetup.html)
* Preference to not recalculate insurance estimates when a claimproc is *Received*. [Account Module Preferences](https://www.opendental.com/manual184/modulesetupaccount.html)
* Preference for number of days to automatically refresh the Appointment Module. [Account Module Preferences](https://www.opendental.com/manual184/modulesetupaccount.html)
* Preference to select payment type for recurring charges. [Account Module Preferences](https://www.opendental.com/manual184/modulesetupaccount.html), [CC Recurring Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recurringcharges.html)
* Preference to remove All as an option for many clinic boxes. [Enterprise Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7enterprisesetup.html)
* Preference to bill UCR fee and mark non-covered services as "do not bill to insurance". [Family Module Preferences](https://www.opendental.com/manual184/modulesetupfamily.html), [Other Benefits](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7otherbenefits.html)
* Setup window for enterprise users. [Enterprise Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7enterprisesetup.html)
* Automate Repeating Charges tool. [Repeating Charge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7repeatingcharge.html), [Account Module Preferences](https://www.opendental.com/manual184/modulesetupaccount.html)

**Security**

* Permission for Insurance Verification. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html), [Insurance Verification List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insverifylist.html)
* Permission for *Production and Income - View All Providers* now applies to graphical reports. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html), [Graphic Reports](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportsgraphic.html)
