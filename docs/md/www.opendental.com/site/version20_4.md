# Version 20.4

See [Versions](versions.html)

Version 20.4 was released on 12/17/2020.

![](images/youtubeLogo.png) Overview of [Version 20.4](https://youtu.be/GZL2dWandCg)

## Major Changes

**2021 CDT Codes**: The 2021 CDT codes are available in version 20.3.48 or 20.4.28 or greater. [Procedure Code Tools](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodetools.html)

* ![](images/youtubeLogo.png) See our video: [How to Update Your CDT Codes](https://youtu.be/_z4X2tl2cEM)

**Zoom**: Allows users to enlarge the overall display of Open Dental.

* Zoom window allows users to view current settings and set zoom scale. [Zoom](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7zoom.html)
* Added *Zoom* permission. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
* ![](images/youtubeLogo.png) See our video: [Using the Zoom Tool](https://youtu.be/Y20BGe1w8tA)

**QuickBooks Online integration**: Connect QuickBooks Online to Open Dental.

* Determine which Deposit account, Income accounts, and Classes you want to display in Open Dental. [QuickBooks Online](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7quickbooksonline.html)
* Set QuickBooks Online as your default deposit software. [Manage Module Preferences](https://www.opendental.com/manual204/modulesetupmanage.html)
* Send deposit information to QuickBooks Online. [Deposit Slip](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7depositslip.html)
* ![](images/youtubeLogo.png) See our Webinar: [QuickBooks Online Integration](https://youtu.be/a1czPuwgoyY)

**Insurance Verification Needed**: Display an icon on upcoming appointments if patient's insurance has not been marked as verified.

* Add Insurance Verification symbol to Appointment Views. [Appointment View Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointmentvieweditwindow.html)
* Add *Verify Insurance* note to the Appointment Bubble. [Display Fields](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7displayfields.html)
* ![](images/youtubeLogo.png) See our video: [Adding Insurance Verification Needed Indicator to Appointments](https://youtu.be/T7JWgDnRqg0)

**PDMP for additional states**: PDMP is now available for all states.

* For users in California, Illinois, Kentucky, Maryland, Utah, and Washington, see [PDMP Bridge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgepdmpillinois.html)
* For users outside of the above states, see [Bamboo Bridge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeappriss.html)
