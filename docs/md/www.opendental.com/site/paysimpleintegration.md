# PaySimple

PaySimple is a [Native Payment Integration](nativepaymentintegrations.html) within Open Dental.

![](images/headerPaySimple.png)

PaySimple is available for U.S. customers only. Visit [paysimple.com](https://paysimple.com/partner/open-dental) for more information and to sign up.

The PaySimple integration allows you to process credit card and ACH payments.

## Getting Started

Contact [PaySimple](https://paysimple.com/partner/open-dental) to sign up. Once you're signed up, set up the PaySimple program link in Open Dental using the username and key provided by PaySimple.

## Using PaySimple

Easily process credit card or ACH payments from the [Patient Payment](0_patpayment.html) window, [Insurance Payment](0_insurancepayments.html) window, or [Patient Portal](patientportal.html).

![](images/paysimpleGraphic.gif)

Payment returns and voids can also be processed from Open Dental. Transactions are automatically posted to the [Account Module](0_account.html).
