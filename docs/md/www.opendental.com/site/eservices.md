# eServices

eServices are internet-based features available through Open Dental that can provide enhanced access for patients and providers, and automate many tasks.

![](images/headereServices.png)

The result is increased productivity and efficiency and a better patient experience, which translates into improved patient retention and practice growth.

![](images/eServicesDevicesLarge.png)

### eService Fees

See [Fees for Software, Support, and Services](https://opendental.com/site/fees.html#eservices).
 To sign up, each location must be on support and have a unique, active registration key. Only customers on support will be able to sign up for any eService. Integrated Texting can still be utilized even if the office cancels support; in these cases, we will only provide support limited to the eService still in use. Practices must remain on support while using all other eServices.

### Other Details

* eServices require Installation of the eConnector, the gateway of communication between your computer and our secure servers.
* For a list of supported browsers, see [Computer Requirements](computerrequirements.html).
* Distributors who would like to resell Open Dental eServices, see [Distributors](distributors.html).
* For information on how data is handled when using eServices, see [Encryption of Data at Rest and in Transit](encryption.html).

| Service | Description |
| --- | --- |
| [eServices Bundle](eservicesbundle.html) | Includes most eServices at a discounted price. |
| [eClipboard](eclipboard.html) | Allow current patients to check in, sign and complete forms, and take a patient picture all from a mobile device.   The app is available in the iOS and Android app stores for supported devices. |
| [eConfirmations](ereminders.html#econfirmations) | Send automated text message and/or email confirmations to remind patients about an upcoming appointment AND allow the patient to e-confirm.  Includes access to Arrivals. |
| [eReminders](ereminders.html#ereminders) | Send automated text message and/or email reminders about upcoming appointments. |
| [Automated Thank-Yous](ereminders.html#thankyou) | Send automated thank-you text message and/or email after a patient schedules an appointment. |
| [General Messages](ereminders.html#general) | Send an automated message after an appointment is set complete. |
| [Integrated Texting](textintegrated.html) | Send SMS text messages directly to patient's mobile devices and receive unlimited inbound text messages from patients. Works with eReminders, eConfirmations, and Web Sched. |
| [Secure Email](secureemail.html) | Send secure email messages to patients and outside providers. |
| [ODMobile](https://opendental.com/site/mobileapp.html)   (Formerly Open Dental Mobile) | View your schedule, edit appointments, 2-way text with patients, and send prescriptions through DoseSpot, all on your mobile device. |
| [Mobile Web](mobileweb.html) | Connect to Open Dental any time, anywhere using a supported internet browser and your mobile device.  \*ODMobile replaces Mobile Web. Existing Mobile Web users may continue to use Mobile Web, or move to ODMobile. |
| [Patient Portal](patientportal.html) | Give patients access to their health information such as appointments, treatment plans, and images. Securely communicate via WebMail. Allow online payments with X-Charge or PayConnect. |
| [Web Forms](webforms.html) | Allow patients to complete forms online using any browser. |
| [Web Sched Recall](websched.html#recall) | Send current patients who are due for a recall a link via text or email that allows them to schedule their appointment online. |
| [Web Sched New Patient](websched.html#newpat) | Allow new patients to schedule their first appointment online via a link on your practice website. |
| [Web Sched Existing Patient](websched.html#existing) | Allow existing patients to schedule their appointments online via your website. |
| [Web Sched ASAP](websched.html#asap) | Send current patients who are on the ASAP list a link via text or email that allows them to schedule their appointment online. |
| [ODTouch](odtouch.html) | Navigate patient appointments via a convenient tablet device. |
