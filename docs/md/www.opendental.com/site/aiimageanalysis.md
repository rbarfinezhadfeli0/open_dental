# AI Image Analysis Integrations

Open Dental is integrated with two AI Image Analysis companies to process images through the Imaging Module.

![](images/headerAIAnalysis.png)

The Open Dental integration with both vendors allows you freedom to choose an AI Image Analysis solution that is right for your practice. Both options provide an easy-to-follow workflow with quick visual analysis of patient images.
