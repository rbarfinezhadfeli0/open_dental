# Version 19.3

See [Versions](versions.html)

Version 19.3 was released on 12/23/2019.

![](images/YouTube_PlayButton_15px.png) Overview of [Version 19.3](https://www.youtube.com/watch?v=QZ7tPAFey5k).

**Appointment Module**

* Complete overhaul of drawing logic for smoothness and speed
* Custom font size on appointments. [Appointments Module Preferences](https://www.opendental.com/manual193/modulesetupappts.html)
* Appointments can overlap in an operatory. [Appointments Module Preferences](https://www.opendental.com/manual193/modulesetupappts.html)
* Horizontal scroll.[Appointments Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html), [Appointment View Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointmentvieweditwindow.html)
* Sort Recall List by type. [Recall List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recalllist.html)
* Right-click option to add autonotes to an appointment note. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html)
* New preference that can blocks changing appointment status or detaching procedures. [Appointments Module Preferences](https://www.opendental.com/manual193/modulesetupappts.html)
* Change width of provider bar on left of appointments. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html), [Appointments Module Preferences](https://www.opendental.com/manual193/modulesetupappts.html)
* Add text to provider bars. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html)

**Account Module**

* Exclude options added to Finance Charges. [Billing/Finance Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7financecharges.html)
* Filter added to Payment window to show only explicitly paid balances. [Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7payment.html)
* Recurring Charges allow user to pick the day in addition to the month. [Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7payment.html), [CC Recurring Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recurringcharges.html)
* Added character limit to narratives sent via the DentalXChange Attachment Service. [DentalXChange Attachment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7dentalxchangeattachmentservice.html)
* Carrier name sent to PaySimple when processing insurance payment. [Finalize Insurance Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimpayfinalize.html)
* Dynamic Payment Plan. [Payment Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentplandynamic.html)

**Chart Module**

* Procedure Info window displays primary teeth when selecting a tooth range. [Procedure Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedureedit.html)
* Clinic level defaults for Chart Layouts. [Sheet Def Defaults](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetdefaults.html)
* Allow the same Ortho Chart display field to set different column widths per ortho chart tab. [Ortho Chart Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7orthochartsetup.html)
* New perio path option. [Perio Chart](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7perio.html)

**Manage Module**

* Archive a task list. [Tasks Area](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7tasksarea.html)
* Display patnum on tasks when patient is attached as object. [Task](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7tasks.html), [Task Window](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7taskswindow.html)
* Removed Show Credit Card Info from Manage Module Preferences. [Manage Module Preferences](https://www.opendental.com/manual193/modulesetupmanage.html)
* Supplemental Backup tool. [Supplemental Backups](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7supplementalbackups.html)
* Click to sort columns in Send Claims window. [Send Claims](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimsend.html)
* Additional columns added to Send Claim window. [Send Claims](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimsend.html)

**Security**

* Audit trail entry when assignment of benefits change for an insurance plan. [Insurance Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insplan.html)
* Additional information added when auditing Repeating Charges. [Repeating Charges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7repeatingcharges.html)
* Additional audit trail entries when setting up eServices. [eServices Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eservicessetup.html)

**Miscellaneous**

* Program links can be enabled by clinic. [Custom Bridges](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgecustom.html)
* Reports can be printed by page range. [Complex Report System](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportcomplex.html)
* Ability to view if your office is on support. [Support Status](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7supportstatus.html)
* Is Hidden column added to Laboratories window. [Laboratories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7laboratories.html)
* Apteryx XVWeb allows option for Image Quality. [Apteryx XVWeb Bridge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeapteryxxvweb.html)
* Sheets windows renamed to be more consistent. [Sheet Field Types](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetsaddelements.html)
* Added the Query Monitor. [Query Monitor](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7querymonitor.html)
* Module preferences renamed to be more consistent. [Module Preferences](https://www.opendental.com/manual193/modulesetup.html)
* Character limit increased to 225 characters on Employer field in Insurance Plans. [Employers](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7employers.html), [Insurance Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insplan.html)
* Added checkbox in Provider Setup window to show patient count. [Providers](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7providers.html)
* Ability to set Web Sched Recall provider rules by clinic. [Web Sched Recall](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7webschedrecallsetup.html)
* Sheet Field Defs can have a Y value greater than 2,000. [Sheet Field Types](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetsaddelements.html)
* Redesigned module preference windows. [Module Preferences](https://www.opendental.com/manual193/modulesetup.html)
* Themes changed to only include icons. [Miscellaneous Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7miscsetup.html), [User Settings](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7usersettings.html)
* Ability to search the Provider List. [Providers](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7providers.html)
* Option to show provider column on unearned income reports. [Unearned Accounts Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportunearnedaccounts.html)
