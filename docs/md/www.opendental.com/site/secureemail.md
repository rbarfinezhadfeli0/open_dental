# Secure Email

Secure Email allows users to easily send secure email messages to patients and outside providers.

![](images/secureEmail.png)

## Feature Highlights

* Send Secure Email Messages, including attachments, to patients and outside providers.
* Patients and outside providers can securely view and reply to messages.
* Two-factor authentication protects PHI and sensitive data.
* Easy to access. No need to memorize login information.
* View communication history from Open Dental.

## Getting Started

Secure Email is included in our [eServices Bundle](eservicesbundle.html) or is available for a standalone monthly fee. See [Fees for Software, Support, and Services](fees.html#eservices) for pricing.

Practices must be on support and have a unique, active registration key to sign up for Secure Email and must remain on support while using the service.

Once signed up, set up Secure Email by setting up hosted email addresses and signatures.
