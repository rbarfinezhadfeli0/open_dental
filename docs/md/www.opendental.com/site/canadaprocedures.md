Edit Procedure Canada Tab

For [Canada](canada.html) users, in the [Procedure](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedureedit.html) Info window, click the Canada tab.

![](images/canadaProcedureEdit.png)

**Procedure Type Code**: By default, none is selected. Change the type code if it is needed on [Canada Claims](canadaclaims.html).

**Lab Fee 1 / 2**: Enter any lab fees associated with this procedure (do not enter on the lab procedure itself). For each lab fee entered on a non-lab procedure, one lab procedure with code 99111 is automatically created (beginning with ^^ to visually indicate it is attached). Also see [Canada Lab Fees](canadalabfees.html).
