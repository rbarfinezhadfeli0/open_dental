# Version 15.3

See [Versions](versions.html).

Version 15.3 was released on 11/18/2015.

**Integrated Texting:** A two-way text messaging service hosted by Open Dental. This is a paid service. [Integrated Texting](textintegrated.html)

* Receive free and unlimited text messages from patient to office.
* Send text messages for a per message fee.
* Dedicated phone numbers for each clinic or office using Open Dental.
* Patients can respond with no time limit.
* CommLogs automatically created for all messages.
* Set monthly text message limits by location.
* Send confirmation text messages directly from the [Appointment Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7appointments.html).
* View text log for selected messages. Reply to selected message from text log.

**Fee Schedules**:

* New fee schedule preference determines whether fees apply globally to all providers/clinics or can be changed for specific providers/clinics. [Fee Schedules](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7feeschedules.html)
* Associate procedure fees in a fee schedule to specific providers and/or clinics. [Procedure Code List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodes.html)
* New definition for fee colors. [Definitions Fee Colors](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7definitionsfeecolors.html)

**Web Sched****Enhancements:**[Web Sched](websched.html)

* Settings that determine available appointment slots can be previewed and edited on the eServices Setup, Web Sched tab. [Web Sched Recall Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7webschedrecallsetup.html)
* Only operatories flagged as 'Is Web Sched' are considered when searching for available time slots. [Operatory Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7operatories.html)
* Set provider considerations when determining available time slots (first available, primary, secondary, last seen hygienist).
* Available time slots are based on appointment length (no longer hour blocks).
* Start/end times are based on time increments of the schedule (no longer limited to on-the-hour start times).

**Central Enterprise Management Tool**: [Central Enterprise Management Tool](central.html)

* Search for patients from the main CEMT window. [Central Enterprise Management Tool ( CEMT )](central.html)
* Log in to an Open Dental database from CEMT. [Central Enterprise Management Tool](central.html)
* Sync all security information for multiple databases simultaneously. [CEMT Security](cemtsecurity.html)
* Run Production and Income report by provider across multiple databases. [CEMT Production and Income Report](cemtprodincreport.html)
* Multi-threaded connection speeds up the retrieval of data from multiple databases.

**Payment Plan**: Recalculate amortization schedules for prepayments or payments to principal while preserving historical payments. [Old Payment Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentplanpatient.html)

**Non-Dental Practices or Clinics**: Set a practice or clinic as "non-dental". [Clinic Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7clinicsetup.html)[Practice Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7practice.html)
 If set as non-dental, most tooth and surface references are removed from the interface. [Non-Dental Practice/Clinic](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7nondentalinterface.html)

* Graphical tooth chart removed from the module
* Tabs for missing teeth, movements and primary removed from the module
* Buttons for Perio Chart, Tooth Chart and Ortho Chart removed from the Chart Module toolbar.
* Labels that reference teeth or surfaces are hidden throughout the software
* Procedure Code column widened in Reports.

**ICD-10 Codes**: New preference to use ICD-10 instead of ICD-9 diagnosis codes on procedures. [ICD Diagnosis Codes](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7icdcodes.html)
