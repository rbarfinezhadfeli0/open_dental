Canada Provider Setup

For [Canada](canada.html) users, in [Providers](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7providers.html) List, click **Add**, or double-click an existing provider to edit.

![](images/canadaProvider.png)

For a description of fields, see [Provider](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7providerseditwindow.html). Some fields are specific to Canada. They are described below.

**CDA Number**: The nine-digit provider identification number. Also known as the Unique ID Number (UIN).

**Office Number**: The four-character office identifier. May contain numbers and letters.

**Is CDAnet Member**: Must be checked to send electronic claims. Includes CDA, CDHA, ACDQ, and DAC members.

## Additional Information

* Taxonomy Code is not sent on Canadian claims.
* Supplemental Provider Identifiers are not used in Canada.
* Hygienists in a Hygiene-Only office should not check *Secondary Provider*.
* Hygienists in an office with a dentist provider should check *Secondary Provider*.
* Any provider used on claims must have a correct CDA Number and Office Number entered, as well as active membership to their respective organizations. See [Register Canadian Providers for E-Claims](canadasetup.html) for more information.
