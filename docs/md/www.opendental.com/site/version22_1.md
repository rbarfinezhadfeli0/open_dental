**eClipboard enhancements**

* Set frequencies for images on eClipboard. [eClipboard Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclipboardsetup.html), [eClipboard Images](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclipboardimages.html)

* eClipboard confirmation warning when a patient is loaded to a device in use. [eClipboard Troubleshooting](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclipboardtroubleshooting.html)
