# Version 7.9

See [Versions](versions.html)

Version 7.9 was released on 7/8/2011

**Credit Cards:** Automatic recurring charges. Only works with X-charge.

**Problems:** The old Diseases feature was renamed to Problems and also split off into Allergies. ICD9s can now also be problems. [Problems](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7problemmaster.html), [Allergies](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7allergiesmaster.html), [ICD9s](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7icdcodes.html).

**Drug Interactions:** Interactions supported include: drug-drug, drug-allergy, and drug-problem. Both custom and default messages supported. [Rx Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7rxsetup.html).

**Mobile:** Appointment colors show, calendar added for jumping to different dates, and appt notes show in edit window. .

**Minor Changes**
 eCW users can now manually change patient fee schedules as an option. [eCW](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeeclinicalworks.html).
 eCW Chart module resizes better.
 patient.PatNum added as an available field to claimforms. Intended for box 23.
 Show broken appointments on unscheduled list using a filter. [Unscheduled List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7unscheduled.html).
