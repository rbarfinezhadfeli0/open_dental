Blank Database Setup Checklist

See [Use Blank Database](usingodblank.html)

Use this checklist after installing the full version of Open Dental as a quick-start guide to setting up your database.

In order to use the links on this page, access the Open Dental website using the [Help Feature](helpfeature.html).

Print the page to check off items as you complete them.

## Before You Start

|  |  |
| --- | --- |
|  | [Data Paths](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paths.html): Make sure the paths to Open Dental folders work on all computers. |
|  | [Program Bridges](programbridges.html): Determine what third-party software you will bridge to (e.g., payment processor, digital imaging, etc.) |
|  | [Clearinghouse](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7clearinghouses.html): Select the clearinghouse you will use to send e-claims, then set it up. |

## Optional Setup

**Master Lists**: Many of the following lists will auto-populate as you enter data for the first time. You can also enter data directly into each list.

|  |  |
| --- | --- |
|  | [Allergy List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7allergiesmaster.html): Allergies must be added to the master list before they can be added to a patient's [Medical Chart - Allergies](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7allergiespatient.html) tab. |
|  | [Problem List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7problemmaster.html): New problems can be added to the master list from the patient's [Medical Chart - Problems](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7problempatient.html) tab. |
|  | [Medication List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7medicationmaster.html): New medications can be added to the master list from the patient's [Medical Chart - Medications](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7medicationspatient.html) tab. |
|  | [Referral List](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7referralsetup.html): New referral sources can be added from the Family Module. |
|  | [Dental Laboratories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7laboratories.html): Dental laboratories must be added to the master list before creating a [lab case](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7labcasemanage.html) . |
