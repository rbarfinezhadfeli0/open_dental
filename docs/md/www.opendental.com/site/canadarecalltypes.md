Canada Recall Types

For [Canada](canada.html) users, in the [Main Menu](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mainmenu.html), click Setup, Appointments, **Recall Types**.

![](images/canadaRecallTypes.png)

The above screenshot is an example of how a Canadian office might set up their Recall.

Add default Recall Types, or reset to default Recall Types by running the [Canada Procedure Code Tools](canadaproccodetools.html) for *Recall Types*.

For details about the Recall feature, see [Recall Types](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7recalltypes.html).
