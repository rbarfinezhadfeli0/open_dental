Audit Trail

The Audit Trail is an important [Security](0_security.html) feature which logs actions taken by users.

In the Main Menu, click Tools, Audit Trail.

![](images/audittrail.png)

Only specified users can access the Audit Trail. The log can be filtered by date, patient, permission, and user, making it easy to track down specific changes or actions.

The log includes:

* Date and time
* Associated patient
* User name and computer
* Permission name
* Additional details on the changes made
