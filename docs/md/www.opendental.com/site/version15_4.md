# Version 15.4

See [Versions](versions.html).

Version 15.4 was released on 2/24/2015.

## Major Changes

**Automated appointment reminders:**

* Send automated reminders for scheduled appointments using e-mail or integrated texting. [Automated Appointment Reminders](ereminders.html)
* Send automated Web Sched e-mails based on recall due dates. [Web Sched Recall](websched.html)

**Clinics:**

* Override clearinghouse settings on a clinic-level. [Clinic Clearinghouse Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7clearinghousesclinics.html)
* Use different X-Charge or PayConnect merchant accounts for different clinics. [X-Charge Clinic Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7xchargeclinic.html), [PayConnect Clinic Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7payconnectclinics.html)

**eServices:** New eConnector service enhances and simplifies communication between Open Dental secure servers and your office server. [eConnector](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7econnector.html)

**Insurance:** Many enhancements including automatic ERA download.

**Patient Information:**  Set certain fields as required on the Edit Patient Information and Add Family windows. [Required Fields](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7requiredfields.html)

**Patient Portal:** New Images category allows patient to view documents stored in designated Images folder(s). [Patient Portal: What the Patient Sees](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7portalpatientsees.html)

**Treatment Plans:**

* Design custom printed treatment plans using Sheets. [Customize Treatment Plans Using Sheets](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetstreatplan.html)
* Create multiple treatment plans (active and inactive). [Treatment Plans](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7treatmentplan.html)
* Treatment plan from the Chart module. [Treatment Planning from the Chart Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7charttp.html)
