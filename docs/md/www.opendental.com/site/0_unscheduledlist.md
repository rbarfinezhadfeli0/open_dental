Unscheduled List

The Unscheduled List is a place to temporarily store appointments that need follow-up (e.g., no-shows, broken appointments).

In [Appointments Module](0_appointments.html), click Lists, **Unscheduled**.

![](images/unschedList.png)

![](images/YouTube_PlayButton_15px.png) See our video: [Managing the Unscheduled List Webinar](https://www.youtube.com/video/IQbk-gicJik).

The list can be narrowed or sorted by specific criteria, inlcuding code range, provider, and more. The list can be easily printed as needed. Use right-click options to quickly select a patient or jump to a chart for more details.
