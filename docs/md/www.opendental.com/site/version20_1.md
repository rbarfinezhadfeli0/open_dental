# Version 20.1

See [Versions](versions.html)

Version 20.1 was released on 05/12/2020.

![](images/YouTube_PlayButton_15px.png) Overview of [Version 20.1](https://opendental.blog/software-update-stable-version-20-1-highlights/).

**Account Module**

* Simplified Income Transfer Manager to allocate all credits to outstanding charges. [Income Transfer Manager](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7incometransfermanager.html)
* Ability to determine when interest begins charging on payment plans. [Old Payment Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentplanpatient.html), [Payment Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7paymentplandynamic.html)
* Create a Dynamic Payment Plan on an Ortho Case. [Ortho Case](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7orthocase.html)

**Chart Module**

* Chart Module preference to force a user to be a provider to sign a note. [Chart Module Preferences](https://www.opendental.com/manual201/modulesetupchart.html)
* Log in added to Ortho Chart. [Ortho Chart](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7orthochart.html)
* Tooth Chart button to clear movements for selected teeth only. [Tooth Movements](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7movements.html)
* 3D Tooth Chart rewritten from scratch to use DirectX 11 instead of DirectX 9. This makes installation easier and it works better.

**eServices**

* Patients can add appointment to their calendar when office is using eConfirmations and Automated Thank-Yous. [Automated Thank-Yous](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7autothankyousetup.html)
* eReminders and eConfirmations can send using the patient's preferred language. [eConfirmations](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7econfirmationsetup.html), , [eReminders](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7ereminderssetup.html)
* Web Sched New Patient allows for restricting appointment types to specific blockouts. [Web Sched New Patient](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7webschednewpatsetup.html)
* Automated thank-you messages can send a when a patient schedules an appointment. [Automated Thank-Yous](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7autothankyousetup.html)

**Images Module**

* Ability to merge Image Categories. [Merge Image Categories](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mergeimagecategories.html)
* Image mounts for radiograph and photo collages. [Imaging Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7images.html), [Mount Definitions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mountdefinitions.html)

**Manage Module**

* Accounting report for Profit and Loss. [Accounting Reports](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7accountingreports.html)
* Updates to Accounting Balance Sheet. [Accounting Reports](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7accountingreports.html)

**Reports**

* Ability to print page range in Audit Trail and User Query windows. [User Query](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7queryoverview.html)
* Report for overpaid payment plans. [Payment Plans Overcharged Report](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7reportdynamicpayplanovercharged.html)

**Security**

* Ability to copy a user. [Security](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7security.html)
* Audit trail entry when merging discount plans. [Audit Trail Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7audittrailpermissions.html), [Merge Discount Plans](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mergediscountplans.html)
* Log off time user override moved to Security. [User Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securityusers.html)
* Patient level audit trail for insurance plans. [Insurance Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7insplan.html)
* New permissions for editing completed procedures. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html), [Audit Trail Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7audittrailpermissions.html)

* EDS Medical added as option for medical clearinghouse. [Electronic Dental Services - Medical](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclaimsedsmedical.html)
* Updated PreXion Bridge and added Prexion Aquire. [PreXion Bridge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeprexion.html), [Prexion Acquire Bridge](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7bridgeprexionaquire.html)
* Enhancements to appearance of Show Features. [Show Features](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7showfeatures.html)
* Showing inactive patients in the Patient Select window defaults to on. [Select Patient](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientselect.html), [Miscellaneous Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7miscsetup.html)
* Prompt before closing Open Dental. [Log On](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securitylogin.html)
* Changes to appearance of Edit Appointment Window. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html)
* Slow Query Log added for troubleshooting. [Slow Query Log Tool](slowquerylog.html)
* Paste a file when using DentalXChange Attachment Service. [DentalXChange Attachment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7dentalxchangeattachmentservice.html)
* Additional information added to the employee list. [Employees](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7employees.html)
* Language translations for sheets. [Edit Sheet Def](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7sheetsetup.html), [Edit Patient Information](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientedit.html)
