Appointment Types

Appointment Types can be used to customize appointment background colors, time patterns, procedures, and more.

In the [Main Menu](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mainmenu.html), click Setup, Appointments, Appointment Types.

![](images/appointmentTypes.png)

The Setup Appointment Types window lists existing appointment types that can be assigned to appointments. Appointment Types can be attached from the Edit Appointment window or when creating a new appointment.

## Appointment Type Edit

Click **Add** to create a new Appointment Type or double-click an existing type to edit.

![](images/appointmentTypeEdit.png)

New appointment types can be created and customized.

* **Name**: Description of the Appointment Type.
* **Color**: This is the color that is used on the schedule for this blockout.
* **Time**: Indicates the length of the Appointment Type.
* **Procedures to Add to the Appointment**: Add procedures that will be automatically treatment planned when you choose this Appointment Type.
* **Required Procedures**: Add procedures that are required when using this Appointment Type.
* **Allowed Blockout Types**: Determine the [Blockouts](0_blockouts.html) where this Appointment Type can be scheduled.
