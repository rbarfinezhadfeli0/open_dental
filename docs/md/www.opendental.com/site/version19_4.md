# Version 19.4

See [Versions](versions.html)

Version 19.4 was released on 04/07/2020.

![](images/YouTube_PlayButton_15px.png) Overview of [Version 19.4](https://youtu.be/vzvZFR87lLY).

**Account Module**

* Income Transfer Manager includes checkbox to include hidden paysplits. [Income Transfer Manager](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7incometransfermanager.html), [Hidden Splits](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7hiddensplits.html)
* Zero dollar payments show in patient account when they make a payment for a patient in a different family. [Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7payment.html)
* New output field for payment plan sheets to show masked credit card number. [Output Text Field Names](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7outputtextfields.html)
* Payment window now has a separate tab for treatment planned procedures. [Payment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7payment.html), [Hidden Splits](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7hiddensplits.html)
* Ortho Case feature. [Ortho Case](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7orthocase.html)

**Appointments Module**

* Preference to force a user to break an appointment before moving, deleting, or sending to the unscheduled list. [Appointments Module Preferences](https://www.opendental.com/manual194/modulesetupappts.html)
* Hygiene time added to appointment time bar. [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html)

**eServices**

* Allow consent forms to be auto-generated in eClipboard. [eClipboard Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclipboardsetup.html)
* View treatment plans on eClipboard. [eClipboard: Treatment Plan](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7eclipboardtreatmentplan.html)
* Ability to add automatic replies when a patient confirms an appointment via eConfirmations. [eConfirmations](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7econfirmationsetup.html)
* Online payments through Patient Portal using Payconnect. [PayConnect Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7payconnectsetup.html), [Patient Portal Settings](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7portalsettings.html)

**Clinics**

* CEMT connections allow option to enable or disable clinic breakdown on reports. [CEMT Connections](cemtconnections.html)
* Provider overrides use default values if no clinic information is entered. [Provider](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7providerseditwindow.html)
* Preference to unhide patients in a restricted clinic. [Enterprise Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7enterprisesetup.html), [Miscellaneous Setup](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7miscsetup.html)
* Fee schedule grouping. [Fee Schedule Groups](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7feeschedulegroups.html)

**Family Module**

* Preference to mask a Social Security Number. [Family Module Preferences](https://www.opendental.com/manual194/modulesetupfamily.html)
* Preference to mask the patient birthdate. [Family Module Preferences](https://www.opendental.com/manual194/modulesetupfamily.html)
* New window to easily add multiple email addresses for a patient. [Edit Patient Information](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientedit.html)
* Added checkbox when consent is given to DoseSpot for access for medical history. [Edit Patient Information: Other Tab](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patienteditother.html)

**Manage Module**

* UNC paths can be immediately accessed from tasks. [Task](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7tasks.html)
* Ability to mark supply orders as received.[Supply Orders](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7supplyorders.html)

**Security**

* Audit trail entry when an insurance payment or write-off is edited. [Claim](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7claimedit.html)
* Audit trail entry when fees are changed in a fee schedule. [Procedure Codes](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodes.html), [Fee Tools](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7feescheduletools.html)
* Audit trail entry when Social Security Number is viewed. [Select Patient](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientselect.html), [Edit Patient Information: Other Tab](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patienteditother.html)
* Audit trail entry when birthdate is viewed. [Edit Patient Information](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7patientedit.html)
* Set logoff time per user. [Global Security Settings](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securitysettings.html), [User Settings](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7usersettings.html)
* Added MySQL Manager to manage MySQL users and passwords. [MySQL User Manager](mysqlusermanager.html)
* MySQL installations allow option for blank password. [MySQL Security](securitymysql.html)
* Security permission for the ability to view Social Security Number. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
* Security permission for the ability to view a patient birthdate. [Permissions](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7permissions.html)
* Permission tree visible in User Edit. [User Edit](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7securityusers.html), [CEMT Security](cemtsecurity.html)

**Treatment Plan Module**

* Preference to prompt for a name when saving a treatment plan. [Treatment Plan Module Preferences](https://www.opendental.com/manual194/modulesetuptreat.html), [Treatment Plan Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7treatmentplan.html)
* Arch specific CDT Codes default to correct arch when treatment planning. [Procedure Code](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7procedurecodeedit.html)
* Added display field for UCR Fee on Category Percentage plans. [Display Fields](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7displayfields.html), [Treatment Plan Module](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7treatmentplan.html)
