Canada Definitions: Appt Procs Quick Add

For [Canada](canada.html) users, in the [Main Menu](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7mainmenu.html), click Setup, Definitions, **Appt Procs Quick Add**.

![](images/canadaApptProcsQuickAdd.png)

The Appt Proc Quick Add definition determines the list of procedures available in the Quick Add section of the [Edit Appointment](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7aptedit.html) window.

This setup is specific to Canada users. Default buttons can be added or reset to default by running the [Canada Procedure Code Tools](canadaproccodetools.html) for *Appt Procs Quick Add*.

See [Definitions: Appt Procs Quick Add](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help253%EA%A4%B7definitionsapptquickadd.html) for additional details.
