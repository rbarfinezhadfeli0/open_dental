Definitions: Claim Error Code

Create claim errors to use when tracking claim status history.

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Claim Error Code.

![](images/definitionsClaimErrorCode.png)

The Claim Error Code definition determines the list of error code options available in the [Edit Claim - Status History Tab](claimtabstatus.html). This list is empty by default. Add Error Codes as needed.

Double-click an existing item or click **Add** to create a new item. The Edit Definition window opens.

![](images/definitionEditClaimErrorCode.png)

* **Name**: Enter the abbreviation for the error code. This displays in the Error Code column of the Claim Custom Tracking Status History grid.
* **Description**: Enter the full description of the error code.
* **Delete**: Remove an error code from the list. Cannot be deleted if the error has been used.

Click **Save** to exit the window and keep changes.
