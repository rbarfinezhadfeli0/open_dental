Definitions: Carrier Group Names

Add customized group names to help organize insurance.

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Carrier Group Names.

![](images/definitionsCarrierGroupNames.png)

The Carrier Group Names definition determines the group options available in the carrier group dropdown when adding or editing [Insurance Carriers](carriers.html). This list is empty by default. Add new group names as needed.

Double-click an existing item or click **Add** to create a new item. The Edit Definition window opens.

![](images/definitionsEditCarrierGroupNames.png)

**Name**: Enter the carrier group name.

**Hidden**: Mark this definition as hidden.

Click **Save** to exit the window and keep changes.
