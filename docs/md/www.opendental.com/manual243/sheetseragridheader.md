ERA Grid Header Layout

The ERAGridHeader shows above every claim paid on a customized ERA Sheet and can be customized to meet an office's needs.

In [Sheets](sheets.html), double-click on a custom sheet with the type of ERAGridHeader.

![](images/sheetseragridheader.gif)

Alternatively, in Sheets:

* Click **New** and choose *ERAGridHeader* as the Sheet Type to create a new ERAGridHeader from blank.
* Select an existing Custom *ERAGridHeader* sheet and click **Duplicate** to create a copy of an existing ERAGridHeader.
* Select the existing Internal *ERAGridHeader* sheet and click **Copy** to create a copy of the default ERAGridHeader that can be edited.

Also see:

* [ERA Sheet](sheetsera.html)
* [Sheet Def Properties](sheetproperties.html)
* [Sheet Field Types](sheetsaddelements.html)

To customize a ERAGridHeader sheet type, see [Edit Sheet Def](sheetsetup.html).
