Cross Code

Medical codes, such as [CPT Codes](cptcodes.html), can be cross coded to dental procedure codes.

## Setup

First, in Lists, [Procedure Codes](procedurecodes.html), click **New** to create add a new [Procedure Code](procedurecodeedit.html) for medical codes as needed.

![](images/crosscodeEditMedCode.png)

After medical codes have been added, enter the medical code in the *Medical Code* on the corresponding dental code.

![](images/crosscodeProcCode.png)

## Charting Codes and Creating Claims

When a *Medical Code* is cross coded to the dental procedure code, only the dental procedure code needs to be charted. If a [Medical Claim](medicalins.html) is created for the dental procedure, the *Medical Code* code is used on the claim instead of the dental *Proc Code*.
