Dental Student Bulk Edit

Update provider colors for multiple Dental School Students at one time.

In [Dental School Providers](dsproviders.html), click Student Bulk Edit.

![](images/dsstudentbulk.png)

For [Dental Schools](dentalschools.html), use the Student Bulk Edit tool to edit the Appointment Color and Highlight Outline Color for multiple students at once. See [Provider](providerseditwindow.html) for more information on these colors.

All students are listed by default. Use the filters to limit the Students list.

* **Classes**: Select a specific [Dental School Class](dsclasses.html) from the dropdown.
* **ProvNum**: Enter a Provider ID to find a specific provider.

Edit Students: Choose the color options to apply to students. Click a colored box to open the [Color](colorselection.html) selector.

Highlight students in the list and click **Save** to apply changes to the selected students.
