Wiki History

Use the Wiki History to view a history of page changes.

In the [Wiki](wiki.html), click **History**.

![](images/wikiHistory.png)

**Page History**: A log of all versions of the page versions. Click an item to display a preview.

**Compare**: Select two items to view changes between versions.
![](images/wikiCompare.png)

**Revert**: Select a version, then click to change the published wiki page back to this version.
