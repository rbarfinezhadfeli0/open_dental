## Major Changes

Added 2025 CDT Codes. [Procedure Code Tools](../autoLogin.aspx%EF%B9%96ReturnUrl%3D%EA%A4%B7help251%EA%A4%B7procedurecodetools.html)

* Also backported to 24.2.48.
* New option to import treatment areas when updating procedure codes.
