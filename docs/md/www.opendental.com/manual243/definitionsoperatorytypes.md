Definitions: Operatory Types

Operatory Types are used to define a group of operatories.

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Operatory Types.

![](images/definitionsOperatoryType.png)

Operatory Types are informational only. This list is blank by default. Add new types as needed. Assign types to an operatory in [Operatory Edit](operatoryedit.html).

Double-click an existing item or click **Add** to create a new item. The Edit Definition window opens.

![](images/definitionsOperatoryTypeEdit.png)

**Name**: Enter the operatory type name.

**Hidden**: Check to hide this type.
