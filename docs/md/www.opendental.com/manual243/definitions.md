Definitions

Definitions allow customization of colors, list options, categories, and some defaults available on various windows.

In the [Main Menu](mainmenu.html), click Setup, Definitions.

![](images/definitions.png)

Definitions are grouped by category. Select a category on the left to view its definitions and current settings. A short description of the category and what the definition controls show in the Guidelines section.

If a category controls list options, types, or categories, the Edit Items buttons are enabled.

* **Add**: Create a new item for the category.
* **Hide**: Highlight an item and click to hide. Hide an item. To unhide, double-click an item and uncheck *Hidden* in the Edit Definition window. Some items cannot be hidden or unhidden.
* **Up/Down**: Highlight an item and click to move the item in the list. This also affects the sort order of the item in pick lists or dropdown menus.
* **Alphabetize**: Sorts the list alphabetically by definition name. Select a category and click Alphabetize. This also affect the sort order of the item in pick lists and dropdown menus.
