Definitions: Prognosis

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Prognosis.

![](images/definitionsPrognosis.png)

The Prognosis definition determines the list of prognosis options available when [Entering Treatment](entertreatment.html). This list is empty by default. Add items as needed.

Double-click an existing prognosis to edit or click **Add** to create a new prognosis. The Edit Definition window will open.

![](images/definitionEditPrognosis.png)

**Name**: Enter the prognosis description.

Click **Save** to keep changes.
