ODMobile Reports

Review Standard Reports from the mobile device.

In the [ODMobile Menu](opendentalmobilemenu.html), tap **Reports**.

![](images/mobileReports.png)

ODMobile includes some of the [Standard Reports](reportsstandard.html) available from Open Dental.

## Run a Report

1. Tap a section to expand report options.
   ![](images/mobileReportsExpand.png)
2. Tap a report. A prompt displays explaining to verify filters.
   ![](images/mobileReportsMain.png)
3. Tap the gear icon to enter filters related to the selected report.

4. Verify report filters. Tap a filter to make changes.
   ![](images/mobileReportsFilters.png)
    Tap **Done** to continue.

5. Tap the run icon to retrieve report data.

6. The report displays. Increase the font size as needed using the **-** and **+** icons. Scroll left and right to view additional report columns.
   ![](images/mobileReportsReport.png)
