EHR Vaccine Setup

In [EHR](ehrsetupwindow.html), click Vaccine Def.

![](images/vaccinedef.gif)

[EHR Vaccine Data](ehrvaccine.html) must be defined before vaccines can be entered for patients.

Note: Before vaccines can be defined, you must import CVX codes (see [Importing Code Systems](codesystemimport.html)) and set up [EHR Vaccine Drug Manufacturers](ehrvaccinedrugman.html).

Click Add, or double click an existing vaccine to edit.

![](images/vaccinedefEdit.gif)

Enter the vaccine information.

1. Click **[...]** to select a CVX code.
2. Enter the **Vaccine Name**.
3. Click the drop down arrow to select the [EHR Vaccine Drug Manufacturers](ehrvaccinedrugman.html).
4. Click OK to save.
