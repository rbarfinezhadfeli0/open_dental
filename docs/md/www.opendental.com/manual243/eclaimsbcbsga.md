BCBS Georgia E-Claims

BCBS of Georgia is a E-Claims Direct Carrier.

In [Clearinghouses](clearinghouses.html), double-click **BCBS GA**.

![](images/eclaimsBCBSGA.png)

Website: [http://www.bcbsga.com/home-providers.html.](http://www.bcbsga.com/home-providers.html)

## Setup

Default settings are preset in this window. Verify settings before sending claims.

Ensure the correct payer ID is entered in the **Payors** box when sending claims to this payer.
