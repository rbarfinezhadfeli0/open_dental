Definitions: Time Card Adj Types

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Time Card Adj Types.

![](images/definitionsTimeCardAdjTypes.png)

The Time Card Adj Types definition determines the list of [Time Card](timecardmanage.html) PTO Type options available in Time Card Adjustments. This list is empty by default. Add new types as needed.

Double-click an existing item to edit or click **Add** to create a new item. The following Edit Definition window opens.

![](images/definitionsEditTimeCardAdjTypes.png)

Enter the adjustment **Name**. The name appears on the Out column in the [Time Card](timeclockedit.html).

Click **Save** to keep changes.
