Edit Clearinghouse or Direct Carrier

Use the Edit Clearinghouse or Direct Carrier window to setup a clearinghouse or direct carrier to send electronic claims.

In [Clearinghouses](clearinghouses.html), click **Add** or double-click an existing clearinghouse.

![](images/clearinghouseEdit.png)

Most clearinghouses available in Open Dental already exist in the Clearinghouses list. Only create a new clearinghouse when directed by Open Dental or the clearinghouse. Not all fields are used for each clearinghouse. Refer to the specific clearinghouse for setup information and usage.

## Payers

This area is used when specific Payer IDs should be sent through a clearinghouse other than the default.

![](images/clearinghousePayor.png)

Enter each Payer ID to send through the specified clearinghouse or direct carrier. IDs should be separated by a comma with no space (e.g., 12345,67890).
