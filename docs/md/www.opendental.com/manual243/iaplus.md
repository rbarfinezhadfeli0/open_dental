Insurance Answers Plus

Note: This bridge no longer works with newer versions of Open Dental as of November 2017.

In the [Main Menu](mainmenu.html), click Setup, [Program Links](programlinks.html). Double-click on **Insurance Answers Plus from www.iaplus.com**.

![](images/bridgeIAP.png)

Open Dental can bridge to the desktop version of Insurance Answers Plus (IAP): [www.insuranceanswersplus.com](http://www.insuranceanswersplus.com/). IAP is currently only available in some states. Contact them for exact location availability.

To install IAP, copy iaplus10.dll from the IAPlus installation directory (usually **C:\IAPlus**) to **C:\Program Files\Open Dental\** folder.

Once the program is installed, enable the program link in Open Dental:

1. Check the **Enabled** box.
2. Verify the **Path of file to open**.
3. Click Save. The IAP button appears on the [Insurance Plan](insplan.html).
