Definitions: Insurance Verification Status

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Insurance Verification Status.

![](images/definitionsInsVerifyStatus.png)

The Insurance Verification Status definition determines the status options available in the [Insurance Verification List](insverifylist.html).

Double-click an existing item or click **Add** to create a new item. The Edit Definition window opens.

![](images/definitionsInsVerifyStatusEdit.png)

**Name**: Give the status a name.

**Hidden**: Check to hide this status.
