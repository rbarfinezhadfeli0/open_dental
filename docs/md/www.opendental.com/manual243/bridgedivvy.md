Divvy Bridge

In the [Main Menu](mainmenu.html), click Setup, [Program Links](programlinks.html). Double-click on **Divvy**.

![](images/bridgeDivvy.png)

Divvy automates direct mail messages.

To enable the bridge:

1. Check the **Enabled** box.
2. Double-click on a row in **Additional Properties** to change settings.
   1. Enter the Divvy **Username**.
   2. Enter the Divvy **Password**.
   3. Enter the **API Key**
   4. Enter a value of **DesignID for Recall Cards**.
3. If using clinics, click **Hide Button for Clinics**. Select which clinics the button should display on.
4. Click Save.
