Procedure Buttons

Procedure buttons allow users to quickly add commonly-used procedures to a patient's chart with a single click.

In the [Main Menu](mainmenu.html), click Setup, Chart, Procedure Buttons.

![](images/procButtonQB.gif)

Procedure buttons are located in the [Chart Module](chart.html) on the [Enter Treatment](entertreatment.html) tab. Users will usually simply select a tooth, then click a procedure button to chart the treatment.

There are two types of procedure buttons:

* [Quick Procedure Button](quickprocedurebutton.html): Designed to quickly chart fillings and sealants on teeth. Each quick button can be associated with one procedure, and if applicable, to tooth surfaces.
* Procedure Button: Can be associated with multiple [Procedure Codes](procedurecodes.html) and [Auto Codes](autocode.html).

![](images/YouTube_PlayButton_15px.png) See our video: [Procedure Buttons, Quick Buttons, and Auto Codes Webinar](https://www.youtube.com/watch?v=QmAf8Ux0f-I).

When opening the window, the Quick Buttons category is selected (see screenshot above). When another category is selected, the window displays as shown below.

![](images/procButtonOther.png)

* **Edit Categories**: Rename, reorder, or add new categories. See [Definitions: Proc Code Categories](definitionsproccodecategories.html).
* **Add**: Create a new procedure button. Opens the [Procedure Button Edit](procedurebuttonedit.html) window.
* **Delete**: Delete the selected procedure button.
* **Up / Down**: Reorder buttons within a category.
