Installation

See [Technical](technical.html).

Install the [Trial Version](../trial.html) to evaluate Open Dental.

Customers on [Support](../site/fees.html) should [Update to Full Version](updatefullversion.html).

To add a new workstation see [Install on Workstation](installworkstation.html).

Microsoft [dot Net ( .NET )](dotnet.html) is required to use Open Dental.

[Open Port 3306](port3306.html) on the Server.

[Use Blank Database](usingodblank.html) is a good starting point for new installs.

[Troubleshooting](troubleshooting.html) is a helpful resource covering common issues with the software. You can also [Contact Us](../contact.html) for assistance.
