Referral Slip Layout

The referral slip sheet is used for patient and provider referrals and can be customized to meet the needs of the practice.

In [Sheets](sheets.html), double-click on a custom sheet with the type of ReferralSlip.

![](images/sheetReferralSlip.gif)

Alternatively, in Sheets:

* Click **New** and choose *ReferralSlip* as the Sheet Type to create a new Referral Slip from blank.
* Select an existing Custom *ReferralSlip* sheet and click **Duplicate** to create a copy of an existing Referral Slip.
* Select the existing Internal *ReferralSlip* sheet and click **Copy** to create a copy of the default Referral Slip that can be edited.

Also see: [Sheet Def Properties](sheetproperties.html) and [Sheet Field Types](sheetsaddelements.html).

The internal referral slip sheet shown above is the default referral slip. Custom referral slips can be created for different referral sources. Assign the custom sheet to a referral source from the [Referral List](referralsetup.html).

To customize a Referral Slip, see [Edit Sheet Def](sheetsetup.html).

[Referral Slips](referralslips.html) support [Electronic Signatures](electronicsignatures.html) and can be added to the sheet.
