RAMQ E-Claims

Ramq is a Canadian clearinghouse. It is not yet available.

In the [Main Menu](mainmenu.html), click Setup, Family/Insurance, [Clearinghouses](clearinghouses.html). Double-click on RAMQ.

![](images/eclaimsRAMQ.png)

This bridge in [Clearinghouses](clearinghouses.html) is only partially complete as of version 17.1. Claims can be output to a file (claim export path).

Uploading files to the RAMQ servers is still in development.
