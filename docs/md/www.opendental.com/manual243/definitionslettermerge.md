Definitions: Letter Merge Cats

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Letter Merge Cats.

![](images/definitionsLetterMergeCats.png)

The Letter Merge Cats definition determines the categories options available for the [Letter Merge](lettermerge.html) feature. This list is empty by default. Add categories as needed.

Double-click an existing item or click **Add** to create a new item. The Edit Definition window opens.

![](images/definitionEditLetterMergeCats.png)

**Name**: Give the category a name.

**Hidden**: Check to hide this category.
