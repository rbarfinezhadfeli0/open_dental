Definitions: Treat' Plan Priorities

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Treat' Plan Priorities.

![](images/definitionsTreatPlanPriority.png)

The Treat' Plan Priorities definition determines the priority options available to assign [Procedures](procedureedit.html).

Assign a priority to a procedure in the procedure info window, [Treatment Plan Module](treatmentplan.html), or Chart Module - [Enter Treatment](entertreatment.html) tab. Set the order of priorities in this definition. The priority affects the order of procedure in the [Edit Appointment](aptedit.html) window and [Chart Module](chart.html) Progress Notes, as well as the order and procedure text color on Treatment Plans.

Double-click an existing item to edit or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionsTreatPlanPriorityEdit.png)

Name: Enter a description for the priority.

Color: Click the box to select the [Color](colorselection.html) used for procedure text in the Treatment Plan.

Click **Save** to keep changes.
