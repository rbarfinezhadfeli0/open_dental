Canada Auto Codes

For [Canada](canada.html) users, in the [Main Menu](mainmenu.html), click Setup, Auto Codes.

![](images/canadaAutoCodes.png)

Auto codes work behind the scenes to insert and validate the correct procedure code depending on a variety of tooth conditions.

* Auto codes can be associated with [Procedure Buttons](procedurebuttons.html).
* Default auto codes are added via [Canada Procedure Code Tools](canadaproccodetools.html).
* For details on fields and setting up additional auto codes, see [Auto Codes](autocode.html).

## Add or Edit Auto Codes

In the Edit Auto Code window, click **Add**, or double-click an auto code to edit. All procedure codes associated with the auto code list.

![](images/canadaAutoCodesEdit.png)

**Add**: Click to add an additional procedure code and conditions to the auto code. The following window opens:

![](images/canadaAutoCodesItemEdit.png)

Click OK to save the auto code.
