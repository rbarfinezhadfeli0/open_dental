Manage Module

The Manage [Module](modules.html) contains miscellaneous management functions that do not apply to any single patient.

![](images/manage.png)

Also see [Webinars](../webinars/webinars.html).

## Daily Area

![](images/manageDaily.png)

* **Send Claims:** Open [Insurance Claims](claimsend.html) window to send or print insurance claims waiting to be sent or view Claim History.
* **Batch Ins:** Enter [Batch Insurance Payments](claimpaymentbatch.html).
* **Billing:** Print or email batch statements and set billing defaults. See [Billing](billing.html).
* **Deposits:** Create and manage [Deposit Slips](depositslip.html).
* **Supply Inventory:** Open [Supply Inventory](supplyinventory.html) to track suppliers, supplies, orders, and equipment.
* **TSI Collections:** Open [TSI Collections](tsicollections.html).
* **Tasks:** Opens the [Tasks Area](tasksarea.html) in a separate window. More than one task list can be open at a time.
* **Backup:** Use Open Dental's manual [Backup Tool](backuptool.html).
* **Accounting:** Open the [Accounting](accounting.html) feature used to set up a chart of accounts and track expenses, income, assets, liabilities and equity.
* **Email Inbox:** Open the [Email Client](emailinbox.html).
* **ERAs:** Open the [ERAs](claimsera835.html) window to process and manage ERAs.
* **Import Ins Plans:** Only visible when [Public Health](publichealth.html) is turned on. This is a very specific function for importing 834 files. [Import Ins Plan 834](importinsplans834.html).

## Time Clock

![](images/manageTimeClock.png)

Employees can use the [Time Clock](timeclock.html) to clock in or out of work..

## Messaging

![](images/manageMessaging.png)

Send internal text, light, and sound messages to all workstations in the office. See [Messaging](messaging.html) for additional details.
