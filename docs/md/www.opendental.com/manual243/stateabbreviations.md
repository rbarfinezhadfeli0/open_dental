State Abbreviations

The State Abbreviations list contains U.S. states and standard abbreviations by default, but can be edited or added to.

In the [Main Menu](mainmenu.html), click Lists, State Abbreviations.

![](images/stateAbbreviations.gif)

State abbreviations are used to enter state information in the [Edit Patient Information](patientedit.html) and [Add Family](familyadd.html) windows. Typing the first letter of the state will prompt a suggestion box of all state abbreviations that match the first entered letter.

![](images/stateAbbrevList.gif)

You may enter any description and abbreviation of your choosing, but we recommend using the standard abbreviations for states, providences, countries, etc. To add or edit state abbreviations, a user must have the *Setup* security permission.

Click Add, or double-click an existing state to edit.

![](images/stateAbbreviationEdit.gif)

* **Description**: Enter an identifying name.
* **Abbreviation**: Enter the abbreviation that will show as a selection for this state. If State is set as a [Required Field](requiredfields.html), the entered state must match a valid two-letter abbreviation in this list.
* **Medicaid ID Length**: Enter the number of digits in the Medicaid ID for this state. This field only shows if validation for Medicaid ID has been turned on in Required Fields setup.

Click **OK** to save.
