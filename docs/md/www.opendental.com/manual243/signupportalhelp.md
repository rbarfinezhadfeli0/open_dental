eServices Signup Help

The eServices Signup Portal allows users to quickly sign up for and change [eService](../site/eservices.html) preferences.

There are five tabs in the Signup Portal.

## Signup

The [eServices Signup](eservicessignup.html) tab is used to enable or disable an eService feature.

* [Basic View](signupportalbasic.html): The default view for single locations.
* [Advanced View](signupportaladvanced.html): The default view when using clinics.

## More Settings

Use the More Settings tab to select [Patient Portal Settings](portalsettings.html).

## About

The [eServices Sign Up About](signupportalabout.html) tab displays information regarding eServices and the registration key. It is primarily used by Open Dental support when troubleshooting.

## Contact Us

Click the Contact Us tab to open [Contact Information](../contact.html) for Open Dental Support.

## Help

Click the Help tab to view this page.
