Lantek

Lantek is an E-Claims clearinghouse for Puerto Rican offices.

In [Clearinghouses](clearinghouses.html), double-click **Lantek Networks Transmission Service**

![](images/clearinghouseLantek.png)

## Setup

Default settings are preset in this window. Set up additional fields as follows:

* **The information below identifies the sender**: Selected.
* **Tax ID Number**: Enter the Lantek account number.
* **Name**: Enter doctor or practice name.
* **Telephone Number**: Enter the practice telephone number.
* **Login ID / Password**: Enter the Lantek credentials.
