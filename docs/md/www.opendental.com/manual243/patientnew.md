Add Patient

Add individual patients or families to Open Dental as needed.

In [Select Patient](patientselect.html), click **Add Pt**.

![](images/patientAddSearch.png)

**Add Pt**: Click to add an individual patient. A last name must have already been searched before the [Edit Patient Information](patientedit.html) window opens. This is to avoid duplicate patient accounts.

**Add Many**: Click to [Add Family](familyadd.html).

When adding a new patient from the Select Patient Window, the following fields carry over:

* First Name
* Last Name
* Birthdate (mm/dd/yyyy)

## Existing Family

To add a new patient to an existing family:

In the [Family Module](family.html), select an existing patient in the family.
![](images/patientAddFamily.png)

In the Toolbar, click **Add**. Enter patient details on the Edit Patient Information window.

When adding a new family member, the following fields carry over from the currently selected family member:

* Wireless Phone
* Work Phone
* Email
* Text Message OK status
* Appt Text status
