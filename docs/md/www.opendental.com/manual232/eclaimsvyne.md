Vyne Dental E-Claims

In [Clearinghouses](clearinghouses.html), double-click Vyne Dental.

![](images/eclaimsVyne.png)

Vyne Dental is a clearinghouse option that provides electronic claim processing. Visit [www.vynedental.com](https://vynedental.com/) to learn more about their services.

## Open Dental Setup

To enable Vyne Dental as the default clearinghouse, set Vyne Dental as the Dental default in the Clearinghouses window.

In the Edit Clearinghouses window, set the following.

![](images/eclaimsVyneSettings.png)

Enter office information.

* **Tax ID Number**: Enter the office Tax ID number.
* **Name**: Enter the office name.
* **Telephone Number**: enter the office phone number, without any formatting (e.g., 5035551234).

Other information in the clearinghouse setup should match the screenshot above.

## Renaissance Users

Some Vyne users may be using the [Renaissance E-Claims](eclaimsrenaissance.html) clearinghouse in Open Dental. Vyne Dental is an updated version and users can switch at their convenience.

All new Vyne users should use the Vyne Dental clearinghouse.

Vyne Dental is moving all existing Renaissance users to Vyne Dental. If claims stop sending while using Renaissance, contact Open Dental Support for assistance moving to Vyne Dental. An Open Dental update may be required.
