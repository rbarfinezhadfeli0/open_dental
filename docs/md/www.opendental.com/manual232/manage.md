Manage Module

The Manage [Module](modules.html) contains miscellaneous management functions that do not apply to any single patient.

![](images/manage.png)

Also see [Webinars](../webinars/webinars.html).

## Daily Area

![](images/manageDaily.png)

* **Send Claims:** Send, print, and manage insurance claims. See [Send Claims](claimsend.html).
* **Batch Ins:** Enter batch insurance payments. See [Batch Insurance Payment](claimpaymentbatch.html).
* **Billing:** Print or email batch statements and set billing defaults. See [Billing](billing.html).
* **Deposits:** Create [Deposit Slip](depositslip.html).
* **Supply Inventory:** Track suppliers, supplies, orders, and equipment. See [Supply Inventory](supplyinventory.html).
* **TSI Collections:** See [TSI Collections](tsicollections.html).
* **Tasks:** Opens the [Tasks Area](tasksarea.html) in a separate window. More than one task list can be open at a time.
* **Backup:** Use Open Dental's manual [Backup Tool](backuptool.html).
* **Accounting:** Set up a chart of accounts and track expenses, income, assets, liabilities and equity. See [Accounting](accounting.html).
* **Email Inbox:** Launch the [Email Client](emailinbox.html).
* **ERAs:** Process [ERAs](claimsera835.html).
* **Import Ins Plans:** Only visible when [Public Health](publichealth.html) is turned on. This is a very specific function for importing 834 files. [Import Ins Plan 834](importinsplans834.html).

## Time Clock

![](images/manageTimeClock.png)

Employees can use the Time Clock to clock in/out of work. See [Time Clock](timeclock.html) for details.

## Messaging

![](images/manageMessaging.png)

Send internal text, light and sound messages to all workstations in the office. See [Messaging](messaging.html).
