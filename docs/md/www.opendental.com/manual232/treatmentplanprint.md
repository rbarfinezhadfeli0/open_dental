Print Treatment Plan

In the [Treatment Plan Module](treatmentplan.html), highlight a saved treatment plan, then click **Print TP**.

![](images/TPprint.gif)

The example above uses the internal treatment plan sheet.

To customize the printed treatment plan, see [Treatment Plan Layout](sheetstreatplan.html).

Additionally, set the following [Preferences](preferences.html):

* *Show completed work on the graphical tooth chart*
* *Itemize Treatment Plan*

In [Definitions: Image Categories](definitionsimagecat.html), set the default Images category for PDF copies of printed treatment plans.
