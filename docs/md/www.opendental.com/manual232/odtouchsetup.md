ODTouch Setup

See [eServices Setup](eservicessetup.html).

ODTouch is an application designed to navigate patient appointments via a convenient tablet device.

Note: ODTouch is currently in Limited Beta. Contact Open Dental Support for signup details.

## Set Up ODTouch in Open Dental

1. Sign up for ODTouch in the [eServices Signup Portal](eservicessignup.html).
2. Set up [Mobile Settings](mobilesettings.html) to use when logging into the app. This only needs done once and is used for all Open Dental apps.
3. Determine [ODTouch Security](odtouchsecurity.html) settings in Open Dental.
4. Set up [eRouting](erouting.html) definitions.
5. Set up [Users](securityusers.html) to log into ODTouch.
   * Users must have the *Mobile Web/ODMobile* security permission.
   * *Passwords must be strong* must be enabled in [Global Security Settings](securitysettings.html).
   * Users must have a strong password.

## Install ODTouch on a Tablet Device

ODTouch is designed for tablet devices, not mobile phones. See [Tablet](../site/tablet.html) for compatible devices and details.

1. Install the app to each desired device. ODTouch is available for iOS and Android.
   * [App Store](https://apps.apple.com/us/app/odtouch/id6444294687)
   * [Google Play](https://play.google.com/store/apps/details?id=com.opendental.clinical)
2. In the app, enter the Mobile Settings to connect the app with the database.
   ![](images/odtouchSignupLogin.png)
3. In Open Dental, enable the device in the [Device Manager](devicemanager.html).
4. In the app, log in using an Open Dental username and password.
   * If *Manually enter log on credentials* is enabled in [Global Security Settings](securitysettings.html), the User must be typed instead of selected.
   ![](images/odtouchUserLogin.png)
5. If *Clinical Security* is enabled in ODTouch Security, determine security options in [ODTouch: Settings](odtouchsettings.html).
