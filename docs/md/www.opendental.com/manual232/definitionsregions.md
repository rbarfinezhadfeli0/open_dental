Definitions: Regions

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Regions.

![](images/definitionsRegion.png)

The Regions definition determines the list of regions options available to assign to [Clinics](clinicsetup.html). This list is empty by default. Add new regions as needed.

This helps identify clinics and can be used to filter the [Insurance Verification List](insverifylist.html).

Double-click an existing item or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionsEditRegion.png)

Enter the region **Name** then click OK to save.
