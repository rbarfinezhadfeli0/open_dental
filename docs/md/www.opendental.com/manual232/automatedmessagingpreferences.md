Automated Messaging Preferences

The Automated Messaging Advanced Settings houses settings regarding multiple Automated Messaging features.

In [Automated Messaging](automatedmessaging.html), click **Preferences**.

![](images/automatedMessagingPreferences.png)

**Clinic**: Select the clinic the web forms are applicable to. Select Defaults to use the same web forms for all clinics.

**Web Forms**: Highlight the web forms patients should fill. Required when using the [NewPatWebFormURL] tag. When the [NewPatWebFormURL] tag is entered on the New Patient Thank-You message in the [Edit Rule](editrule.html) window, patients are sent to the selected web forms.

**Patients under age 18 - Send to Guarantor**: Determine if patients under the age of 18 should receive automated messages.

* Checked: Send automated messages to the Guarantor of the account, instead of the patient when the patient is under the age of 18.
* Unchecked: Send automated messages to the patient, even if the patient is under the age of 18.
