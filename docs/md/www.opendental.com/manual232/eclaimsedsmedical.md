Electronic Dental Services - Medical

In [Clearinghouses](clearinghouses.html), double-click **EdsMedical**.

![](images/clearinghouseEDSMedical.png)

EDS Medical is an e-claims clearinghouse that offers medical claims. Users must also be using [EDS Dental](eclaimseds.html) in order to use this clearinghouse.

## Setup

Enter the following details when enabling EDS Medical:

* **Login ID**: Supplied by EDS Medical.
* **Password**: The EDS Medical password.
* **Comm Bridge**: Set to *EdsMedical* if not already done.
