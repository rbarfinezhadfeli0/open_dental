EHR Set Meaningful Use Stage

The meaningful use (MU) stage setting determines which measures list on the [EHR Dashboard](ehrdashboard1.html) and in [EHR Measure Reports](ehrmeasures.html). Stage settings can be set globally or by provider.

## Setting globally

The global setting affects all providers whose EHR Meaningful Use setting on the Edit Provider Window is 'Use Global'.

1. In the [Main Menu](mainmenu.html), click Setup, Chart, EHR.
   ![](images/ehrMUSettings.gif)
2. In the upper left corner, click Settings.
   ![](images/ehrSettingsMUStage.gif)
3. Under Global Settings, click the dropdown to select the stage.There are three options.
   * Stage 1
   * Stage 2
   * [EHR Modified Stage 2](../site/ehrmodified2.html) (recommended for all providers in 2017)
4. Click OK to save.

## Setting by provider

Stage can also be set by provider. When the provider is the primary provider, the EHR dashboard and measure reports will reflect their EHR Meaningful Use stage setting.

1. In the main menu, click Lists, [Providers](providers.html).
2. Double click the provider.
3. For EHR Meaningful Use, click the dropdown to select the provider's stage.
   * Use Global: Use the global setting on the [EHR Settings](ehrsettings.html) window. This is the default.
   * Stage 1
   * Stage 2
   * Modified Stage 2 (recommended for all providers in 2017).
4. Click OK to save.
