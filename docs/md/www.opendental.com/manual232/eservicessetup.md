eServices Setup

In the [Main Menu](mainmenu.html), click **eServices**.

![](images/eservicesSetup.png)

Sign up and setup for [eServices](../site/eservices.html) is managed from the eServices Setup window.

* **Signup**: Sign up for eServices using the Signup Portal. [eServices Signup](eservicessignup.html).
* **eConnector Service**: Install and monitor the [eConnector](econnector.html). This service is used by, and required for all eServices hosted by Open Dental. It only needs installed once.
* **Manage Devices**: Open the [Device Manager](devicemanager.html) for use with ODTouch and eClipboard.
* **Mobile Web**: Set up options for the Mobile Web. [Mobile Web](mobilewebhostedbyod.html).
* **ODTouch**: Determine settings for [ODTouch Security](odtouchsecurity.html).
* **Patient Portal**: Set up options for [Patient Portal](portalhostedbyod.html) (when hosted by Open Dental).
* **Payment Portal**: Copy the link used for [Payment Portal](paymentportalsetup.html).
* **Texting Services**: View usage summaries for Integrated [Texting](textintegratedusage.html).
* **Automated Messaging**: Set up options for [Automated Messaging](automatedmessaging.html).
* **eClipboard**: Set up options for [eClipboard](eclipboardsetup.html).
* **Miscellaneous**: Set [Miscellaneous](eservicesmisctab.html) settings, such as the run time of automated emails and text messages for Web Sched Recall reminders, eReminders, and eConfirmations.
* **Secure Email**: Access [Secure Email Setup](secureemailsetup.html).
* **Mobile Settings**: Create [Mobile Settings](mobilesettings.html) for eClipboard, ODMobile, and ODTouch.

Web Scheduling:

* **Recall**: Set up options for [Web Sched Recall](webschedrecallsetup.html).
* **New Patient**: Set up options for [Web Sched New Patient](webschednewpatsetup.html).
* **Existing Patient**: Set up options for [Web Sched Existing Patient](webschedexistingpatsetup.html).
* **Advanced**: Set up URL options. See [Web Sched Advanced](webschedadvanced.html).

Other eService setup includes [ODMobile](opendentalmobile.html) and [ODTouch Setup](odtouchsetup.html).
