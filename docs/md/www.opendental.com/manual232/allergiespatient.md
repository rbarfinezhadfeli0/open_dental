Allergies

Allergy information can be entered for each patient.

In the [Medical](medical.html) window, click the Allergies tab.

![](images/medicalAllergies.png)

**Add Allergy**: Add an allergy from the [Allergy List](allergiesmaster.html).

**Show Inactive Allergies**: Check to show allergies that have been marked inactive.

**Allergies**: The grid displays a list of allergies entered for the patient. The [EHR InfoButton](ehrinfobutton.html) may show if EHR is turned on.

Active allergies show in the Chart Module - Patient Info area in bold red. Every time a user adds, edits or deletes a patient allergy, an entry is made in the [Audit Trail](audittrail.html).

## Allergy Edit

Click **Add Allergy**, or double-click an existing allergy to edit.

![](images/allergyEdit.png)

* **Allergy**: Select the allergy. Only allergies added to the master Allergy List are available. If needing to add an allergy that is not in the dropdown, add it to the Allergy List first.
* **SNOMED CT Reaction**: (Optional) Click **[...]** to select [SNOMED CT Codes](snomedct.html) (sent in the XML of a [EHR Continuity of Care Document ( CCD )](ehrccd.html)/summary of care). Click **None** to clear the box.
* **Reaction Description**: (Optional) Enter the allergic reaction.
* **Date Adverse Reaction**: (Optional) Enter the date of the reaction.
* **Is Active**: By default the box is checked. To mark an allergy inactive, uncheck the box.
* **Delete**: Click to remove this allergy from the patient.

Click **OK** to save changes.
