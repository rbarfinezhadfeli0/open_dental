Mobile Web

In [eServices Setup](eservicessetup.html), click **Mobile Web**.

Note: Mobile Web has been replaced by [ODMobile](opendentalmobile.html), an app that is installed right to your mobile device! New features will not be added to Mobile Web, however it can still be used.

![](images/MobileWebSetup.gif)

To begin using Mobile Web, first [Sign up](eservicessignup.html) for the service.

Note: Security [Permissions](permissions.html) granted for the user also applies to the user when using Mobile Web.

## Set up Mobile Web Users

To access Mobile Web, the following is required:

* The *Mobile Web/ODMobile* security permission. See [User Group](securityusergroups.html).
* *Passwords must be strong* must be enabled in [Global Security Settings](securitysettings.html).
* Users must have a strong password. See [Change Password](changepassword.html).

A entry is logged in the audit trail every time a user logs into the Mobile Web.

## Get the Mobile Web URL

To access your Mobile Web, use the hosted URL that is linked to your unique Open Dental registration key. To find the URL:

* In the the window above, right-click on the Hosted URL, and copy it to the clipboard.
* In the [Signup Portal](signupportalbasic.html), hover over Mobile Web. To copy the URL to the clipboard, click the button to the left of the field.
