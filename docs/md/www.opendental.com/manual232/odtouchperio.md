ODTouch: Perio

View and create perio charts for the patient.

From the [ODTouch: Home Screen](odtouchhome.html), tap **Perio**.

![](images/odtouchPerioHome.png)

This screen is not available when using a mobile phone device.

All perio exams for the patient list. Tap the slider on an exam to delete it.

Tap an exam to open it, or tap **+** to create a new exam.

![](images/odtouchPerio.png)

Use the keypad to enter measurements. Tap the caret to view additional options:

* **Voice Command**: Tap to use voice commands to enter perio measurements. See [Voice Perio Charting](periochartingvoice.html) for a list of available commands.
* **More Options**: Tap to view additional options.
  + **Settings**: Determine different perio charting settings. For details on settings, see [Perio Chart](perio.html).
    ![](images/odtouchPerioSettings.png)
  + **Summary**: View a read-only summary of the perio chart.
    ![](images/odtouchPerioSummary.png)
  + **Graphical**: View a graphical summary of the perio chart.
