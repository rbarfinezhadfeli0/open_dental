ASAP List

The ASAP List shows patients who would like to be contacted when an earlier appointment becomes available.

In [Appointment Lists](appointmentlists.html), click **ASAP**.

![](images/asapList.png)

The ASAP List can remain open while other areas of Open Dental are open. Use the list to identify patients and quickly notify them about available openings. Appointments can also be scheduled from here.

![](images/YouTube_PlayButton_15px.png) See our video: [ASAP List](https://www.youtube.com/playlist?list=PLyjZ9PCVynClCgYyiViHUQRMECyrtKE6l).

Note: Sign up for these eServices to easily send and manage ASAP notifications:

* [Web Sched ASAP](../site/websched.html): Send ASAP messages about available openings and allow patient to reschedule online.
* [Integrated Texting Feature](../site/textintegrated.html): Send text messages about ASAP openings.

Double-click an appointment to view appointment details. Right-click an appointment to view quick options:

* Select Patient: Open the patient's record with the current module active.
* See Chart: Open the patient's record with the Chart Module active.
* Send to Pinboard: Send the appointment to the Pinboard for scheduling.
* Remove from ASAP: Remove the appointment's ASAP status, thus removing it from the ASAP List.

There are 4 ways to place a patient on the ASAP List:

* Mark a scheduled appointment as ASAP. See [Edit Appointment](aptedit.html).
* Mark a Planned Appointment as ASAP. See [Planned Appointments](apptplanned.html).
* Mark an Unscheduled Appointment as ASAP. In the [Unscheduled List](unscheduled.html), double-click the appointment to mark it as ASAP.
* Mark Recall for a patient as ASAP. See [Recall](recall.html) or [Recall List](recalllist.html).

## Settings

To narrow results on the Appointments or Unschedled Recalls tab, select other filter criteria then click **Refresh** to update.

* **Provider**: Only view appointments for a specific [Provider](providers.html).
* **Clinic**: Only view appointments for a specific [Clinic](clinics.html).
* **Site**: Only view appointments for a specific site. See [Site List](sites.html).
* **Code Range**: Only view appointments with specified procedures codes attached.

**Text**: Only available when Integrated Texting is enabled. See Send an ASAP Text Message below.

**Web Sched ASAP Sign up/ Send / History**:

* **Sign up**: Sign up for [Web Sched ASAP](../site/websched.html) and/or [Integrated Texting Feature](../site/textintegrated.html). See [eServices Signup](eservicessignup.html). Once Web Sched ASAP is turned on, Send/History buttons show instead.
* **Send**: Send is only an option when the ASAP List is accessed by right clicking on an available time slot in the Appointment schedule and select Text ASAP List. See [Web Sched ASAP](webschedasap.html).
* **History**: View [Web Sched ASAP History](webschedasaphistory.html).

**Print List**: Print the list of the open tab (Appointments or Unscheduled Recalls).

## Appointments Tab

![](images/asapListApptTab.png)

By default, the Appointments tab opens. This lists scheduled, unscheduled, planned, and broken appointments that have been marked ASAP.

* **Appointment status**: Only view appointments with selected appointment status.
* **Show**: Filter the list by:
  + **All Appointments**: Display all appointments regardless of procedure or appointment type.
  + **Non-Hygiene Appointments**: Display only appointments not marked *Is Hygiene*. See [Edit Appointment](aptedit.html).
  + **Hygiene Appointments**: Display only appointments marked *Is Hygiene*.

## Unscheduled Recalls Tab

![](images/asapListRecallsTab.png)

Click the Unscheduled Recalls tab to view patients with a recall marked *Schedule ASAP* on the [Edit Recall](recall.html) window, even if they have received a recall reminder already.

* **Start / End Date**: Determines which appointments list based on recall due date. Defaults to the current week.
* **Group Families**: Group members within a family together. Defaults to the setting in [Setup Recall](recallsetup.html).
* **Show Reminders**: Only list patients with a specific number of recall reminders.

## Send an ASAP Text Message

To enable this option, sign up for Integrated Texting.

1. (Optional) In the ASAP List, highlight the patient(s) to send a text message to.

   Note: If patients are selected before clicking Text, only those patients will be texted. If patients are not selected first, all patients with a wireless number are auto-selected.
2. Click Text.
   ![](images/webSchedASAPManualText.png)
3. Enter the date and time of the available appointment, then click OK.
   ![](images/webSchedASAPTextReview.png)
4. Review the text message and modify if needed. The default text message is the Text Manual message. See [ASAP Message Setup](asapmsgs.html).
5. Click Send. A commlog is added to each patient's account.
