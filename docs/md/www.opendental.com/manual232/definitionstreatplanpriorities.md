Definitions: Treat' Plan Priorities

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Treat' Plan Priorities.

![](images/definitionsTreatPlanPriority.png)

The Treat' Plan Priorities definition determines the priority options available to assign [Procedure](procedureedit.html) codes.

Assign a priority to a procedure in the procedure info window, [Treatment Plan Module](treatmentplan.html), or [Chart Module](chart.html) progress notes. Set the order of priorities in this definition. The priority affects the order of procedure codes in the [Edit Appointment](aptedit.html) window and progress notes, as well as the order and procedure text color on treatment plans.

Double-click an existing item or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionsTreatPlanPriorityEdit.png)

Enter the priority **Name** and select the procedure text [Color](colorselection.html) associated to the priority. Click OK to save.
