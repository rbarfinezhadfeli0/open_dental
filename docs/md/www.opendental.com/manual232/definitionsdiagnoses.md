Definitions: Diagnosis Types

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Diagnosis Types.

![](images/definitionsDiagnosisTypes.png)

The Diagnosis Type definition determines the list of diagnosis options available when [Entering Treatment](entertreatment.html).

Double-click an existing item or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionEditDiagnosisTypes.png)

* **Name**: Enter the diagnosis.
* **1 or 2 letter abbreviation**: Enter a short abbreviation for the diagnosis. The abbreviation will show in the [Chart Module](chart.html) progress notes.

Click OK to save.
