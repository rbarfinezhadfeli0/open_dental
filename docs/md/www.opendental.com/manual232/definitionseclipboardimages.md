Definitions: eClipboard Images

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), eClipboard Images.

![](images/definitionseClipboardImages.png)

The eClipboard Images definition determines the name and description when Image Captures are allowed in [eClipboard Setup](eclipboardsetup.html).

Double-click an existing item or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionseClipboardEdit.png)

* **Name**: Name the item. This will display in the eClipboard Setup window.
* **Patient Instructions**: Enter the instructions visible to the patient on the eClipboard device.

Click **Hide**, or check the **Hidden** checkbox to remove the item from the eClipboard Setup window.
