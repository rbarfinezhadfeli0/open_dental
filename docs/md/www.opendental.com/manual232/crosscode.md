Cross Code

Medical codes, such as [CPT Codes](cptcodes.html), can be cross coded to dental codes.

## Setup

First, add a procedure code for each medical code.

![](images/crosscodeEditMedCode.png)

Then, on the corresponding dental code, add the medical code in the *Medical Code* box.

![](images/crosscodeProcCode.png)

## Creating Claims

When a [Medical Claim](medicalins.html) is created for the procedure, the [Medical](medical.html) code is used on the claim instead of the dental code.
