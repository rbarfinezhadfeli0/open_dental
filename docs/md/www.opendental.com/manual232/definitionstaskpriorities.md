Definitions: Task Priorities

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Task Priorities.

![](images/definitionsTaskPriorities.png)

The Task Priorities definition determines the priority status options available to assign to [Tasks](tasks.html).

The priority determines the task color and sort order in [Task Lists](tasklists.html).

Double-click an existing item or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionEditTaskPriorities.png)

* **Name**: Enter the priority status.
* **D = Default, R = Reminder**: Set the priority default.
  + Enter **D** to designate the default priority for all newly created tasks (only one can be flagged as default).
  + Enter **R** to designate the default [Task Reminder](taskreminders.html) priority.
* **Color**: Click the colored block to select the task background [Color](colorselection.html) for this priority.

Click OK to save.
