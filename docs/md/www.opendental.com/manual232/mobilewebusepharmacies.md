Mobile Web Pharmacies

Note: Mobile Web has been replaced by [ODMobile](opendentalmobile.html), an app that is installed right to your mobile device! New features will not be added to Mobile Web, however it can still be used.

In the [Mobile Web Home](mobilewebuse.html) screen, press Pharmacies.

![](images/mobilewebPharm.gif)

The Pharmacies view is a list of all pharmacies and contact information as entered in the [Pharmacies](pharmacies.html).

If [Clinics](clinics.html) are enabled, pharmacies for the currently selected clinic will list. To view pharmacies from a different clinic press the dropdown arrow in the top right.

Select a pharmacy to view pharmacy address, phone, fax, and notes. Press a phone number to call it.

![](images/mobilewebPharmDetail.gif)
