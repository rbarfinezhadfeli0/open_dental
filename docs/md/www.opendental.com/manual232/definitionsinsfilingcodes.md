Definitions: Insurance Filing Code Group

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Insurance Filing Code Group.

![](images/definitionsInsFilingCodeGroup.png)

The Insurance Filing Code Group definition determines the categories available to organize [Insurance Filing Codes](insfilingcodes.html). This list is empty by default. Add new types as needed.

Double-click an existing item or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionsEditInsFilingCode.png)

Enter the group **Name** then click OK to save.
