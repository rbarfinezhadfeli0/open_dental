ODTouch: Exam Sheets

Fill out mobile enabled Exam Sheets from ODTouch.

From the [ODTouch: Home Screen](odtouchhome.html), tap **Exam Sheet**.

![](images/odtouchExam.png)

Custom [Exam Sheets](examsheets.html) created in Open Dental can be filled out on the tablet device.

Tap an existing exam to view a PDF version on the right. Use the **Start / End** Dates to filter exams by date created.

Tap **+** to create a new Exam Sheet, or select an exam and tap the Edit icon to make changes.

![](images/odtouchExamEdit.png)

Make changes, then tap **Save** to save the Exam Sheet to the patient.

Tap **Delete** to delete the Exam Sheet from the patient.
