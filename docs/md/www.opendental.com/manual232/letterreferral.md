Referral Letter

In the [Main Toolbar](toolbar.html), click the [Letter](letterspatient.html) dropdown.

![](images/letterReferral.gif)

To create a referral letter, you must first:

+ Set up at least one custom referral letter sheet. See [Referral Letter Layout](sheetsreferrals.html).
+ Have a referral attached to the patient. See [Referrals](referrals.html).

All attached referrals will list under *Referrals:* Select the referral name, then pick the sheet from the list.

![](images/letterReferralPick.gif)

Double-click a letter to open and fill out.

* The default content of the letter is defined in the sheet setup.
* Text on the letter is editable. Click in the letter and begin to type to make changes.

![](images/letterReferralExample.png)

Use the options at the bottom of the [Fill Sheet](sheetsfillout.html) window to send the letter. Sent letters can be viewed in the patient's Chart Module and Account Module.
