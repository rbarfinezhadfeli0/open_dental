Dental Student Reports

In [Evaluations](dsevaluations.html), click Reports.

![](images/dsReports.gif)

The [Dental Schools](dentalschools.html) Report generates a list of student evaluation grades by dental course.

To generate the report:

1. Select one or more [Dental Courses](dscourses.html). A list of instructors display.
2. Select one or more [Instructors](dsinstructors.html). A list of students display.
3. Select one or more [Students](dsstudents.html) to report on.

Click OK to generate the report.

![](images/dsReportPrint.gif)

* **Arrows**: Click the left/right arrow to jump pages in a multi-page report.
* **Query View**: Open the report in the [Query](queryoverview.html) window.
* **Export**: Save the report as a TXT or XLS file.
* **Print**: Print the report to the default printer.
* **Close**: Close the report window.
