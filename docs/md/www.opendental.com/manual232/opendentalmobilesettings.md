ODMobile Settings

Determine settings specific to ODMobile.

In the [ODMobile Menu](opendentalmobilemenu.html), tap **Settings**.

![](images/mobileSettings.png)

Settings are saved per Open Dental User.

## Security

![](images/mobileSettingsSecurity.png)

* **Use Fingerprint**/ **Use Face ID**: Options vary depending on device. Enable to use the fingerprint or face ID associated with the device to unlock the app.
* **Use Pin Code**: Enable to use a custom pin code to unlock the app.
* **Lock Frequency**: Determine when the app locks after a period of inactivity.

## SMS

![](images/mobileSettingsSMS.png)

* **Filter Automated Messages**: Enable to filter [Automated Messages](../site/ereminders.html) and display message type only when viewing [ODMobile Messages](opendentalmobilemessages.html) Conversations.
* **Vibrate on New Messages**: Enable to notify user of new text messages by vibrating.
