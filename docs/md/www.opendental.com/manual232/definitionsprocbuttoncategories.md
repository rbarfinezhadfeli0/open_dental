Definitions: Proc Button Categories

In the [Main Menu](mainmenu.html), click Setup, [Definitions](definitions.html), Proc Button Categories.

![](images/definitionsProcedureButtonCats.png)

The Proc Button Category definition determines the category options available to group procedure code buttons in the [Chart Module](chart.html).

Also edit categories or assign procedures to a category in the [Procedure Button Setup](procedurebuttons.html).

Double-click an existing item or click **Add** to create a new item. The Edit Definition window will open.

![](images/definitionEditProcedureButtonCats.png)

Enter the category **Name** then click OK to save.
