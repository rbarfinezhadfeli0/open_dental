ODMobile Messages

Communicate with patients via text message using Integrated Texting.

In [ODMobile](opendentalmobile.html), tap the **Messages** tab.

![](images/mobileMessages.png)

The office must be signed up for [Integrated Texting](../site/textintegrated.html) to utilize 2-way texting with patients.

## Send a New Text

Tap the message icon to send a message. Search by patient name or phone number.

## View a Conversation

Tap a conversation to view messages.

![](images/mobileMessagesConversation.png)

Reply to a message by typing in the Message field, then tap **Send**.

Note: All messages (sent from both the Open Dental program and the ODMobile App) can be viewed from this screen.
