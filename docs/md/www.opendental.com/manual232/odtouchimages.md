ODTouch: Images

View files saved to the patient's Imaging Module.

From the [ODTouch: Home Screen](odtouchhome.html), tap **Images**.

![](images/odTouchImages.png)

Only Image Categories in the [Imaging Module](images.html) that contain files will display. Tap a file to view it. Pull down on the images list to refresh it.

Files in the following formats are supported: JPEG, PNG, GIF, PDF, BMP, and TIF.
