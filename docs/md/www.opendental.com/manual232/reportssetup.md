Reports Setup

To access report setup options, users must have the *Setup* permission.

In the [Main Menu](mainmenu.html), click Setup, Reports.

![](images/reportSetupMain.png)

Alternatively, from the [Standard Reports](reportsstandard.html) window, click **Setup** in the upper-left corner.

There are four setup areas. Click a tab to access the area:

* [Display Settings](reportdisplaysettings.html): Control the appearance of the Reports window, including which Standard Reports are listed, the sort order of reports, and report name.
* [Security Permissions](reportsecurity.html): Control access to individual reports by user group.
* [Report Server](reportserver.html): Set up a report server.
* [Misc Settings](reportmiscsettings.html): Set default report options.
