Dental School Setup

In the [Main Menu](mainmenu.html), click Setup, Dental Schools.

![](images/dentalSchoolSetup.png)

For [Dental Schools](dentalschools.html), assign the default student and instructor [User Groups](securityusergroups.html); this determines the user group assignment when adding [Dental School Students](dsstudents.html) and [Dental School Instructors](dsinstructors.html).

**Students**: Click [...] to select the user group that is assigned to all students.

**Instructors**: Click [...] to select the user group that is assigned to all instructors.

**Grading Scales**: Click to add or edit [Dental School Grading Scales](dsgradingscale.html).

**Evaluations**: Click to add or edit the [Dental School Evaluation Setup](dsevaluationsetup.html). To view student evaluations, see [Dental Student Evaluations](dsevaluations.html).

To add students and instructors to each user group, see [Dental School Students](dsstudents.html) and [Dental School Instructors](dsinstructors.html).
