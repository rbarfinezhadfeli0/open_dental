Static Text Field Names

See [Sheet Static Text Field](sheetsstatictext.html).

Static text fields are a type of [Sheet Field](sheetsaddelements.html) used for fixed text. This list is in progress. Each field may not have a description.

Select the appointment the letter refers to.
