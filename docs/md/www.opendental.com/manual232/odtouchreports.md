ODTouch: Reports

Run standard reports to quickly get an overview of office metrics.

From the [ODTouch: Home Screen](odtouchhome.html), tap **Reports**.

![](images/odtouchReports.png)

Reports available in ODTouch match the [Standard Reports](reportsstandard.html) found in Open Dental.

## Run a Report

1. Tap the report desired from the list on the left-hand side.
2. To determine report filters, tap the caret in the upper right, then tap the Settings icon. Edit report filters as needed.
3. Tap the Run icon in the lower left. Report results display.
4. Tap the Expand icon to display the report in full screen.
