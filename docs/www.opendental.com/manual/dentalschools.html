Dental Schools

Open Dental works well in dental schools or dental hygiene schools.

Setup

Dental Schools must be set up in the following order.

  1. Enable Dental Schools in Show Features.
  2. Create Dental School Classes.
  3. Create Dental School Courses.
  4. Create User Groups for students and instructors in Dental School Security. Assign permissions to each group.
  5. In Dental School Setup:
    • Assign the User Group defaults.
    • Set up Dental School Grading Scales.
    • Set up Dental School Evaluations.
  6. In Dental School Providers:
    • Add Dental School Instructors.
    • Add Dental School Students.
  7. Add, copy, and manage course Requirements Needed.

Dental Schools should have their IT team set up the server and workstations with Open Dental. Also see Dental School Servers.

Using Dental Schools

  • Create Dental Student Evaluations.
  • Fill out Student Evaluations.
  • Generate Dental Student Reports as needed.
