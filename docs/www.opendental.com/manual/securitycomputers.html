Computer Security

See Computer and Network Setup.

Security for computer systems is a multi-faceted issue that is essential for HIPAA compliance and data protection. Addressing these issues correctly from the start prevents issues later on.

When considering how computer systems are configured, practices need to address security in three areas:

Network Security: For the security of the network, practices need to address:

  • Firewall
  • MySQL Security
  • Share A to Z Folder

Computer Security: For the security of computers, practices need to address:

  • Windows Update
  • Network Users (user names/passwords for Windows)
  • Antivirus
  • Encryption of Data at Rest and in Transit
  • Backups

Security in Open Dental: Define who has permission and access to specific features.

  • Open Dental Users, User Groups, and Permissions.
  • User credentials for third-party programs (e.g., imaging software).
