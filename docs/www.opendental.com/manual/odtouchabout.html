ODTouch: About

View ODTouch device information.

From ODTouch: Settings, tap About.

Most of the details on this screen are informational only.

  • App Version: The currently installed ODTouch version.
  • eConnector Version: The eConnector version. This should match the Open Dental version.
  • RegKeyNum: An identifier for the practice.
  • ClinicNum: An identifier for the selected clinic.
  • Device ID: An identifier for the device.

Device Name: Tap to assign a name to the device. Useful for identifying devices in the Device Manager.

View Licenses: Tap to view licenses pertaining to the ODTouch app.
