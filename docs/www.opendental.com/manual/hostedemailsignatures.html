Hosted Email Signatures

An email signature must be created to use Secure Email.

In the Secure Email Setup window, click Email Signatures.

Plain Text Signature: Enter a plain text email signature for the selected clinic.

HTML Signature: Displays the HTML signature used for the selected clinic.

  • Edit HTML Signature: Click to view the HTML editer and add HTML to the signature.
