Definitions: Treat' Plan Priorities

In the Main Menu, click Setup, Definitions, Treat' Plan Priorities.

The Treat' Plan Priorities definition determines the priority options available to assign Procedures.

Assign a priority to a procedure in the procedure info window, Treatment Plan Module, or Chart Module - Enter Treatment tab. Set the order of priorities in this definition. The priority affects the order of procedure in the Edit Appointment window and Chart Module Progress Notes, as well as the order and procedure text color on Treatment Plans.

Double-click an existing item to edit or click Add to create a new item. The Edit Definition window will open.

Name: Enter a description for the priority.

Color: Click the box to select the Color used for procedure text in the Treatment Plan.

Click Save to keep changes.
