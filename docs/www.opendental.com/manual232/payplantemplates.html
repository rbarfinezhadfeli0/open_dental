Payment Plan Templates

Set up Payment Plan Templates to quickly determine payment plan terms.

In the Main Menu, click Setup, Account, Pay Plan Template.

Payment Plan Templates are used when creating Dynamic Payment Plans from Open Dental or from within ODTouch Payment Plans.

Filters:

  • Clinic: Filter templates by clinic.
  • Show Hidden: Display templates marked as Hidden.

Click Add to create a new template, or double-click an existing template to edit.

Template Name: Name the template.

Clinic: Select the clinic the template applies to.

Set the terms for Down Payment, APR, Payment Amount or Number of Payments, Charge Frequency, and Handle Treatment Planned. For details on these terms, see Dynamic Payment Plan.

Click Save to save the template. Check Hidden if the template is no longer in use.
