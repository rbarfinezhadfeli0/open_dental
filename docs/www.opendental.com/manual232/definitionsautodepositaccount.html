Definitions: Auto Deposit Account

In the Main Menu, click Setup, Definitions, Auto Deposit Account.

The Auto Deposit Account definition determines the account options available when creating an automatic Deposit Slip while finalizing insurance payments. This list is empty by default. Add accounts as needed.

Double-click an existing item or click Add to create a new item. The Edit Definition window will open.

  • Name: Enter the item name. This identifies the item in the Auto Deposit Account dropdown in the Edit Insurance Payment window.
  • Account Number: The deposit account number.

Click OK to save.

To remove an item from the Auto Deposit Account list:

  • If the item has not been attached to an insurance payment, click Delete.
  • If the item has been attached to an insurance payment, check Hidden.
