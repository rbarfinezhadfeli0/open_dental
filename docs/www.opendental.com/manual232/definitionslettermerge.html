Definitions: Letter Merge Cats

In the Main Menu, click Setup, Definitions, Letter Merge Cats.

The Letter Merge Cats definition determines the categories options available for the Letter Merge feature. This list is empty by default. Add categories as needed.

Double-click an existing item or click Add to create a new item. The Edit Definition window will open.

Enter the category Name and click OK to save.
