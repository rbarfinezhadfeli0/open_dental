eServices Signup Help

The eServices Signup Portal lets you quickly sign up for and change eService preferences.

There are five tabs in the Signup Portal.

Signup

The eServices Signup tab is used to enable or disable an eService feature.

  • Basic View: The default view for single locations.
  • Advanced View: The default view when using clinics.

More Settings

Use the More Settings tab to select Patient Portal Settings.

About

The eServices Sign Up About tab displays information regarding eServices and the registration key. It is primarily used by Open Dental support when troubleshooting.

Contact Us

Click the Contact Us tab to open Contact Information for Open Dental Support.

Help

Click the Help tab to view this page.

More Information

For more information about an eService, click the corresponding link:

  • eServices Bundle
  • Automated Messaging (eReminders, eConfirmations, Automated Thank-You messages, Arrivals)
  • Integrated Texting Feature
  • Mobile Web Feature / ODMobile
  • eClipboard Feature
  • Patient Portal Feature
  • Web Forms Feature
  • Web Sched New Patient
  • Web Sched Recall
  • Web Sched Recall
  • Web Sched ASAP

Note:
  • Fees always reflect charges for all enabled eServices, per clinic/location.
  • Patient Portal and Web Forms are free for those on support. Thus in the Signup Portal these options are always checked.
