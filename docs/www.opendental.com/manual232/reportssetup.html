Reports Setup

To access report setup options, users must have the Setup permission.

In the Main Menu, click Setup, Reports.

Alternatively, from the Standard Reports window, click Setup in the upper-left corner.

There are four setup areas. Click a tab to access the area:

  • Display Settings: Control the appearance of the Reports window, including which Standard Reports are listed, the sort order of reports, and report name.
  • Security Permissions: Control access to individual reports by user group.
  • Report Server: Set up a report server.
  • Misc Settings: Set default report options.
