Definitions: Treat' Plan Priorities

In the Main Menu, click Setup, Definitions, Treat' Plan Priorities.

The Treat' Plan Priorities definition determines the priority options available to assign Procedure codes.

Assign a priority to a procedure in the procedure info window, Treatment Plan Module, or Chart Module progress notes. Set the order of priorities in this definition. The priority affects the order of procedure codes in the Edit Appointment window and progress notes, as well as the order and procedure text color on treatment plans.

Double-click an existing item or click Add to create a new item. The Edit Definition window will open.

Enter the priority Name and select the procedure text Color associated to the priority. Click OK to save.
