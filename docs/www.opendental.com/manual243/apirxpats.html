RxPats GET (single)

Version Added: 24.2.35

Gets a single rxpat.

RxNum: Required in URL.

Example Request:
GET /rxpats/13

Example Response:
{
"RxNum": 13,
"PatNum": 125,
"RxDate": "2024-09-26",
"Drug": "Amoxicillin 500 mg",
"Sig": "Take 4 caps 1 hour prior to dental appointment.",
"Disp": "4",
"Refills": "0",
"ProvNum": 13,
"Notes": "",
"PharmacyNum": 2,
"IsControlled": "false",
"DateTStamp": "2024-09-26 14:43:42",
"SendStatus": "Unsent",
"IsErxOld": "false",
"IsProcRequired": "false",
"ProcNum": 0,
"DaysOfSupply": 0.0,
"PatientInstruction": "Take all 4 caps 1 hour prior to dental appointment.",
"ClinicNum": 3
}

200 OK
404 NotFound (with explanation)
