Delete Appointment

On rare occassions, it may be appropriate to delete an appointment.

In the Appointments Module, select an appointment. In the toolbar, click Delete.

Alternatively:

  • In the Appointments Module, right-click on the appointment. Select Delete.
  • In the Chart Module Progress Notes, right-click the appointment. Select Delete.
  • Double-click an appointment. In the Edit Appointment window, in the bottom right, click Delete.

Appointments should not be deleted without considering how they will be tracked.

  • If the appointment was missed or was cancelled without 24-hour notice, then use Break Appointment.
  • After breaking the appointment, consider sending it to the Unscheduled List for followup.
  • If the patient really does not want followup, make sure the appointment is still in the Planned Appointments list so that it will show in the Planned Appointment Tracker.
  • Recall appointments can be deleted as they will continue to show on the Recall List for followup.
  • Completed appointments cannot be deleted if procedures are attached and Prevent changes to completed appointment with completed procedures is enabled in Preferences.
