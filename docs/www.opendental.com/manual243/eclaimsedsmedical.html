Electronic Dental Services - Medical

EDS Medical is an e-claims clearinghouse that offers medical claims.

In Clearinghouses, double-click EdsMedical.

Users must also be using EDS Dental in order to use this clearinghouse.

Setup

Enter the following details when enabling EDS Medical:

  • Login ID: Supplied by EDS Medical.
  • Password: The EDS Medical password.
  • Comm Bridge: Set to EdsMedical if not already done.
