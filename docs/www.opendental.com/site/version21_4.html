Version 21.4

See Versions.

Version 21.4 was released on 03/03/2022.

Overview of Version 21.4.

Major Changes

New options when printing appointments

  • Print in full color, partial color, or black and white. Print Appointments

Enter insurance information when scheduling appointments via Web Sched

  • Available for Web Sched New Patient and Web Sched Existing Patient. Web Sched Existing Patient: What Patient Sees, Web Sched New Patient: What Patient Sees
  • Select which carriers to allow or not allow. Web Sched Carrier Rule
  • Patients enter insurance information when scheduling. Web Sched Existing Patient: What Patient Sees, Web Sched New Patient: What Patient Sees

Create Continuity of Care Documents from the Chart Module Continuity of Care Document (CCD)

Changes by Category

Account Module

  • Delete payment plan charges from the Dynamic Payment Plan window. Payment Plan
  • Preference to run a limited income transfer on overpaid procedures attached to a received claim. Allocations Setup
  • Users must select a status when using Custom Claim Tracking. Edit Claim - Status History Tab
  • Disallow negative adjustments on dynamic payment plans. Payment Plan

Appointments Module

  • Reorder appointment field defs. Appointment Field Defs
  • Warn users when deleting the last procedure from an appointment. Edit Appointment, Procedure, Chart Module

Chart Module

  • Preview an exam sheet. Exam Sheet
  • Commlog preference to not set type, mode, or sent/received status. Enterprise Setup, Commlog
  • New commlog modes for Phone and Text. Commlog

eServices

  • Open Dental Mobile considers selected clinic when running report. ODMobile Reports
  • Display progress bar when downloading web forms. Web Forms Retrieve
  • Track which part of eClipboard a patient is in. eClipboard Setup, Kiosk Manager
  • Option not to save card information when making payment via eClipboard. eClipboard: Payment
  • Alert if Web Sched Recalls are unable to send. Alerts, Recall List
  • Update Payment Plan title when sending to eClipboard. eClipboard: Payment Plans
  • Open Dental Mobile visual changes in Appointments Module. (Also backported to Version 21.3.34) ODMobile Appointments
  • eService Activity Log tracks Patient Portal Payments. eService Activity Log
  • Ability to restrict Web Sched Recall appointments to specific blockouts. Web Sched Recall
  • Allow CCDs in the Patient Portal without requiring EHR settings. Patient Portal: What Patient Sees, Continuity of Care Document (CCD)
  • Display history of previous secure email messages. Secure Email: Workflow
  • Web Sched Recall preference to determine how many months into the future a patient can schedule their appointment. Web Sched Recall
  • eClipboard option to ignore specific sheets. eClipboard Setup
  • Secure Email alert includes patient name. (Also backported to Version 21.3.45) Alerts
  • Print secure email messages. Secure Email: Workflow

  • Ability for patients with a patient clone to schedule Web Sched Existing Patient appointments. Web Sched Existing Patient: What Patient Sees
  • Notes are optional on payments made via Patient Portal. Payment Portal
