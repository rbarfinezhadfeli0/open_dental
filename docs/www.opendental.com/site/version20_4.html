Version 20.4

See Versions

Version 20.4 was released on 12/17/2020.

Overview of Version 20.4

Major Changes

2021 CDT Codes: The 2021 CDT codes are available in version 20.3.48 or 20.4.28 or greater. Procedure Code Tools

  • See our video: How to Update Your CDT Codes

Zoom: Allows users to enlarge the overall display of Open Dental.

  • Zoom window allows users to view current settings and set zoom scale. Zoom
  • Added Zoom permission. Permissions
  • See our video: Using the Zoom Tool

QuickBooks Online integration: Connect QuickBooks Online to Open Dental.

  • Determine which Deposit account, Income accounts, and Classes you want to display in Open Dental. QuickBooks Online
  • Set QuickBooks Online as your default deposit software. Manage Module Preferences
  • Send deposit information to QuickBooks Online. Deposit Slip
  • See our Webinar: QuickBooks Online Integration

Insurance Verification Needed: Display an icon on upcoming appointments if patient's insurance has not been marked as verified.

  • Add Insurance Verification symbol to Appointment Views. Appointment View Edit
  • Add Verify Insurance note to the Appointment Bubble. Display Fields
  • See our video: Adding Insurance Verification Needed Indicator to Appointments

PDMP for additional states: PDMP is now available for all states.

  • For users in California, Illinois, Kentucky, Maryland, Utah, and Washington, see PDMP Bridge
  • For users outside of the above states, see Bamboo Bridge
