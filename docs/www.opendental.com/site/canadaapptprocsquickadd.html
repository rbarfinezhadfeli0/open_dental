Canada Definitions: Appt Procs Quick Add

For Canada users, in the Main Menu, click Setup, Definitions, Appt Procs Quick Add.

The Appt Proc Quick Add definition determines the list of procedures available in the Quick Add section of the Edit Appointment window.

This setup is specific to Canada users. Default buttons can be added or reset to default by running the Canada Procedure Code Tools for Appt Procs Quick Add.

See Definitions: Appt Procs Quick Add for additional details.
