Version 13.2

See Versions

Version 13.2 was released on 10/30/2013.

2014 Codes: The CDT codes for 2014 were released on 12/26/2013 as part of version 13.2.22. CDT Codes.

EHR: Enhancements for MU and progress toward 2014 certification. Smoking status uses SNOMED CT, race/ethnicity can be "declined to provide", medical problems now also store ICD9 and SNOMED CT.

Patient Portal: (This feature is for EHR customers only. It does not have much value for other users yet.) Now connects directly to office database instead of synching with the Open Dental corporate servers, making it more reliable. Patient Portal

Ontario Fee Schedules: The fee schedules for the Ontario Dental Association can be downloaded. Canada Fee Schedules

Repeating Charges: Enhanced to work better for ortho offices, with automatic claim creation. Repeating Charges

Minor Changes
TaskEdit permission adds security for editing and deleting tasks. Permissions
ProcFeeEdit permission tracks changes to procedure fees in the audit trail. Permissions
InsPlanChangeCarrierName permission tracks changes to insurance carriers in the audit trail. Permissions
Distributors can customize the Open Dental splash screen. Programming Resources
Claim Connect authentication error message is enhanced. Claim Connect
Statement billing note shows in the Account module and is printed on statements. Statement
Changing the appointment status on the Edit Appointment window causes the same actions as pushing the Break button. Break an Appointment
Apex clearinghouse. Apex
Credit card information from one patient account can be moved to a different patient account. Credit Cards
Distributors can offer electronic prescriptions to customers. NewCropPartnerName has been added to preference table. Distributors
NPI and State License numbers added to available output fields for Rx sheets. Rx Setup
Remove Nulls tool added to Database Maintenance. Database Maintenance
DentalStudio bridge. DentalStudio
