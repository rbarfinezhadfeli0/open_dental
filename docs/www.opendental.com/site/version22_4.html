Overview of Version 22.4.

Appointments Module

  • Associate Blockouts to Appointment Types. Appointment Types, Blockouts

  • Added Operatory Types. Definitions: Operatory Types, Operatory Edit

  • Group rows in Dynamic Payment Plan Amortization Schedule. Payment Plan
  • Payments prompt for Payment Type preference includes integrated processors. Credit Card Payment, CareCredit Payment

Chart Module

  • Ability to hide orthodontic hardware. Ortho

  • Web Sched users can track incoming clicks by unique link. Web Sched Advanced
  • Moved eConfirmation Exclusion Days to Automated Messaging Advanced Settings. eConfirmation and Web Sched Recall Exclusion Days
  • Updated wording of eServices Activity log actions. eService Activity Log

Family Module

  • Field for Hospital Discharge Date. Edit Patient Information: Hospitals Tab
  • Admit Date and Discharge Date added to Display Fields. Display Fields
  • Include Super Family Billing in Patient Billing Type Edit permission. Permissions
  • Provider Fee Overrides taken into consideration with Discount Plans. Fee Schedule Logic

Manage Module
