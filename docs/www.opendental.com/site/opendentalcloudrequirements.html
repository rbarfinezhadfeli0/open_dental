Open Dental Cloud Requirements

Open Dental Cloud requires certain hardware and setup.

Computer Requirements

These requirements pertain to each individual workstation accessing Open Dental Cloud.

  • ODCloudClient must be installed on a Windows machine to bridge to third-party softwares (such as imaging software, credit card processing, etc.).
  • Workstations should meet basic Open Dental Computer Requirements to run the ODCloudClient.
  • High Speed internet. 20 mbps download. 10 mbps upload.
  • Chrome is the recommended internet browser, however Open Dental Cloud may work with other browsers as well.
