Version 22.2 was released as stable on 09/08/2022.

Overview of Version 22.2.

Major Changes

Added Orthodontic Tooth Graphics

  • Webinar: Orthodontic Charting
  • Added Ortho tab to view and chart orthodontic treatment. Ortho
  • Add custom ortho prescriptions for easier charting. Ortho Prescriptions Setup
  • Add custom ortho hardware for ortho chart. Setup Ortho Hardware Specs

Enhancements to Patient Information

  • Option to note pronouns in Edit Patient Information window. Edit Patient Information, Display Fields
  • Gender options for Other. Edit Patient Information

Changes by Category

Appointments Module

  • Default a tooth number on specific Appt Proc Quick Add procedure. Definitions: Appt Procs Quick Add
  • Appointment Module Preference, Week View start day. Appointments Module
  • Quickly jump to specified date on Appointments calendar. (Also backported to V22.1.45) Appointments Module

Chart Module

  • Ortho chart opens in read-only mode for users without edit permission. Ortho Chart
  • Removed Ortho Chart Locking notice. Ortho Chart

eServices

  • Automatically install eConnector upon update if one is not detected. eConnector
  • Integrated texting users who are no longer on support can change their SMS texting limit. (Also backported to V22.1.31) eServices Signup Basic
  • Format Secure Email Access Code in HTML. Secure Email: Workflow
  • Removed mass email filter for Days last received. Advertising - Select Patients
  • Character limit added to Web Sched carrier rules. Web Sched Carrier Rule
  • ProvName tag for Automated Messaging includes hygienist. Edit Rule
  • Operatory headers use abbreviation instead of name. ODMobile Appointments

Imaging Module

  • Setting for thumbnails in Imaging Module is now separate from thumbnails in Chart Module. Definitions: Image Categories
  • Default image category for new images taken via video. Video Capture
  • Default measurement scale for new images. Preferences, Draw
  • Mount setting to default to Adj Mode after taking acquiring images. Mount Definitions
  • Mount setting to automatically flip images upon acquire. Mount Definitions
  • Mount setting to disable text background color. Mount Definitions

Security

  • New permission for Setup Wizard. Permissions
  • New permission for Show Features. Permissions
  • New permission for Printer Setup. Permissions
  • New permission for Provider Add. Permissions
  • New permission for Provider Edit. Permissions

  • Dental School permissions are only active when Dental Schools is enabled. Permissions
  • Added additional details for ProviderAdd and ProviderEdit Audit Trail permissions. Audit Trail Permissions

Sheets & Forms

  • Pre-fill patient forms using data from the database and previously filled forms. Patient Forms
  • Add Imaging Module images to patient forms. Fill Sheet
  • ComboBox option to select gender identity. Sheet ComboBox
  • Overhauled Edit Sheet Def window. Edit Sheet Def
