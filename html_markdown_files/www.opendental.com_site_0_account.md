<h2>Account Toolbar</h2>
<img src="images/accountToolbar.png" width="775" height="30"/><p><b>Payment</b>: Process <a href="0_patpayment.html">Patient Payments</a>.</p>
<p><b>Adjustment</b>: Add positive or negative adjustments, like discounts or billing charges, to the patient's account. Single or multiple adjustments can be created, and adjustments can be attached to procedures. </p>
<p><b>New Claim</b>: Create an insurance claim. If the patient has multiple insurance plans, claims are created for for up to 3 dental plans at once. Other claims, like medical, can be created from the dropdown.</p>
<p>See <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimtypes.html">Claim Types</a> for additional details on how claims are created. </p>
<p><b>Payment Plan</b>: Create patient or insurance repayment agreements. </p>
<p><b>Add Proc</b>: Quickly chart a procedure to the account by typing in a code or selecting from a customizable dropdown list. </p>
<p><b>Repeating Charge</b>: Add <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7repeatingcharges.html">Repeating Charges</a> to the patient's balance at a set interval. </p>
<p><b>Statement</b>: Generate a statement for the patient or entire family. </p>
<p class="MarginBottomZero">Third-Party Integrations: Other third-party integrations and <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7programlinks.html">Program Links</a> (i.e., bridges) may show a button in the toolbar, depending on setup. The following buttons show by default, but can be hidden from Program Links: </p>
<ul class="MarginBottomGap">
<li><b>TrojanCollect</b>: See <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7trojancollect.html">Trojan Express Collect</a>. </li>
<li><b>CareCredit</b>: Launch the <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carecreditaction.html">CareCredit Action</a> window. </li>
</ul>
