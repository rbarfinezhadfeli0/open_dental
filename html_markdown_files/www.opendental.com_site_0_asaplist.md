<p>The ASAP List shows patients who would like to be contacted when an earlier appointment becomes available. </p>
<p>In <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7appointmentlists.html">Appointment Lists</a>, click <b>ASAP</b>.</p>
<img src="images/asapList.png" width="915" height="416"/><p>The ASAP List can remain open while other areas of Open Dental are open. Use the list to identify patients and quickly notify them about available openings. Appointments can also be scheduled from here.</p>
<p><img src="images/YouTube_PlayButton_15px.png" width="16" height="16" style="margin-bottom: 0px"/> See our video: <a href="https://www.youtube.com/playlist?list=PLyjZ9PCVynClCgYyiViHUQRMECyrtKE6l">ASAP List</a>.</p>
<p>Easily open the <a href="0_apptedit.html">Edit Appointment</a> window to view additional details by double-clicking an appointment in the list.</p>
<p>Use right-click options to quickly select a patient or jump to their chart, send the appointment to the Pinboard to be scheduled (see <a href="0_apptsearch.html">Appointment Pinboard and Search</a>), or remove a patient from the list.</p>
<p>The list can be filtered by provider or code range, making it simple to find the right patients to fill an opening.</p>
<p>The <a href="textintegrated.html">Integrated Texting Feature</a> can be used to send text messages to a patient right from the list. Use the WebSched ASAP feature (see <a href="websched.html">Web Sched Feature</a>) to message patients and allow them to claim an opening online.</p>
<h2>Appointments Tab</h2>
<p>This is the default tab for the ASAP list and includes any appointments marked ASAP. Easily filter the list by appointment status and if the appointment is hygiene or not. </p>
