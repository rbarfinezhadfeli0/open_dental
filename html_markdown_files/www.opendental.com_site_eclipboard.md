<h2>Use in a Web Browser</h2>
<p>Let patients complete forms and other checklist items from the convenience and comfort of home. eClipboard Web is a web-based alternative to the eClipboard app. Easily provide patients with a link they can access using their personal computer or mobile device.</p>
<h2>Key Features</h2>
<p class="MarginBottomZero">In all versions of eClipboard (app and web-based), patients can: </p>
<ul class="MarginBottomGap">
<li>Fill out electronic intake forms via <a href="0_sheets.html">Sheets</a> (patient registration, medical history, consent forms, etc.)</li>
<li>Take photos that are uploaded to Open Dental (e.g., patient selfie, insurance card)</li>
<li> View statements and make payments using the <a href="paymentportal.html">Payment Portal</a></li>
</ul>
<p class="MarginBottomZero">In the eClipboard 24 and eClipboard 25 apps, patients can: </p>
<ul class="MarginBottomGap">
<li>Check into the Waiting Room when they arrive for their appointment</li>
<li>Review and sign <a href="0_treatmentplan.html">Treatment Plans</a></li>
<li>Review and sign <a href="0_paymentplan.html">Payment Plans</a></li>
<li>Review and sign Consent forms </li>
</ul>
<p class="MarginBottomZero">The following features are only available in eClipboard 25 and eClipboard Web: </p>
<ul class="MarginBottomGap">
<li>Updated interface</li>
<li>Dark mode</li>
<li>Accessibility options for font size and high contrast </li>
<li>Electronic patient forms via <a href="featureeforms.html">eForms</a> or <a href="0_sheets.html">Sheets</a><ul>
<li>When utilizing <a href="0_sheets.html">Sheets</a>, forms can also be filled out via Web Forms or printed</li>
<li>When utilizing <a href="0_eforms.html">eForms</a>, forms are exclusively filled out using the eClipboard 25 app or eClipboard Web</li>
</ul>
</li>
</ul>
<p><div class="Note">Note: <a href="0_eforms.html">eForms</a> are an alternative to Sheets that are optimized for usage on mobile devices. They are distinct from the Sheets feature, but both Sheets and eForms can be used in the eClipboard 25 app or eClipboard Web concurrently. eForms include conditional fields, which saves patient time by only including fields relevant to the patient. For more details, see <a href="featuredigitalforms.html">Digital Forms</a>.</div>
</p>
<p>Visit <a href="0_eclipboardsees.html">eClipboard: What Patient Sees</a> for a closer look at eClipboard 25.</p>

<h2>Getting Started</h2>
<p>eClipboard is included in our <a href="eservicesbundle.html">eServices Bundle</a> or is available for a standalone monthly fee. See <a href="fees.html">Fees for Support and Services</a> for pricing. Monthly pricing includes access to all versions of eClipoard, including eClipboard 25, eClipboard Web, and eClipboard 24</p>
<p>Practices must be on support and have a unique, active registration key to <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7eservicessignup.html">sign up</a> for eClipboard. Practices must remain on support while using the service. </p>
<p>Features available in the app correspond with the Open Dental version and app version your office is using. We recommend using the eClipboard 25 app and eClipboard Web. Routinely update the app and Open Dental for the latest features and enhancements. </p>
<p>See <a href="tablet.html">Tablet Devices</a> for details on app compatibility.</p>
<h2>Switching from eClipboard 24 to eClipboard 25</h2>
<p>Users who are already using eClipboard 24 (previously named <i>eClipboard</i>) may already notice the name change when installing the latest eClipboard update.</p>
<p class="MarginBottomZero">To utilize eClipboard 25: </p>
<ol class="MarginBottomGap">
<li>Update Open Dental to version 24.4 or greater.</li>
<li>On each tablet device, install eClipboard 25.  <ul>
<li>eClipboard 25 can be installed alongside eClipboard 24 for users who would like to test the app before fully switching.</li>
</ul>
</li>
<li>Log into the app on each device with the office's Signup Portal Login credentials. </li>
<li>In Open Dental, enable each device from the Device Manager. </li>
<li>Once comfortable with the eClipboard 25 app, uninstall eClipboard 24 from each device. </li>
<li>Delete each eClipboard 24 device from the Device Manager. </li>
</ol>
