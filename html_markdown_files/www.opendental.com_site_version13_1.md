<p>See <a href="versions.html">Versions</a></p>
<p>Version 13.1 was released on 6/25/2013.</p>
<p><b>Procedure Locking:</b> Will not be used by most offices. Includes ability to append notes to a locked procedure. A locked procedure cannot be edited or deleted, but it can be invalidated. All completed procedures in a date range may be locked at one time with a tool. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7procedurelocking.html">Procedure Locking</a></p>
<p><b>Wiki:</b> A full-blown wiki is now built into Open Dental. This can be a place for any staff to put any shared content, including employee manual, processes, etc. We use this feature heavily at Open Dental HQ, so it will be constantly enhanced. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7wiki.html">Wiki</a> (documentation incomplete) </p>
<p><b>Email: </b> Email setup now handles multiple email addresses. A default email address can be set for each clinic.</p>
<p><b>Spell Check: </b>Some important text boxes in Open Dental are now spell checked. Incorrect words are underlined with the option to replace with a suggested word or add to the custom dictionary. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7spellcheck.html">Spell Check</a><br><br><b>Electronic Prescriptions:</b> There is no longer a need to manually double-enter <a href="electronicrx.html">Electronic Rx</a>.<br></p>
<p><b>Minor Changes</b><br>  HL7 interfaces now have additional settings for showing specific modules.<br>  Primary providers for patients can now be reassigned by most used provider using a tool in the providers window. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7providers.html">Providers</a><br>  On the Procedure Code window, the Description column now wraps so the entire description is visible. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7procedurecodes.html">Procedure Codes</a><br>  Sheets now print exactly the same as the on-screen display instead of with slightly different spacing. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7sheets.html">Sheets</a><br>  In the Images module, the 100 button sets the image zoom to 100%. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7images.html">Images</a><br></p>
