<p>In the Treatment Plan <a href="0_modules.html">Module</a>, staff can manage and prioritize treatment plans for a patient.</p>

<h2>Treatment Plan Toolbar</h2>
<img src="images/treatmentToolbar.png" width="656" height="28"/><p class="MarginBottomZero">From the top toolbar, the following options are available. </p>
<ul class="MarginBottomGap">
<li><b>Preauthorization</b>: Create and manage treatment preauthoriztions from the Treatment Plan Module.</li>
<li><b>Discount</b>: Add a Procedure Discount to selected treatment planned procedures.</li>
<li><b>Update Fees</b>: For this patient only, update all fees and insurance estimates for all active and inactive treatment.</li>
<li><b>LabCase</b>: Create a new <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7labcaseedit.html">Lab Case</a>.</li>
<li><b>Consent</b>: Genereate and print a <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7consentforms.html">Consent Form</a>.</li>
<li><b>Print TP</b>: Print the selected treatment plan. The Treatment Plan Layout can be customized. </li>
<li><b>Email TP</b>: Email the selected treatment plan.</li>
<li><b>Sign TP</b>: Electronically sign a saved treatment plan.</li>
<li><b>CareCredit</b>: Launch the <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carecreditaction.html">CareCredit Action</a> window. </li>
</ul>
<h2>Treatment Plans</h2>
<img src="images/treatmentSaved.png" width="434" height="159"/><p>The Treatment Plans grid in the upper-left corner lists all active, inactive, and saved treatment plans for this patient. </p>
<h2>Buttons</h2>
<img src="images/treatmentButton.png" width="96" height="152"/><p>Save treatment plans, create new plans, send a plan to the <a href="eclipboard.html">eClipboard</a> app for a patient to sign, or create <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7apptplanned.html">Planned Appointments</a> using the options next to the Treatment Plans grid.</p>
<h2>Show Tab</h2>
<p>Customize what information is displayed in the treatment plan from the Show tab. Use the Sort By tab to determine if treatment is sorted by tooth or the order entered. </p>
<img src="images/treatmentShow.png" width="170" height="155"/><h2>Set Priority &amp; Estimates as of</h2>
<p>The <i>Set Priority</i> list is used to assign priorities to the procedures on the Treatment Plan. This affects how treatment is sorted in the Procedures grid. A subtotal is calculated for each priority. Use priorities to ensure treatment estimates are calculated in the correct order. </p>
<p>The <i>Estimates as of</i> dropdown allows you to select a future date to see what Treatment Plan estimates will be. This is helpful if treatment is expected to be completed in a new benefit year or once a waiting period has ceased. </p>
<img src="images/treatmentPriority.png" width="190" height="219"/><h2>Insurance Grids</h2>
<p>When the patient has an insurance plan, this area displays remaining family and individual insurance benefits at a glance. </p>
<img src="images/treatmentIns.png" width="200" height="248"/><h2>Discount Plan Grid</h2>
<p>When the patient is assigned a <a href="0_discountplan.html">Discount Plan</a>, this area displays Discount Plan benefits at a glance.</p>
