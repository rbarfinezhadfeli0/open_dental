<p>See <a href="versions.html">Versions</a></p>
<p>Version 6.7 was released on 10/14/09. It includes the following new features:</p>
<p><b>Account module:</b> Now shows financial breakdown by procedure as an option to be printed on statements. Also optionally shows notes for payments and adjustments. <a href="https://www.opendental.com/manual174/modulesetupmanage.html">Manage Module Preferences</a>.</p>
<p><b>Insurance Benefits: </b>Added categories for x-rays and for adjunctive services. Added tool to set all insurance categories to defaults. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7inscatssetup.html">InsCatSetup</a>. Overhauled ClaimProc edit window. Deductible is now always calculated before percentage. Deductible and annual max are now included in all the estimates. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimprocedure.html">Claim Proc</a>. The following are now included in calculations: fluoride age limit, simultaneous individual and family deductibles and maximums, ortho lifetime max, single procedure codes, and exclusions. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7benefitinfo.html">Benefit Info</a>.</p>
<p><b>eClinicalWorks:</b> Checks to make sure admin password is blank before enabling bridge. Added option to show Images module. Commlog button now shows in Chart module. Chart module layout is changed to allow eCW right panel to be on the right. Allows both tight integration and standalone modes. Rx button launches eCW medication interface. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgeeclinicalworks.html">eClinicalWorks</a>.</p>
<p><b>Electronic Benefits and Eligibility:</b> Use Claim Connect as your default clearinghouse. Sign up for their real-time services. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7ebenefits.html">Electronic Benefits</a>.</p>
<p><b>Minor Changes:</b><br>  Referral slip sheet added field for referral.phone2.<br>  Sheet static text fields added: clinicDescription, clinicAddress, clinicPhone, dateToday, nameFLFormal, nameLF, dateTimeLastAppt, nameF, nameL, and age. Added checkbox to referral letter sheet and consent sheet. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7sheets.html">Sheets</a>.<br>  The update service now allows use of web proxy. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7update.html">Update</a>.<br>  Choose database window now hides the password. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7choosedatabase.html">Choose Database</a>.<br>  Deductibles are now always subtracted before percentage. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimprocedure.html">Claim Proc</a>.<br>  Renew month added to insurance plans. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7benefitinfo.html">Benefit Info</a>.<br>  Insurance plan benefit adjustment can be deleted. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7insplan.html">Ins Plan</a>.<br></p>
