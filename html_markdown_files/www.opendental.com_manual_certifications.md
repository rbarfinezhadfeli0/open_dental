<div class="TopBar2"><p>Certifications</p></div>
<div class="GeneralPageContent">
<p> Certifications are useful for tracking the progress of employee training. </p>
<p>In the <a href="mainmenu.html">Main Menu</a>, click Tools, Certifications.</p>
<img src="images/certifications.png" width="915" height="575"/><p><b>Setup</b>: Click to access <a href="certificationsetup.html">Certification Setup</a>. </p>
<p><b>Show Only Incomplete</b>: Check to only view certifications for the selected employee with no date completed. </p>
<p><b>Sort by Date Completed</b>: Check to sort certifications by date completed in descending order. Certifications with no date are listed below certifications with a date. </p>
<p><b>Employee Search</b>: Search for an employee by first name to filter the Employee list. Leave blank to see all employees, not marked hidden, added to the <a href="employees.html">Employees List</a>. </p>
<p><b>Report To</b>: Filter employees by who they report to (e.g., lead, supervisor, etc.)</p>
<p><b>Employee</b>: Lists all <a href="employees.html">Employees</a> matching the Employee Search and Reports To filters. Highlight an employee to view certification progress. </p>
<p><b>Categories</b>: Select one or more categories to filter certifications by. </p>
<p class="MarginBottomZero">View By: </p>
<ul>
<li><b>Categories</b>: View certifications by category. Displays the certification list as shown above. </li>
<li><b>Cert Complete</b>: View completed certifications by category and certification. Displays as shown below.</li>
<br/></ul>
<p class="MarginBottomGap"><img src="images/certificationsCertComplete.png" width="915" height="324"/></p>
<p><b>Print</b>: Print the certifications list. Must be printed in landscape to view all columns.</p>
<h2>Employee Certification</h2>
<p></p>
<p class="MarginBottomZero">To mark a Certification as complete for an employee: </p>
<ol class="MarginBottomGap">
<li>Ensure <i>View by Category</i> is selected. </li>
<li>Highlight the employee who completed the certification. </li>
<li>Double-click the certification from the grid on the right. The following window will open:<br/><img src="images/certificationsEmployee.png" width="379" height="226" class="ImageInParagraph"/><ul>
<li>Enter the Date Completed or click <b>Today</b> to enter today's date.</li>
<li>Add any relevant notes about the certification.</li>
<li>Click <b>Save</b> to complete. </li>
</ul>
</li>
</ol>
<p>To delete a certification for an employee, double-click the certification from the grid, then click <b>Delete</b>. </p>
<p>If a certification was renewed (i.e., already had a date), double-click the certification and change the date.</p>
