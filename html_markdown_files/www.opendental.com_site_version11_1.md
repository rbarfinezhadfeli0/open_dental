			<h1>Version 11.1</h1>
		</div>
		<div id="Content">
<p>See <a href="versions.html">Versions</a></p>
<p>Version 11.1 was released on 12/12/2011</p>
<p>Prepare ahead of time: Version 11.1 requires <a href="dotnet.html">dotNet</a> 4.0, which should already be installed on newer computers and computers that get regular updates.</p>
<p><b>Scanning:</b> Automatic document feeder support added. Scans multiple pages to a single pdf. Scans duplex (double sided). <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7images.html">Scanning and Importing</a>.</p>
<p><b>Sheets Setup:</b> Toolbox and hotkeys added to allow click and drag to select, align left or top, copy/paste, rubber stamp, and tab order. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7sheets.html">Sheets</a>.</p>
<p><b>Dashboard:</b> A single window showing many important practice parameters in graphical format. Dashboard.</p>
<p><b>Batch Insurance Payment:</b> EOBs and the insurance checks that come with them can now be entered in a single batch window. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimpaymentbatch.html">Batch Insurance Payment</a>.</p>
<p><b>E-claims:</b> Medical and Institutional e-claims. New 5010 format available for all 3 claim types. Many fields added to support the new formats. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7medicalins.html">Medical Eclaims</a>.</p>
<p><b>Referred Procedure Tracking:</b> Individual procedures can now be referred out and their status tracked in a tool. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7referrals.html">Referred Proc Tracking</a>.</p>
<p><b>Superfamilies:</b> Groups of families for unique situations such as nursing homes. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7superfamily.html">Superfamily</a>.</p>
<p><b>Appointment Printing:</b> The daily printout now prints sharper and has size customization options. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7appointments.html"> Appointments</a>.</p>
<p><b>Minor Changes</b><br>  Payer ID table can be edited to control which electronic IDs are treated as Medicaid.<br>  Unscheduled list, multiple appointments can be deleted at once. <br>  Medications list can be printed. <br>  All patients for one provider can be moved to another provider. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7providers.html">Providers</a>.<br>  Sheets, many more fields added, especially for routing slips.<br>  Image export. <br>  Procedurecode list, the Show Hidden button now has a default setting and is initially set to not hide anything. <br>  Sending claims, clinic must be same for batch.<br> Chart module, when show by teeth, group notes for associated teeth now show.<br> Security permissions added for Providers, EcwAppointmentRevise, ReferralAdd, ProcedureNote, and perio red numbers. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7security.html">Security</a>.<br> Apixia bridge. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgeapixia.html">Apixia</a><br> Appointment search behavior can now also look for open operatories and has better provider options. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7appointments.html">Appointments</a>.<br> Chart module optimized to run twice as fast when entering new procedures. Only noticeable for users with slow connections.<br> Group notes can now aggregate individual procedure notes. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7procgroupnote.html">Procedure Group Note</a>.<br></p>
