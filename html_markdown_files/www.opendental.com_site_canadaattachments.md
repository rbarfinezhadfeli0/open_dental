<p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li><b> Send Attachments: </b> Click to send selected Claim Attachments. If nothing is selected, any unsent claim attachments are sent.</li>
<li><b> Add: </b> Click to open the Select Image for Claim Attachment window. See the following section for details. </li>
<li><b> Perio: </b> Click to attach a copy of the most recent perio exam.</li>
<li><b> Export: </b> Export all attached files to the path listed in the <i>Claim attachment export path</i> preference.</li>

<li>Attachment Name: The file name.</li>
<li>File Size: The size of the file. Up to 30 files can be added. Total attached files cannot be larger than 7 MB. <div class="Note">Note:  Some insurance carriers may impose different restrictions such as limited file type support or smaller file sizes. </div>

<h2>Select Image for Claim Attachment</h2>
<img src="images/canadaSelectAttachment.png" width="577" height="343"/><p>The Existing Images grid lists images and files already saved to the Imaging Module. Select a file to add as an attachment. </p>
<p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li><b>Import</b>: Import a new image or file from the File Explorer. </li>
<li><b>Snipping Tool</b>: Click to launch the Windows Snipping Tool. Use the tool to snip an image on the screen.</li>
<li><b>Paste Image</b>: Paste an image already saved to the clipboard.</li>
</ul>
<h2>Attachment Process</h2>
<p class="MarginBottomZero"></p>
<ol class="MarginBottomGap">
<li>Send the claim or predetermination as normal, without any attachments. </li>
<li>The response returns as an Acknowledgment. Attachments are generally not needed if the response returns an EOB or Rejection.</li>
<li>Navigate to the Canada Claim Attachments window.</li>
<li>Add images or files that are needed. </li>
<li>Click <b>Send Attachments</b>. </li>
</ol>
<p>Attachments can be sent immediately after receiving the Acknowledgment response or much later if the attachments were requested by mail. Some carriers send the final approval EOBs directly to Open Dental as a mailbox item, see <a href="canadaclaimsend.html">Canada Claim History and ROT</a>.</p>
<p>To confirm sent attachments, view the <i>ClaimSend</i> permission in the <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7audittrail.html">Audit Trail</a>.</p>
<p>Edits made using the <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7draw.html">Draw</a> tool are not reflected in the attachment when using the Import button.</p>
<p><div class="Note">Note: If attachments cannot be sent electronically, they can be mailed, or the carrier can be contacted for more submission options (e.g., the carrier's online portal, CDA SecureSend, BrightSquid, Fax, or Email).</div>
</p>
<h2>Setup</h2>
<p class="MarginBottomZero">To utilize attachments:  </p>
<ul class="MarginBottomGap">
<li>An image category must be marked as <i>Claim Attachments</i> in <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7definitionsimagecat.html">Definitions: Image Categories</a>.</li>
<li>The carrier must have a Supported Transaction Type of <i>Attachment</i>.</li>
</ul>
