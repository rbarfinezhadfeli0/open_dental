<p>See <a href="versions.html">Versions</a></p>
<p>Version 14.3 was released on 11/11/2014. </p>
<p><b>Electronic EOBs (ERAs): </b> Simplified and semi-automated entry of insurance payment amounts by claim. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimsera835.html">Electronic EOBs (ERA 835)</a></p>

<p><b>Appointments: </b>Customize the information that shows when you hover over an appointment in the schedule. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7displayfields.html">Display Fields</a><br>  Control appointment color by appointment type. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7appointmenttypes.html">Appointment Type Setup</a></p>

New database check to find procedures with multiple claim procedures for same insurance plan, and new tool to clear out old etrans message entries. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7databasemaintenance.html">Database Maintenance</a></p>
<p><b>Setup menu: </b>Menu items grouped by category. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7mainmenu.html">Main Menu</a></p>

<p><b>Sheets:</b> Create multi-page sheets. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7sheets.html">Sheet Setup</a></p>
<p><b>Insurance payment plans: </b> No longer requires deletion of charges. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7paymentplanpatient.html">Old Payment Plan</a></p>

<p><b>HL7:</b> New generic HL7 message definition that follows HL7 version 2.6 message structure standards. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7hl7genericmsg.html">Generic HL7 Message Structure</a></p>
