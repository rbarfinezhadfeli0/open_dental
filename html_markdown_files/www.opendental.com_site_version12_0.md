<p>See <a href="versions.html">Versions</a></p>
<p>Version 12.0 was released on 2/10/2012</p>
<p><b>Coordination of Benefits:</b> COB rules can be set per insurance plan as well as a global default. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7cob.html">COB</a>.</p>
<p><b>EOB scanning:</b> Insurance Payments have a place to attach scanned EOBs for later reference. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimpayfinalize.html">Finalize Insurance Payment</a>.</p>
<p><b>Command-line Arguments:</b> Many more command-line arguments added so that specific Open Dental databases, users, and patients may be launched from batch files, from shortcuts, and from other programs. <a href="commandline.html">Command-line</a>.</p>
<p><b>Central Enterprise Management Tool: </b>Customers with multiple locations and separate databases now have a way to magage and interact with those separate databases from a centralized tool. <a href="central.html">Central Manager</a>.</p>
<p><b>Popups for Family: </b>Popups can now be family level or even superfamily level. Multiple popups per patient and/or family. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7popups.html">Popups</a>.</p>
<b>Minor Changes</b><br>  Billing default messages for email subject and body. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7billing.html">Billing</a>.<br>  Email statement fields: nameF, nameFL, PatNum. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7billing.html">Billing</a>.<br>  CC recurring charges can be applied to payment plans. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7recurringcharges.html">Recurring Charges</a>.<br>  Consent forms can now be sent to the kiosk.<br>  Security permission for changing subscriber on existing plan. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7security.html">Security</a>.<br>  Chart module can be made to load more quickly by not showing as many object types in the Show tab. <br>  Chart module Progress Notes, order of rows has been overhauled. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7showtabchart.html">Show Tab Chart</a>.<br>  Appointment views, when show ops for sched provs, clinic filter added. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7appointmentviews.html">Appointment Views</a>.<br>  UB04 claimform now included. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7medicalins.html">Medical Ins</a>.<br>  FeeSchedules can no longer be hidden if in use by a Provider. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7feeschedules.html">Fee Schedules</a>.<br>  Claimform 1500 now part of database for everyone rather than requiring a download. <br>  Sheets with multiple large background images work better. <br>  Clinics: Filter recall list, confirmation list, unscheduled list, and billing list. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7appointmentlists.html">Appointment Lists</a>.<br>
