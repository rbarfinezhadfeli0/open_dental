<div class="TopBar2"><p>Definitions: Prog Note Colors</p></div>
<div class="GeneralPageContent">
<p>In the <a href="mainmenu.html">Main Menu</a>, click Setup, <a href="definitions.html">Definitions</a>, Prog Notes Colors.</p>
<img src="images/definitionsProgNotesColors.png" width="713" height="571"/><p>The Prog Notes Colors definition determines the text and background colors for various progress note entry types in the <a href="chart.html">Chart Module</a>.</p>
<p>If a progress note entry color is changed, consider also changing the corresponding <a href="definitionschartgraphiccolors.html">Chart Module Graphic Color</a> to match.</p>
<p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li><b>Status Treatment Planned</b>: Treatment Planned <a href="procedureedit.html">Procedures</a>.</li>
<li><b>Status Complete</b>: Completed Procedures.</li>
<li><b>Status Existing Current Prov</b>: Procedures marked Existing, Current Provider.</li>
<li><b>Status Existing Other Prov</b>: Procedures marked Existing, Other Provider.</li>
<li><b>Status Referred</b>: Procedures marked as Referred out to a specialist.</li>
<li><b>Rx</b>: <a href="rx.html">Prescriptions</a>.</li>
<li><b>CommLog</b>: <a href="commlog.html">CommLogs</a>.</li>
<li><b>LabCase</b>: <a href="labcaseedit.html">LabCases</a>.</li>
<li><b>Appointment Text - Today</b>: Text in <a href="appointments.html">Appointments</a> for the current date.</li>
<li><b>Appointment Background - Today</b>: Background in Appointments for the current date.</li>
<li><b>Past Appointment Text</b>: Text in prior appointments.</li>
<li><b>Past Appointment Background</b>: Background in prior appointments.</li>
<li><b>Future Appointment Text</b>: Text in scheduled appointments for a future date.</li>
<li><b>Future Appointment Background</b>: Background of scheduled appointments for a future date.</li>
<li><b>Broken/Unsched Appt Text</b>: Text in Broken Appointments, or those added to the <a href="unscheduled.html">Unscheduled List</a>.</li>
<li><b>Broken/Unsched Appt Background</b>: Background in Broken Appointments, or those added to the Unscheduled List.</li>
<li><b>Planned Appointment Text</b>: Text in <a href="apptplanned.html">Planned Appointments</a></li>
<li><b>Planned Appointment Background</b>: Background in Planned Appointments.</li>
<li><b>Patient Note Text</b>: Text in Patient attached <a href="notes.html">Appointment Module Notes</a>, as displayed in the Chart Module.</li>
<li><b>Patient Note Background</b>: Background in Patient attached Appointment Module Notes, as displayed in the Chart Module.</li>
<li><b>Completed Pt Note Text</b>:Text in Patient attached Appointment Module Notes marked complete, as displayed in the Chart Module.</li>
<li><b>Completed Pt Note Background</b>:Background in Patient attached Appointment Module Notes marked complete, as displayed in the Chart Module.</li>
<li><b>Status Condition</b>: Charted Conditions.</li>
</ul>
