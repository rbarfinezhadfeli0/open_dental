<p>See <a href="versions.html">Versions</a></p>
<p>Version 5.7 was released on 7/6/08.</p>
<p><b>New Family:</b> A scripted window for handling incoming phone calls and quickly entering a new family. All family members, Ins, etc entered from one single window saves a lot of time and mouse clicks. Especially useful in pedo offices. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7familyadd.html">Family Add</a>. </p>
<p>PPO writeoffs: Alternative production reports that show writeoffs as applied on the day of service rather than on the day of the insurance payment. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7insplantypes.html">Ins Plan Types</a>. </p>
<p>Claims now have more fields for attachment quantities and types. The info (not the attachments themselves) will go out on both e-claims and paper claims. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimedit.html">Claim Edit</a>. </p>
<p>Appointments have a new field called TimeLocked. This checkbox can turn on or off the automatic time calculations. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7aptedit.html">Appointment Edit</a>. </p>
<p>Lab due dates now show on appointments.</p>
<p>Tooth ranges now show in Chart, Account, TP, and appointment edit windows.</p>
<p>Appointment edit window now uses layman's term on each procedure. </p>
<p>TaskLists: Inboxes can be set for each user. Automatic refreshing now works for all lists. Reply button added. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7tasklists.html">Task List</a>. </p>
<p>Deposit Slips will now print the account number at the bottom in the same font as on a preprinted deposit slip. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7depositslip.html">Deposit Slip</a>. </p>
