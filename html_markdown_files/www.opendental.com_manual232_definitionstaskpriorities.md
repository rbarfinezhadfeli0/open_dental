<div class="GeneralPageContent">
<p>In the <a href="mainmenu.html">Main Menu</a>, click Setup, <a href="definitions.html">Definitions</a>, Task Priorities.</p>
<img src="images/definitionsTaskPriorities.png" width="816" height="532"/><p>The Task Priorities definition determines the priority status options available to assign to <a href="tasks.html">Tasks</a>.</p>
<p>The priority determines the task color and sort order in <a href="tasklists.html">Task Lists</a>.</p>
<p>Double-click an existing item or click <b>Add</b> to create a new item. The Edit Definition window will open.</p>
<img src="images/definitionEditTaskPriorities.png" width="433" height="238"/><p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li><b>Name</b>: Enter the priority status.</li>
<li><b>D = Default, R = Reminder</b>: Set the priority default. <ul>
<li>Enter <b>D</b> to designate the default priority for all newly created tasks (only one can be flagged as default).</li>
<li>Enter <b>R</b> to designate the default <a href="taskreminders.html">Task Reminder</a> priority.</li>
</ul>
</li>
<li><b>Color</b>: Click the colored block to select the task background <a href="colorselection.html">Color</a> for this priority.</li>
</ul>
<p>Click OK to save.</p>
