<div class="GeneralPageContent">
<p>Open Dental provides some fee guides for users in Canada to import.</p>
<p>For <a href="canada.html">Canada</a> users, in the <a href="feescheduletools.html">Fee Tools</a> window, click <b>Import Canada</b>.</p>
<img src="images/feeSchedPickRemote.gif" width="455" height="294"/><h2>Updating Canada Fee Guides</h2>
<p>When a new fee guide is released, we recommend making a copy of the original fee guide for reference, then importing the new fees into the original fee guide. By doing it this way, the fees are updated and fee guides do not need to be reassigned to providers.</p>
<p><b>Update Open Dental</b><br/> Some fee guides require an Open Dental <a href="updateinstall.html">Update</a> prior to importing the latest fee guide. Find the fee guide in the list below to see the required Open Dental version. </p>
<p class="MarginBottomZero"><b>Create a backup of the original fee guide:</b></p>
<ol class="MarginBottomGap">
<li>Create a new fee guide named after the original (e.g., UCR2023_copy). See <a href="feeschedules.html">Fee Schedules</a>.</li>
<li>Under Lists, <a href="procedurecodes.html">Procedure Codes</a>, click Fee Tools.</li>
<li>Under Select Fees, select the fee guide to copy (the original).</li>
<li>In Copy To, select the fee guide to copy the fees into (e.g., the new fee guide, UCR2023_copy).</li>
<li>Click Copy. The new fee guide will now be a copy of the original fee guide.</li>
</ol>
<p class="MarginBottomZero"><b>Import fees into the original fee guide:</b></p>
<ol class="MarginBottomGap">
<li>Clear the original fee guide: <ol>
<li>Click Fee Tools.</li>
<li>Under Select Fees, select the original fee guide.</li>
<li>Click Clear to remove the fees from the original fee guide.</li>
</ol>
</li>
<li>Import the new fees into the original fee guide:  <ol>
<li>Click Fee Tools.</li>
<li>Under Select Fees, select the original fee guide.</li>
<li>Click Import Canada.  <ul>
<li>Some guides require a Member ID and Password from the providing province.</li>
<li>If fees have not been cleared (the fees still exist in the original), a warning displays. Click OK to proceed, or Cancel to close without importing.</li>
</ul>
</li>
<li>The Fee Sched Pick Remote window displays all of the fee guides currently available for download. To import, double-click on a fee guide or select it then click OK. A message indicates when the import is finished.</li>
</ol>
</li>
</ol>
<p><div class="Note">Note: <ul>
<li>Custom codes created by the office may need fees re-entered after updating a fee guide.</li>
<li>To update the fees on treatment plans for all patients, see Update Fees in <a href="feescheduletools.html">Fee Tools</a>.</li>
</ul>
</div>
</p>
