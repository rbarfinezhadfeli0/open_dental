<div class="TopBar2"><p>Lab Slip Layout</p></div>
<div class="GeneralPageContent">
<p>Lab Slips can be customized for the office or specific laboratories.</p>
<p>In <a href="sheets.html">Sheets</a>, double-click on a custom sheet with the type of LabSlip.</p>
<img src="images/sheetLabSlip.png" width="915" height="705"/><p class="MarginBottomZero">Alternatively, in Sheets:  </p>
<ul class="MarginBottomGap">
<li>Click <b>New</b> and choose <i>LabSlip</i> as the Sheet Type to create a new lab slip from blank.</li>
<li>Select an existing Custom <i>LabSlip</i> sheet and click <b>Duplicate</b> to create a copy of an existing lab slip.</li>
<li>Select the existing Internal <i>LabSlip</i> sheet and click <b>Copy</b> to create a copy of the default lab slip that can be edited.</li>
</ul>
<p>Also see: <a href="sheetproperties.html">Sheet Def Properties</a> and <a href="sheetsaddelements.html">Sheet Field Types</a>.</p>
<p>Lab slips are utilized with <a href="labcaseedit.html">Lab Cases</a>. Lab slips are typically customized by scanning the slip provided by a specific dental lab and using it as a background Static Image. Other required elements can be placed on top, such as instructions, due date, and license number. Lab slips support <a href="electronicsignatures.html">Electronic Signatures</a>.</p>
<p>To set a default slip for a dental lab, see <a href="laboratories.html">Laboratories</a>.</p>
<p>To create a lab slip for a patient, see <a href="labslips.html">Lab Slip</a>.</p>
<p>To customize a LabSlip sheet type, see <a href="sheetsetup.html">Edit Sheet Def</a>.</p>
