<p>Version 6.3 was released on 1/23/09.</p>

Includes both patient letters and referral letters. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7letterspatient.html">Patient Letters</a>, <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7letterreferral.html">Referral Letters</a>. </p>
<p><b>Mobile Version:</b> A program has been built for the Windows Mobile Pro platform. Synchronizing functionality has been added to the main program. Includes viewing of appointments and patients. </p>
<p>Option to handle blank co-pay fee schedule entries as zeros. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7insplantypes.html">Insurance Plan Types</a>.<br>  Changes to statement for Australia. </p>
