<div class="GeneralPageContent">
<p>In the <a href="mainmenu.html">Main Menu</a>, click Setup, <a href="definitions.html">Definitions</a>, Claim Custom Tracking.</p>
<img src="images/definitionsClaimCustomTracking.png" width="816" height="526"/><p>The Claim Custom Tracking definition determines the list of claim tracking statuses available on the <a href="claimtabstatus.html">Edit Claim - Status History Tab</a>.</p>
<p>Use custom statuses to filter the <a href="reportoutins.html">Outstanding Insurance Claims Report</a> by status and to exclude claims from the report for a certain number of days after a status is changed. Set the number of days in this definition. </p>
<p>Double-click an existing item or click <b>Add</b> to create a new item. The Edit Definition window will open.</p>
<img src="images/definitionEditClaimCustomTrack.png" width="433" height="248"/><p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li><b>Name</b>: Enter the claim tracking status.</li>
<li><b>Days Suppressed</b>: Enter the number of days to exclude a claim from the outstanding insurance claims report for this status.</li>
</ul>
<p>Click OK to save.</p>
<p>To remove an item from custom tracking status lists, double-click the item and click <b>Delete</b>. If the item has been used it cannot be deleted. Alternatively, mark the item as <b>Hidden</b> if not in use.</p>
