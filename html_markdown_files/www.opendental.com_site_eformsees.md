<p>eForms are dynamic forms designed for use in eClipboard.</p>
<p>On a tablet device, open an eForm via the <a href="eclipboard.html">eClipboard</a> 25 app.</p>
<img src="images/eFormSees.png" width="482" height="723"/><p>Alternatively, open an eForm via <a href="eclipboardwebsees.html">eClipboard Web</a>.</p>
<p>eForms can be set up to be dynamic, so only fields relevant to the patient are displayed. </p>
<p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li>Use the <b>Accessibility</b> button in the upper right to select the font size, display mode, and contrast mode.</li>
<li>Use the <b>Previous</b> and <b>Next </b>buttons to navigate the form. </li>
</ul>
<h3>Conditional Questions</h3><p>eForms allow you to set conditional logic for questions, making it easy to show patients only the questions that are relevant to them.</p>
<img src="images/eformsConditionalQuestions.gif" width="600" height="500"/><p>For example, in a Registration Form, if a patients answers "No" to the question "Do you have dental insurance", any fields related to insurance are hidden. If they answer "Yes", then they'll be shown fields to enter their insurance information.</p>
