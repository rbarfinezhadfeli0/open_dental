		<div class="GeneralPageContent">
<p>See <a href="usingodconversion.html">Use Converted Database</a>.</p>
<p>Most final conversions do not convert outstanding claims. Before claim payments can be received in Open Dental, create each claim.</p>
<p><div class="Note">Note: <a href="conversions.html">Conversions</a> from EZ Dental and Dentrix convert outstanding primary claims, but not secondary claims.</div>
</p>
<p>First, print an outstanding claim report from the old software (as of the final conversion date). In order to start tracking outstanding claims in Open Dental, start recreating these outstanding claims in Open Dental as soon as possible. </p>
<p>As claims are recreated in Open Dental, mark the claim off the printed list. Once all outstanding claims from the old software have been recreated in Open Dental, the <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7reportoutins.html">Outstanding Insurance Claims Report</a> and <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7reportaging.html">Accounts Receivable (A/R)</a> report can be used and will be accurate.</p>
<p class="MarginBottomZero"><b>To recreate claims in Open Dental</b>:<br> First, verify the patient's insurance benefit information. See <a href="convinspending.html">After a Conversion: Verify Insurance</a>. This ensures insurance estimates are accurate and reduce data entry when entering claim payments.<br><br></p>
<ol class="MarginBottomGap">
<li>In the <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7account.html">Account Module</a>, select the procedures and click <b>New Claim</b>. This creates the claim to be modified.</li>
<li>Change the claim status from <i>Waiting to Send</i> to <i>Sent - Verified</i>.</li>
<li>Change the <i>Date Sent</i> to the actual sent date for the claim.</li>
<li>Click <b>OK</b> to save changes.</li>
</ol>
<p>Once claim payment has been received, the claim can be processed and payment finalized. See <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimreceive.html">Receive Claim</a> and <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimpayfinalize.html">Finalize Insurance Payment</a>.</p>
