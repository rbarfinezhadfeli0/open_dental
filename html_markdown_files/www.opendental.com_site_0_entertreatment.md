<p>Chart procedures for a patient using the Enter Treatment options in the <a href="0_chart.html">Chart Module</a>.</p>
<p>In the Chart Module, select the Enter Treatment tab.</p>
<img src="images/entertreatment.png" width="915" height="300"/><p>Easily plan treatment and add referred or existing treatment. Charted treatment is displayed on <a href="0_graphicaltoothchart.html">Graphical Tooth Chart</a>. Procedures can be grouped together for multiple visits (e.g., implant, bridge, etc.)</p>
<p><img src="images/YouTube_PlayButton_15px.png" width="16" height="16" style="margin-bottom: 0px"/> See our video: <a href="https://youtu.be/1muCgXGk2f0">Clinical Charting 1: Entering Treatment Webinar</a>.</p>
<p></p>
<p><b>Tooth Surface Buttons (B/F, V, M, O/I, D, L):</b> When charting treatment, select surfaces in addition to a tooth number to populate the treatment area.</p>
<p><b>Entry Status</b>: Easily choose which status (e.g., treatment planned, referred, etc.) to chart a procedure. Status colors are customizable.</p>
<p><b>Date</b>: Easily update the date assigned to procedures when charting. This is helpful if entering in previously completed procedures. </p>
<p><b>Diagnosis (optional)</b>: Assign a diagnosis (e.g., Caries) to a procedure. Options are customizable.</p>
<p><b>Prognosis (optional)</b>: Assign a prognosis (e.g., Good, Fair, etc.) to a procedure. Options are customizable.</p>
<p><b>Priority (optional)</b>: For treatment planned procedures, assign a priority to affect treatment order. Options are customizable.</p>
<p class="MarginBottomZero">There are multiple ways to chart procedurs: </p>
<ul class="MarginBottomGap">
<li><b>Procedure List</b>: Open the <a href="0_procedurecodes.html">Procedure Codes</a> list to search for and select a procedure code.</li>
<li><b>Procedure Buttons</b>: Select an option to quickly chart one or multiple procedures. These buttons are entirely customizable.</li>
<li><b>Type Proc Code</b>: Manually type a procedure code to chart. </li>
<li><b>Quick Buttons</b>: These buttons are associated with a single code and designed to quickly chart fillings and sealants. Options are customizable and buttons can be assigned to surfaces. </li>
</ul>
<p><b>Treatment Plans</b>: Show treatment plan infromation in place of the Progress Notes area of the <a href="0_chart.html">Chart Module</a>.</p>
<p><b>Group Note</b>: Create a Group Note for the selected patient. Multiple procedures can be attached to the Group Note.</p>
<h2>Additional Options to Chart Procedures</h2><p class="MarginBottomZero">In addition to charting from the Enter Treatment tab in the Chart, procedures can also be charted using the following methods: </p>
<ul class="MarginBottomGap">
<li><b>Quick Procs</b>: From the <a href="0_account.html">Account Module</a>, users can quickly chart a procedure code as complete. Codes can be typed manually or you can set up a customizable list to choose from.</li>
<li><b>Appt Proc Quick Adds</b>: Easily chart procedures when creating an <a href="0_apptedit.html">Appointment</a> by choosing from a customizable list of procedure combinations.</li>
<li><b>Add from Appointment</b>: When setting up an <a href="0_apptedit.html">Appointment</a>, easily treatment plan procedures that haven't been charted yet.</li>
<li><b>Auto Codes</b>: Auto Codes work behind the scenes to insert and validate the correct procedure code depending on a variety of conditions. These are customizable and can be helpful to ensure the correct procedure code is used for things like fillings, bridges, etc.</li>
</ul>
