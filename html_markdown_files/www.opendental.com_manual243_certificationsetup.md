<div class="GeneralPageContent">
<p>Use Certification Setup to create and edit certifications for employees.</p>
<p>In <a href="certifications.html">Certifications</a>, click <b>Setup</b>.</p>
<img src="images/certificationSetup.png" width="855" height="399"/><p class="MarginBottomZero"> There are three steps to set up Certifications. </p>
<ol class="MarginBottomGap">
<li>Create Certification Categories. See <a href="definitionscertificationcats.html">Definitions: Certification Categories</a>.</li>
<li>Add <a href="employees.html">Employees</a> and to use the Report To filter, assign a <i>Reports To</i> for each employee.</li>
<li>Create certifications. See Certification Edit below. </li>
</ol>
<p><b>Categories</b>: Highlight a category to filter the certifications showing. </p>
<p><b>Up/Down</b>: Highlight a certification to move it up or down the list. </p>
<p><b>Add</b>: Create a new certification. Opens Certification Edit. </p>
<h2>Certification Edit</h2>
<p>Click<b> Add</b> to create a new certification or double-click an existing certification from the grid to edit. </p>
<img src="images/certificationEdit.png" width="428" height="375"/><p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li><b>Description</b>: Name of the certification. </li>
<li><b>Wiki Page</b>: (optional) Enter the name of the corresponding <a href="wiki.html">Wiki</a> page for the certification. </li>
<li><b>Categories</b>: Select a category to assign to the certification.</li>
<li><b>Hidden</b>: Hide the selected certification. </li>
<li><b>Delete</b>: Delete the selected certification. Certifications cannot be deleted if they are in use by an employee.</li>
</ul>
