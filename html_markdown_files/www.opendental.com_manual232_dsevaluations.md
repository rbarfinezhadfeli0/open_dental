<div class="GeneralPageContent">
<p>Student evaluations can be added and tracked in Open Dental.</p>
<p>In the <a href="mainmenu.html">Main Menu</a>, click Tools, Evaluations.</p>
<img src="images/dsevaluations.png" width="910" height="451"/><p><a href="dentalschools.html">Dental Schools</a> student evaluations can only be created by <a href="dsinstructors.html">Dental School Instructors</a>. Users with the <i>Admin Eval Edit</i> permission can view or edit student evaluations, but cannot create them.</p>
<p>A list of all evaluations created that meet the filter criteria will list. To view or edit an evaluation, double-click on it.</p>
<p class="MarginBottomZero">Filter Options:  </p>
<ul class="MarginBottomGap">
<li><b>Date</b>: Determines which evaluations list. Enter a beginning and end date.</li>
<li><b>Course</b>: Filter the evaluations by Dental Course.</li>
<li><b>Student Filters</b>: Filter the evaluations by student's last name, first name or ProvNum.</li>
<li><b>Admin Filters</b>: Select the instructor who administered the evaluation.</li>
</ul>
<p><b>Add</b>: Click to add a new evaluation. See <a href="dsfillevaluation.html">Fill out Student Evaluations</a>.</p>
<p><b>Reports</b>: Click to view <a href="dsreports.html">Dental Student Reports</a>. </p>
