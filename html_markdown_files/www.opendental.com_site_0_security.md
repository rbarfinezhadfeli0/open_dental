<p>A variety of security settings are available in Open Dental program to help protect patient data and track user access. This is an important component of a security plan as well as a requirement for <a href="hipaa.html">HIPAA</a> compliance.</p>
<p>In the Main Menu, click Setup, Security.</p>
<img src="images/securityWindow.png" width="915" height="705"/><p>Users, including a username and password, can be made for all employees and providers. Only specified users have access to edit security settings. Actions taken by users are logged in the <a href="0_audittrail.html">Audit Trail</a> for easy tracking.</p>
<h2>User Groups and Permissions</h2><p>User groups are a way to customize the permissions assigned to specified users. Users can be assigned to multiple groups making it easy to ensure users have only necessary permissions. </p>
<img src="images/securityUserGroups.png" width="915" height="497"/><h2>Global Security Settings</h2><p>Set security settings, like time card editing, password requirements, and more, that affect all users.</p>
<img src="images/securityGlobalSecuritySettings.png" width="372" height="582"/><p>Also set global locks to prevent users from making changes after a certain date.</p>
<h2>Security Badges</h2><p>Badge IDs can be associated with users to allow them to log onto Open Dental by scanning or swiping a badge.</p>
