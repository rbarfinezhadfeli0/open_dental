<div class="GeneralPageContent">
<p>Create claim errors to use when tracking claim status history.</p>
<p>In the <a href="mainmenu.html">Main Menu</a>, click Setup, <a href="definitions.html">Definitions</a>, Claim Error Code.</p>
<img src="images/definitionsClaimErrorCode.png" width="713" height="386"/><p>The Claim Error Code definition determines the list of error code options available in the <a href="claimtabstatus.html">Edit Claim - Status History Tab</a>. This list is empty by default. Add Error Codes as needed.</p>
<p>Double-click an existing item or click <b>Add</b> to create a new item. The Edit Definition window opens.</p>
<img src="images/definitionEditClaimErrorCode.png" width="425" height="218"/><p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li><b>Name</b>: Enter the abbreviation for the error code. This displays in the Error Code column of the Claim Custom Tracking Status History grid.</li>
<li><b>Description</b>: Enter the full description of the error code.</li>
<li><b>Delete</b>: Remove an error code from the list. Cannot be deleted if the error has been used. </li>
</ul>
<p>Click <b>Save</b> to exit the window and keep changes.</p>
