<p>See <a href="versions.html">Versions</a></p>
<p>Version 11.0 was released on 9/30/2011.</p>
<p><b>EHR (electronic health record):</b> Open Dental is now a fully certified Ambulatory EHR, and qualified providers are eligible for incentive payments. <a href="http://opendental.com/manual/ehrlicense.xml">EHR</a>.</p>
<p><b>Canadian e-claims:</b> Open Dental is now certified with CDAnet. <a href="canada.html">Canada</a></p>
<p><b>Mobile version:</b> Logout button added on every window, appointment notes added in three different places, cells made slightly taller, Pharmacy list added, Allergies added. </p>
<p><b>Ortho Chart:</b> A grid with customizable columns where dated entries can be made at each visit. Can track things like Uwire, Lwire, Hyg, Notes, Next visit, Interval. etc <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7orthochart.html">Ortho Chart</a>.</p>
<p><b>Referrals:</b> Referrals can be added faster and with less confusion. IsDoctor field added to begin to separate professional referrals from other types. Referral window now has search box at the top and scrolling has been improved. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7referralsetup.html">Referral Setup</a>.</p>
<p><b>Automation:</b> Added action for Popups, trigger for Open Patient, and 6 conditions. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7automation.html">Automation</a>.</p>
<p><b>Minor Changes</b><br>  Major version numbers will match year from now on.<br>  Insurance tables normalized (patplan.PlanNum removed) for easier bug tracking. This will break some queries.<br>  If you open multiple instances of OD, and then go to Choose Database window, correct current database will show. <br>  eCW provider ID field added. Abbreviation can now be changed independently.<br>  Patient fields: Checkbox type added. <br>  Automatic logoff after a certain number of minutes. <br>  Rx State ID field added to provider.<br>  Lab cases window sortable by any column, has instructions column, and has print button.<br>  GroupName added as an available field for claimforms. <br>  Treatment Finder report, additional column for carrier, exclude patients with appointments. (enhanced on 9/22/11) <br></p>
