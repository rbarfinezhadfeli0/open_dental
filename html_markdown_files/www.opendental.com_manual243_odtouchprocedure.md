<p>Edit details for a procedure.</p>
<p>In <a href="odtouchchart.html">ODTouch: Chart</a>, tap an existing procedure to edit it.</p>

<p><b>Unlock / Lock</b>: When unlocked, the selected tooth or tooth range can be edited.</p>
<p><b>Edit Note</b>: Add or edit procedure notes.</p>
<p></p>
<p><b>Sign</b>: Sign the procedure note.</p>
<p>Tap <b>Delete</b> to delete the procedure or tap <b>OK</b> to save changes.</p>
<h2>Procedure Edit Dropdown</h2>
<p>Tap the caret in the upper right to view additional options. </p>
<img src="images/odtouchChartProcedureDropdown.png" width="159" height="89"/><p><b>Mixed Dent / Primary / Permanent</b>: Toggle the tooth number display between mixed dentition, all primary, and all permanent.</p>
<p><b>Supernumerary / Numerary</b>: Toggle the tooth number display between numerary teeth (1 - 32) and supernumerary teeth (51-82). </p>
