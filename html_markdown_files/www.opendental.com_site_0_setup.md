<p>From the Setup menu, you can customize settings, like permissions, preferences, and more.</p>
<p>In the Main Menu, click Setup.</p>
<img src="images/mainmenuSetupOthers.png" width="533" height="748"/><p>This page discusses some available setup options.</p>
<h2>Preferences</h2><p>In the Main Menu, click Setup, Preferences.</p>
<img src="images/preferences.png" width="754" height="724"/><p>Preferences allow Open Dental to be highly customizable and fit your practice's needs and workflow. Determine default settings or options for each module and other program features. This includes things like appointment behavior, insurance and claim defaults, account settings, and more. </p>
<a name='showfeatures'></a><h2>Show Features</h2><p>In the Main Menu, click Setup, Advanced Setup, Show Features.</p>
<img src="images/showFeatures.png" width="516" height="404"/><p>Show or hide features from Open Dental based on your practice's needs. This includes insurance features (e.g., Medicaid, Capitation, Medical Insurance), public health features, and more.</p>
<h2>Definitions</h2><p>In the Main Menu, click Setup, Definitions.</p>
<img src="images/definitions.png" width="711" height="442"/><p>Definitions allow customization of colors, list options, categories, and some defaults available on various windows. This includes payment types, chart graphic colors, blockout types and colors, appointment module colors, and more.</p>
<h2>Display Fields</h2><p>In the Main Menu, click Setup, Display Fields.</p>
<img src="images/displayFields.png" width="551" height="466"/><p>Display fields can be customized to determine what information fields appear in various areas of Open Dental. Choose what information you shows up in areas like the <a href="0_apptedit.html">Edit Appointment</a> window, Patient Information area in the <a href="0_chart.html">Chart Module</a> or <a href="0_family.html">Family Module</a>, Select Patient window, and more.</p>
