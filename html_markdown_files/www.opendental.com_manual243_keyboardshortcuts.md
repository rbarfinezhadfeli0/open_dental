<div class="TopBar2"><p>Keyboard Shortcuts</p></div>
<div class="GeneralPageContent">
<p>See <a href="advancedtopics.html">Advanced Topics</a>.</p>
<p>Open Dental has some keyboard shortcuts to help navigate the program.</p>
<p>General Shortcuts: <table class="simpletable">
<tr>
<th style="width:30%">Shortcut</th>
<th style="width:70%">Action</th>
</tr>
<tr>
<td>Shift + F1</td>
<td>Open the Open Dental Manual search page (<a href="../site/searchSite.html">https://www.opendental.com/site/searchsite.html)</a></td>
</tr>
<tr>
<td>Alt + F</td>
<td>Open the File menu.</td>
</tr>
<tr>
<td>Alt + S</td>
<td>Open the Setup menu. </td>
</tr>
<tr>
<td>Alt + L</td>
<td>Open the Lists menu.</td>
</tr>
<tr>
<td>Alt + R</td>
<td>Open the Reports menu. </td>
</tr>
<tr>
<td>Alt + T</td>
<td>Open the Tools menu. </td>
</tr>
<tr>
<td>Alt + C</td>
<td><ul>
<li>Open the Clinics menu. </li>
<li>When in a window, selects the Cancel or Close button.</li>
</ul>
</td>
</tr>
<tr>
<td>Alt + E</td>
<td>Open the eServices Setup window.</td>
</tr>
<tr>
<td>Alt + A</td>
<td>Open the Alerts menu.</td>
</tr>
<tr>
<td>Alt + H</td>
<td>Open the Help menu.</td>
</tr>
<tr>
<td>Alt + O </td>
<td><ul>
<li>Logs off Open Dental.</li>
<li>When in a window, selects the OK button.</li>
</ul>
</td>
</tr>
<tr>
<td>Ctrl + D</td>
<td>Inserts date into a text box.</td>
</tr>
<tr>
<td>Ctrl + P</td>
<td><ul>
<li>Opens the Select Patient window.</li>
<li>When in the Imaging Module, and a PDF is selected, opens the print window.</li>
</ul>
</td>
</tr>
<tr>
<td>Ctrl + Q</td>
<td>When in a note box, opens the Quick Paste Notes window.</td>
</tr>
<tr>
<td>Ctrl + X</td>
<td>Open the Referrals for Patient window.</td>
</tr>
<tr>
<td>Ctrl + Shift + C</td>
<td>Open the Contacts window.</td>
</tr>
<tr>
<td>Ctrl + Shift + F</td>
<td>Open the Procedure Codes window.</td>
</tr>
<tr>
<td>Ctrl + Shift + W</td>
<td>Open the Wiki.</td>
</tr>
<tr>
<td>Esc (or other keys that function as Esc)</td>
<td>Close the active Open Dental window, unless it is a module window (e.g., Account Module, Manage Module). Most changes are not saved.</td>
</tr>
<tr>
<td>F1:F12</td>
<td><ul>
<li>When in the Appointments Module, cycles through Appointment Views.</li>
<li>When in the Chart Module, cycles through Chart Views.</li>
</ul>
</td>
</tr>
</table>
</p>
<p>When in the Edit Appointment window: <table class="simpletable">
<tr>
<th style="width:30%">Shortcut</th>
<th style="width:70%">Action</th>
</tr>
<tr>
<td>s</td>
<td>Select<i> Save.</i> When cursor is not selected in a note box or drop-down menu, close the window and save changes.</td>
</tr>
</table>
</p>
<p>Menu Shortcuts: <table class="simpletable">
<tr>
<th style="width:30%">Shortcut</th>
<th style="width:70%">Action</th>
</tr>
<tr>
<td>Up / Down</td>
<td>When a menu is open, move up or down the list.</td>
</tr>
<tr>
<td>Left / Right</td>
<td>When a menu is open, move to the submenu if one exists.</td>
</tr>
<tr>
<td>Any Character</td>
<td>Jump to the next entry in the menu starting with that character.</td>
</tr>
</table>
</p>
<p>Alt Shortcuts:</p>
<p>In any window, tap the <span class="codeblock">Alt</span> key on the keyboard to show any buttons that have <span class="codeblock">Alt + [Character]</span> shortcuts. Available character shortcuts are underlined. </p>
<img src="images/keyboardShortcutAlt.png" width="532" height="388"/><p class="MarginBottomZero">For example for the Prompt List window shown above, the following shortcuts are valid:  </p>
<ul class="MarginBottomGap">
<li>Alt + A: Select All:</li>
<li>Alt + O: Select None</li>
<li>Alt + P: Preview Auto Note</li>
<li>Alt + R: Remove Auto Note prompt</li>
<li>Alt + N: Next prompt</li>
<li>Alt + E: Exit Prompt List</li>
</ul>
