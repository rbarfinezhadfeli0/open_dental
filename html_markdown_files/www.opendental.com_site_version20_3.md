		<div id="Content">
<p>See <a href="versions.html">Versions</a></p>
<p>Version 20.3 was released on 10/13/2020.</p>
<p><img src="images/youtubeLogo.png" width="16" height="16" style="margin-bottom: 0px"/> Overview of <a href="https://youtu.be/8Si8-n-vfvg">Version 20.3</a></p>
<h2>Major Changes</h2>
<p><b>2021 CDT Codes</b>: The 2021 CDT codes are available in version 20.3.48 or 20.4.28 or greater. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7procedurecodetools.html">Procedure Code Tools</a></p>
<br><p class="MarginBottomZero"><b>New CareCredit Integration</b>: Maximize treatment acceptance with the full CareCredit integration.  </p>
<ul class="MarginBottomGap">
<li>Pre-screen patients for approval using QuickScreen. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carecreditaction.html">CareCredit Action</a></li>
<li>Take CareCredit payments and issue refunds. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carecreditpayment.html">CareCredit Payment</a>, <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carecreditrefund.html">CareCredit Refund</a></li>
<li>Review transactions. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carecredittransactions.html">CareCredit Transactions</a></li>
<li><img src="images/youtubeLogo.png" width="16" height="16" style="margin-bottom: 0px"/> See our video: <a href="https://youtu.be/owhEAk6TIdI">CareCredit Integration Demo</a></li>
</ul>
<p class="MarginBottomZero"><b>Bring Your Own Device (BYOD)</b>: BYOD allows patients to check-in for appointments, complete patient forms, and take a patient picture, all from their mobile device. </p>
<ul class="MarginBottomGap">
<li>Enable BYOD in <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7eclipboardsetup.html">eClipboard Setup</a>.</li>
<li>Send a link via text message to the patient. Texts can be sent manually or automatically using Arrivals. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7byodworkflow.html">BYOD: Workflow</a></li>
</ul>
<p><b>Web Sched Existing Patient</b>: Allow existing patients to schedule their appointments online. Available in V20.3.49. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7webschedexistingpatsetup.html">Web Sched Existing Patient</a></p>
<br><p class="MarginBottomZero"><b>Updated Blue Book Feature</b>: The improved Blue Book feature makes estimates easier than ever for your out-of-network insurance plans.  </p>
<ul class="MarginBottomGap">
<li>As claims are received, enter the allowed fees to improve estimates for future procedures. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimreceive.html">Receive Claim</a></li>
<li>Set up preferred Blue Book calculations. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bluebooksetup.html">Insurance Blue Book Setup</a></li>
<li><img src="images/youtubeLogo.png" width="16" height="16" style="margin-bottom: 0px"/> See our video: <a href="https://youtu.be/Jghg56V6uRA">Blue Book Webinar</a></li>
</ul>
<p><b>Language Translations for Web Forms</b>: When language translations are created for sheets, patients can select their preferred language when filling web forms. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7webformspatient.html">Web Forms: What Patient Sees</a></p>
<p class="MarginBottomZero"><br><b>Patient Responsibility Column</b>: Now available in Edit Claim and Enter Payment windows. <a href="https://www.opendental.com/manual203/modulesetupaccount.html">Account Module Preferences</a>, <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimreceive.html">Receive Claim</a></p>
<ul class="MarginBottomGap">
<li><img src="images/youtubeLogo.png" width="16" height="16" style="margin-bottom: 0px"/> See our video: <a href="https://youtu.be/q6hGlEB_gAU">QuickTip: Adding the Patient Responsibility Column</a></li>
</ul>
<h2>Changes By Category</h2>
<p class="MarginBottomZero"><b>Account Module</b></p>
<ul class="MarginBottomGap">
<li>Assign a Payment Type override to individual Recurring Charges. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7recurringchargesauthorize.html">Authorize Recurring Charges</a></li>
<li>Account Module preference disallows primary claim status to be marked as 'Hold Until Pri Received'. <a href="https://www.opendental.com/manual203/modulesetupaccount.html">Account Module Preferences</a></li>
<li>EdgeExpress added as alternative to XCharge. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7edgeexpress.html">Edge Express Payment</a></li>
</ul>
<p class="MarginBottomZero"><b>eServices</b></p>
<ul class="MarginBottomGap">
<li>Automated email birthday messages. <a href="https://www.opendental.com/manual203/massemailbirthdays.html">Mass Email Birthdays</a></li>
<li>Alert created when the MySQL and eConnector have different times. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7alerts.html">Alerts</a></li>
<li>Option to exclude a patient from receiving any automated appointment messaging. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7patientedit.html">Edit Patient Information</a></li>
</ul>
<p class="MarginBottomZero"><b>Miscellaneous</b></p>
<ul class="MarginBottomGap">
<li>Enhanced security so providers must be associated with a user to access PDMP. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgepdmpillinois.html">PDMP Bridge</a></li>
<li>Added fields to send electronically on HCFA 1500 Claim Form. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimform1500.html">HCFA 1500 Claim Form</a></li>
<li>User setting to show existing other color on tooth chart when a different provider in the practice completed work. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7usersettings.html">User Settings</a></li>
<li>Automatically load task search when searching by tasknum. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7tasksearch.html">Task Search</a></li>
<li>Appointment buttons moved to top of Appointment Module. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7appointments.html">Appointments Module</a></li>
<li>Enterprise preference to turn off email address auto-complete in Edit Email Message window. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7enterprisesetup.html">Enterprise Setup</a></li>
<li>Daily Production, Production and Income report, and Monthly Production Goals reports calculate treatment plan discounts into adjustments. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7reportproductiongoal.html">Monthly Production Goal Report</a>, <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7reportprodinc.html">Production and Income Reports</a></li>
<li>Filter out archived wiki pages when adding internal links. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7wikiaddedit.html">Wiki Edit</a></li>
<li>Changing the referral associated to a patient logs an entry in the Audit Trail. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7referrals.html">Referrals</a></li>
<li>Determine level of data sent to secondary insurance carriers. <a href="https://www.opendental.com/manual203/modulesetupfamily.html">Family Module Preferences</a>, <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carriers.html">Carriers</a></li>
<li>Customize border color for windows within Open Dental. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7definitionsmisccolors.html">Definitions: Misc Colors</a></li>

</ul>
