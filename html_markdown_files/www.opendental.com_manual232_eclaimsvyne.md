<p>In <a href="clearinghouses.html">Clearinghouses</a>, double-click Vyne Dental.</p>
<img src="images/eclaimsVyne.png" width="915" height="689"/><p>Vyne Dental is a clearinghouse option that provides electronic claim processing. Visit <a href="https://vynedental.com/">www.vynedental.com</a> to learn more about their services. </p>
<h2>Open Dental Setup</h2>
<p>To enable Vyne Dental as the default clearinghouse, set Vyne Dental as the Dental default in the Clearinghouses window.</p>
<p>In the Edit Clearinghouses window, set the following. </p>
<img src="images/eclaimsVyneSettings.png" width="483" height="141"/><p class="MarginBottomZero">Enter office information.  </p>
<ul class="MarginBottomGap">
<li><b>Tax ID Number</b>: Enter the office Tax ID number.</li>
<li><b>Name</b>: Enter the office name. </li>
<li><b>Telephone Number</b>: enter the office phone number, without any formatting (e.g., 5035551234).</li>
</ul>
<p>Other information in the clearinghouse setup should match the screenshot above. </p>
<h2>Renaissance Users</h2>
<p>Some Vyne users may be using the <a href="eclaimsrenaissance.html">Renaissance E-Claims</a> clearinghouse in Open Dental. Vyne Dental is an updated version and users can switch at their convenience.</p>
<p>All new Vyne users should use the Vyne Dental clearinghouse.</p>
<p>Vyne Dental is moving all existing Renaissance users to Vyne Dental. If claims stop sending while using Renaissance, contact Open Dental Support for assistance moving to Vyne Dental. An Open Dental update may be required. </p>
