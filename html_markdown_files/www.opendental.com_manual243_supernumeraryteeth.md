<div class="TopBar2"><p>Supernumerary Teeth</p></div>
<div class="GeneralPageContent">
<p class="MarginBottomZero">Patients with supernumerary teeth can have procedures charted and sent to insurance using the process below. </p>
<ol class="MarginBottomGap">
<li>Chart procedure as normal.</li>
<li>On the <a href="procedureedit.html">Procedure</a>, change the tooth number.  <ul>
<li>For supernumerary teeth: <ul>
<li>US users: Valid values are 51-82 and AS-TS</li>
<li>Canada and Foreign users: Valid value is 99. </li>
</ul>
</li>
<li>Permanent supernumerary tooth numbers add 50 to the tooth number (tooth 1 = 51).</li>
<li>Primary supernumerary tooth numbers add an S (tooth A = AS).</li>
</ul>
</li>
<li>Click OK and send procedure to insurance.</li>
</ol>
<p>Procedures for supernumerary teeth do not show on the graphical tooth chart, but they do get billed to insurance. To show a supernumerary tooth on the graphical tooth chart use the <a href="chartdrawing.html">Draw</a> tool to draw in a tooth.</p>
