<p>See <a href="versions.html">Versions</a></p>
<p>Version 12.2 was released on 6/28/2012 </p>
<p><b>Claim Fields:</b> More claim fields added for 5010 and DentiCal: Original Reference Number, CLM01 Claim Identifier, Correction Type, and Date Resent. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimedit.html">Claim</a>.</p>
<p><b>Scanning:</b> Six more scanning settings, and the settings are per workstation. This should help with some duplex scanning problems as well as speeding up the scanning process. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7imagingsetup.html">Imaging Setup</a>.</p>
<p><b>eCW:</b> Third integration mode called "full" which opens up nearly all Open Dental features yet still passes all HL7 messages back and forth. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgeeclinicalworks.html">eCW</a>.</p>
<p><b>Timecards: </b>Report for all employees on one page, exportable for payroll service. Timecards print for all employees at once, multiple pages with detail. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7timeclock.html">Time Clock</a>.</p>
<b>Minor Changes</b><br>  Appointment color override per appointment. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7aptedit.html">Edit Appointment</a>.<br>  Certain procedure codes assigned to one provider by default. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7procedurecodeedit.html">Procedure Code Edit</a>.<br>  eCW completed procedures automatically attached to appointments. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgeeclinicalworks.html">eCW</a>.<br>  QB bridge supports choosing from multiple income accounts when making deposit. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7quickbooks.html">QuickBooks</a>. <br>  Security permissions for adding/removing a referral to/from patient. Included in audit trail. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7permissions.html">Permissions</a>.<br>  Claim Send window now has a date column and sorts by date. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7claimsend.html">Sending and Printing Insurance Claims</a>.<br>  Appointments for Patient window, added clinic and provider columns. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7apptspatient.html">Scheduling an Appointment</a>.<br>  X-Charge, better support for voids and returns. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7xcharge.html">X-Charge</a>.<br>  Equipment is now searchable by location. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7equipment.html">Equipment</a>. <br>  BioPAK bridge. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgebiopak.html">BioPAK</a>.<br>  RayMage bridge. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgeraymage.html">RayMage</a>. <br>  New Claim Button Behavior. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7account.html">Account Module</a>. <br>
