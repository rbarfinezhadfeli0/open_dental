<div class="GeneralPageContent">
<p>In <a href="canadaclaims.html">Canada Claims</a>, click <b>History</b>.</p>
<img src="images/canadaEtransEdit.png" width="897" height="519"/><p>The Etrans Edit window displays the history of the last action taken on a claim. A history of Etrans for previous actions taken can be found in <a href="claimsend.html">Send Claims</a>, History section. </p>
<p><b>Message Text Sent</b>: Displays the message sent electronically to the carrier. </p>
<p><b>Message Text Received</b>: Displays the message received from the carrier. </p>
<p><b>Preview</b>: Click to generate a new printable document containing the response from the carrier.  <div class="Note">Note: Previews generated manually may not contain historically accurate data. See <a href="canadaclaims.html">Canada Claims</a>, Automatic Printing for additional details.</div>
</p>
<h2>Carrier Response</h2>
<p>After a claim is received, a carrier will send back an EOB, Rejection, or Claim Acknowledgment.</p>
<img src="images/canadaResponse.png" width="915" height="564"/><p class="MarginBottomZero"><b>Write Amounts to Claim</b>: Enter the insurance benefit amounts from the claim response to the procedure estimates on the claim. Only usable for claims that receive an Explanation of Benefits. </p>
<ul class="MarginBottomGap">
<li>If the Clearinghouse Setup is set to <i>Download EOBs and Auto Receive</i>, the amounts will also be written and the insurance insurance payments and claim will be marked as received. </li>
</ul>
<p><b>Print Patient Copy / Print Dentist Copy</b>: Prints the currently displayed claim response and appends the title with Patient Copy or Dentist Copy, respectively. </p>
