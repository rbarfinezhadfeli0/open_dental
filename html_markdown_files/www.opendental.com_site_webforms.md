		<div id="Content">
<p>Web Forms are an <a href="eservices.html">eService</a> that allows a patient to fill out and submit forms online using any online browser. </p>
<div align="center">
<img src="images/webFormsSiteHeader.png" width="900" height="300"/></div>
<p><img src="images/youtubeLogo.png" width="16" height="16" style="margin-bottom: 0px"/> Watch our videos: <a href="https://youtu.be/q55BHBIE8k4">Web Forms Webinar</a> &amp; <a href="https://youtu.be/vxTM-m3JHBU">Mobile Web Forms</a></p>
<p class="MarginBottomZero"></p>
<ul class="MarginBottomGap">
<li>Patient Forms and Medical History <a href="0_sheets.html">Sheets</a> can be used as Web Forms. This includes things like: <ul>
<li>Patient registration forms</li>
<li>Medical histories</li>
<li>Financial and HIPAA agreements</li>
</ul>
</li>
<li>Patients can fill out forms using a link you provide them.</li>
<li>Forms are matched to existing patients and can easily be attached to a patient if no match is found.</li>
<li>Data entered on forms (e.g., patient information, allergies, etc.) can be imported into the Open Dental database. </li>
</ul>
<h2>Getting Started</h2>
<p class="MarginBottomZero">The Web Forms feature is available at no additional fee for practices on support with a unique, active registration key. There are two options to create a Web Form: </p>
<ul class="MarginBottomGap">
<li>Have Open Dental convert your forms for you. See <a href="fees.html#customsheets">Fees</a>.</li>
<li>Design the form yourself from <a href="0_sheets.html">Sheets</a></li>
</ul>
<p>Once your form has been created as a Sheet, add the sheet to <i>Available Web Forms</i> in Web Form Setup and construct a URL. The URL can then be shared with patients via email, a link on your website, and more.</p>
