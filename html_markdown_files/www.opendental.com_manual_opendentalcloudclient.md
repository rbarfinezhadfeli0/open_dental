	  <h1>Open Dental Cloud Client</h1>
<p><a href="../site/opendentalcloud.html">Open Dental Cloud</a> requires the OpenDentalCloudClient to launch third-party integrations. </p>
<img src="images/cloudSignInODClientError.png" width="915" height="697"/><p>The OpenDentalCloudClient is not required to access Open Dental, but users receive a warning if it is not installed.</p>
<h2>Installation</h2>
<p>Click to install the <a href="../resources/ODCloudClientInstaller.msi">ODCloudClientInstaller.msi</a> installer. </p>
<p class="MarginBottomZero">Alternatively, install from the Open Dental Cloud site: </p>
<ol class="MarginBottomGap">
<li>Click the <i>download OpenDentalCloudClient</i> link.</li>
<li>The installer will appear in the downloads folder. </li>
<li>Double-click <i>ODCloudClientInstaller</i> to run it.<br/><ul>
<li>If Microsoft Defender blocks the installation, click <i>More Info</i>, then <i>Run Anyway</i>.</li>
</ul>
</li>
</ol>
<p>Computers installing the OpenDentalCloudClient must meet the <a href="https://opendental.com/site/computerrequirements.html">Computer Requirements</a>. </p>
<h2>Purpose</h2>
<p class="MarginBottomZero">The purpose of OpenDentalCloudClient is to: </p>
<ul class="MarginBottomGap">
<li>allow copy/paste using the clipboard.</li>
<li>capture the local computer name for various settings.</li>
<li>facilitate <a href="programlinks.html">Program Links</a>.</li>
<li>access TWAIN drivers to acquire images.</li>
</ul>
