		<div class="GeneralPageContent">
<p>There are many different places and tools for notes in Open Dental. </p>
<p><img src="images/YouTube_PlayButton_15px.png" width="16" height="16" style="margin-bottom: 0px"/> See our video: <a href="https://youtu.be/J6f_nED9epo">Clinical Charting II: Clinical Notes Webinar</a>.</p>
<h2>Procedure and Group Notes</h2><p>In the <a href="0_procedureedit.html">Procedure</a> Edit window, at the right, is the Procedure Note area.</p>
<img src="images/procEditNotes.png" width="519" height="279"/><p>Notes can be added to a single procedure from the <a href="0_procedureedit.html">Procedure Info</a> window. Notes can be added when a procedure is still treatment planned or after it has been set complete. These notes can be electronically signed or stamped.</p>
<p>Additionally, the Group Notes feature can be used to add a single note for multiple procedures.</p>
<p>In the <a href="0_chart.html">Chart Module</a>, highlight multiple procedures for a patient. Right-click and select <b>Group Note</b>.</p>
<img src="images/procGroupNote.png" width="915" height="554"/><p>Both procedure notes and group notes can be viewed from the Progress Notes area of the Chart Module.</p>
<img src="images/chartProgress.png" width="701" height="351"/><h3>Chart Letters</h3><p>Can be used for longer notes, typically oral surgery. These notes are written in a Word document which is easily accessible from the Chart Module and warns a user if the notes have been edited externally.</p>
<h2>Note Templates</h2><p>The Auto Notes and Quick Paste Notes features can be used to easily input commonly-used note information.</p>
<h3>Auto Notes</h3><p>Place cursor in a Procedure Note box. Above the box, click Auto Note.</p>
<img src="images/autoNoteCompose.png" width="838" height="639"/><p>Auto Notes are most useful for longer form notes that require input (e.g., SOAP notes). </p>
<p>These notes both input text and prompt for additional information. Prompts can be set up to allow you to choose from a preset list of options or enter free-form text. Auto Note templates and prompts are customizable. </p>
<h3>Quick Paste Notes</h3><p>In the Main Menu, click Setup, Quick Paste Notes.</p>
<img src="images/quickPasteNotesSelect.png" width="915" height="463"/><p>Quick paste notes are useful to easily insert text. Abbreviations can be assigned to a Quick Paste Note so that when you type the abbreviation (e.g., <i>?dlb</i>), the full note is added. Alternatively, in many text boxes, you can right-click to open a list of Quick Paste Notes and select a note to insert.</p>
<h2>Appointment Module Notes</h2><p>There are various ways to add notes that appear in the <a href="0_appointments.html">Appointments Module</a> or on an individual <a href="0_apptedit.html">Appointment</a>.</p>
<h3>Appointment Notes and Fields</h3><p>There is a Note area in the <a href="0_apptedit.html">Edit Appointment</a> window. This allows you to add additional appointment information that can be displayed on the appointment from the schedule.</p>
<img src="images/apptEditNote.png" width="394" height="59"/><p>Customizable appointment fields can also be used to add appointment-specific notes. </p>
<img src="images/apptFields.png" width="260" height="101"/><p>Both Appointment Notes and Appointment Fields can be viewed and edited from the <a href="0_apptedit.html">Edit Appointment</a> window and they can be added to appointment views to see from the schedule.</p>
<img src="images/apptSchedule.png" width="347" height="410"/><h3>Patient Notes</h3><p>Patient notes can be placed on the schedule and can act as a reminder.</p>
<img src="images/apptPatientNote.png" width="263" height="48"/><p>These appear similar to appointments on the schedule, but do not require you to schedule an actual appointment.</p>
<h3>Blockouts</h3><p><a href="0_blockouts.html">Blockouts</a> can be created on the schedule to add notes or designate specific time blocks.</p>
<img src="images/blockout.png" width="236" height="78"/><h2>Patient-Specific Notes</h2><p>There are various area to add notes related to a specific patient or family.</p>
<h3>Address and Phone Notes</h3><p>Address and Phone Notes can be added for additional details about a patient's contact information (e.g., call before 3 PM). </p>
<img src="images/patEditAddressNotes.png" width="482" height="89"/><p>These notes are displayed in various area of Open Dental, like the Recall List, Appointments, etc.</p>
<h3>Financial Notes</h3><p>In the <a href="0_account.html">Account Module</a>, you can add important financial notes.</p>
<img src="images/accountMain.png" width="190" height="576"/><p>Urgent notes display in the account and various other areas of Open Dental.</p>
