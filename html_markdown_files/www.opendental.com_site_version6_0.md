<p>See <a href="versions.html">Versions</a></p>
<p>Version 6.0 Released on 10/17/08.</p>
<p><b>Tooth Chart Drawing:</b> Freehand drawing on top of the 3D (or 2D) tooth chart. Includes eraser, color pickers, and color changer. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7chartdrawing.html">Chart Drawing</a>. </p>
<p><b>Multiple Recall Types:</b> Distinguishes between prophy and perio. Tracks due dates for BW, pano, FMX, etc. Can add other types, also. Endo followup for example. Displays due dates in highly visible manner on main screen of Family module. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7recalltypes.html">Recall Types</a>. </p>
<p><b>Recall E-mail: </b>Bulk sending of recall e-mails. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7recalllist.html">Recall List</a>. </p>
<p><b>Lab Cases:</b> List now shows all incomplete lab cases rather than a date range. Can right click to jump straight to appointment. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7labcaseedit.html">Lab Cases</a>. </p>
<p><b>Appointment Times:</b> New fields to track TimeArrived, TimeSeated (in op), and TimeDismissed. List of patients in waiting room shows at lower right of Appointments module along with the amount of time they've been waiting. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7aptedit.html">Appointment Edit</a>, Time Tracking section. </p>
<p><b>Payment Plans:</b> Previous assumption was that the treatment had already been done. Payment plans have now been enhanced to easily handle situations where work has not yet been complete or has been partially completed. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7paymentplanpatient.html">Payment Plan</a>. </p>
<p> Recall statuses now change automatically after sending postcards or email. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7recalllist.html">Recall List</a>. <br>  Bridge to Dolphin Imaging. <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7bridgedolphin.html">Dolphin</a>. <br>  Referrals in Family module now has a background color to help with visibility.<br>  Routing slip font size increased. <br>  Chart module quick add buttons: amalgams can be hidden. <a href="https://www.opendental.com/manual174/modulesetupchart.html">Chart Module Preferences</a>. <br>  Practice info added to top of perio printout.</p>
