<p>Canadian carriers can easily be added and maintained, without manually creating each one.</p>
<p>For <a href="canada.html">Canada</a> users, click Lists, <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carriers.html">Insurance Carriers</a>.</p>
<img src="images/canadaCarriersList.PNG" width="915" height="688"/><p> Insurance carriers marked as <i>CDAnet</i> are capable of sending electronic claims to ITRANS or Claimstream (TELUS).</p>
<p class="MarginBottomZero">Search criteria:  </p>
<ul class="MarginBottomGap">
<li><b>Carrier</b>: Search the list by carrier name.</li>
<li><b>ElectID</b>: Search the list by electronic ID.</li>
<li><b>Phone</b>: Search the list by carrier phone number.</li>
<li><b>Show Hidden</b>: Check to include carriers marked as hidden.</li>
<li><b>CDAnet Only</b>: Check to only show carriers marked as CDAnet.</li>
</ul>
<p>Click <b>Refresh</b> to update the list.</p>
<p class="MarginBottomZero">To update the Carriers list, select the criteria and click <b>Update Carriers</b>. This button is for <a href="itrans20.html">ITRANS 2.0</a> and <a href="claimstream.html">Claimstream</a> users only. Carriers are updated based on the Electronic ID.<br></p>
<ul class="MarginBottomGap">
<li><b>Address</b>: Update carrier addresses if no address is present.</li>
<li><b>Phone</b>: Update carrier phone number if no phone number is present.</li>
<li><b>Add Missing</b>: Add any missing CDAnet carriers (e.g., new carriers) based on carrier name and carrier ID. Carriers that share a carrier ID or don't support electronic claims need to be added manually.</li>
<div class="Note">Note: Regardless of whether any boxes are checked, the CDAnet information in the bottom half of each carrier is still updated when the Update Carrier tool is run. </div>
</ul>
<p></p>
<p>Click <b>Add</b> to add a new carrier. Click <b>Combine</b> to <a href="../autoLogin.aspx%EF%B9%96ReturnUrl=%EA%A4%B7help253%EA%A4%B7carrierscombine.html">Combine Carriers</a>. </p>
<p>Double-click on a carrier to view carrier information and CDAnet information.</p>
<img src="images/canadianCarrier.png" width="704" height="704"/><p>The Carrier information in the top half is editable by the office. </p>
<p class="MarginBottomZero">The information under <i>CDAnet</i> is maintained by the CDA. </p>
<ul class="MarginBottomGap">
<li>This information is updated when running the Update Carriers tool.</li>

</ul>
